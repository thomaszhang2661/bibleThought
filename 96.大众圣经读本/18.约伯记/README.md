# 《人民圣经·约伯记》中文翻译

## 📚 目录结构

本翻译**忠于原书大纲**组织。原书（The People's Bible: Job, Rudolph E. Honsey）按"引论 + 五大部分"编排，其中论述部分再分三轮，文件划分与之对应：

| 序号 | 文件 | 经文范围 | 内容 |
|------|------|--------|------|
| 00 | 约伯记引论 | — | 编者前言、约伯记简介（标题/日期/地点/作者/文学特征/主题/意义）、全书大纲 |
| 01 | 序言 | 1:1–2:13 | 约伯的敬虔、撒但的两次挑战、约伯的苦难、三友来访 |
| 02 | 第一轮论述 | 3:1–14:22 | 约伯的哀歌，以利法、比勒达、琐法首次发言及约伯的回应 |
| 03 | 第二轮论述 | 15:1–21:34 | 三友再次发言与约伯的回应 |
| 04 | 第三轮论述 | 22:1–31:40 | 以利法第三次发言、比勒达第三次发言、约伯的回应与最后长篇独白（含"智慧篇"第28章） |
| 05 | 以利户的演说 | 32:1–37:24 | 年轻人以利户的四篇演说 |
| 06 | 神对约伯的演说 | 38:1–41:34 | 神的两篇演说，自然界、动物界、巨兽与利未亚坦彰显神的大能与智慧 |
| 07 | 尾声 | 42:1–17 | 约伯的悔改、神责备三友、约伯的复兴 |

## 🎯 翻译特点

- ✅ **完整翻译** — 所有段落、脚注、括注、学者引用均已翻译，无省略
- ✅ **忠于原书结构** — 按原书"引论 + 五部分（论述分三轮）"组织，第41章利未亚坦按原书归入"神的演说"
- ✅ **人名统一中文** — 圣经人物名统一为标准中译（约伯、以利法、比勒达、琐法、以利户），不中英夹杂
- ✅ **圣经引用** — 保留章节编号，附中文书卷名
- ✅ **学术规范** — 庄重学术文风，参考文献作者名（Delitzsch、Kretzmann、Habel 等）保留原文

## 📖 使用建议

1. **按序阅读**：从 00_引论 开始，依原书大纲顺序阅读
2. **快速查阅**：文件名中的经文范围可快速定位
3. **深入研究**：适合作为约伯记的学术参考与讲道素材
