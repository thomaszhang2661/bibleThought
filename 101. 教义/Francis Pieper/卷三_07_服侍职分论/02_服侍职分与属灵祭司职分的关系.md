# §2 服侍职分与众信徒属灵祭司职分的关系

一切基督徒，即一切藉信心归向基督的人，皆是属灵祭司，因而都蒙召传讲福音。彼得前书2:9所言："你们是……圣洁的祭司（*hierateuma hagion*，属灵的祭司职分）……要宣扬那召你们出黑暗、入奇妙光明者的美德"，所称呼的并非长老或主教，乃是全体基督徒。公众服侍职分固然以这一属灵祭司职分为前提。神的旨意与定规乃是：长老或主教当从信徒——即属灵祭司——中间选出，而非从不信者中间选取，这从提摩太前书第3章与提多书第1章所列长老、主教应具备之资质明显可见。所列积极资质（善于教导、有耐心等）与消极资质（不争竞、不贪财等），无不以对基督的个人信仰为前提。

然而，圣经仍将属灵祭司职分与服侍职分明确区分。盖除了圣经赋予每位基督徒的一般教导能力之外，[注1] 还需要特别的教导才能；除了属灵祭司传讲福音的普遍呼召之外，[注2] 还需要专门的呼召。若有会众以抽签、按字母顺序或其他类似方式任命公众执事，并以"众基督徒皆属灵祭司、皆善于教导、皆受圣灵感动"为由为此辩护，则该会众实是违背了神的规定。不，圣经反而在提摩太前书5:22警戒说："不可急促给人按手"，并敦促按照所列资质（提摩太前书3:1-7；提多书1:6-12）来选择长老或主教。路德说："会众应选择何等资质的主教或牧师，保罗在提多书1:5及以下与提摩太前书3:2及以下已充分教导。"（《路德全集》圣路易版，第十卷，1598页。）

路德有力地阐明了属灵祭司职分的权利、特权与义务，同时又清楚区分信徒的普遍祭司职分与服侍职分。就属灵祭司职分，他说："当我们藉这位祭司及其祭司职分（即藉基督）成为基督徒，并在洗礼中藉信心与祂联合时，我们也领受了在众人面前传讲并宣认从祂所领受之道的权利与能力，人人各按其地位与呼召。因为，虽然我们并非都身处公众职务与呼召之中，每位基督徒仍当并可以在任何时地发现有人需要时，用神的话语教导、训诫、劝慰、安慰、责备自己的邻舍；例如，父母对子女与仆人，弟兄邻居、市民或农夫对其他人。因为基督徒当然可以教导尚在蒙昧或软弱中的另一人，以十诫、信经、祷告等训诫他；而听者有责任从他那里接受此言为神的话，并与之公开同声认信。"（《路德注解诗篇110:4》；《路德全集》圣路易版，第五卷，1038页。）

路德也指出，恩典之具无论由普通基督徒还是由执事在公众职务中施行，其本质、能力与功效皆相同。他写道："我们坚定主张，并无另一种神的话语，唯有那向全体基督徒所晓谕要传讲之道；并无另一种洗礼，唯有全体基督徒皆可施行之洗；并无另一种圣餐记念，唯有任何基督徒皆可庆行之圣餐；同样，并无另一种罪，唯有每位基督徒皆可捆绑或释放之罪；此外，并无另一种祭牲，唯有每位基督徒的身体；同样，唯有基督徒方能祷告；而且，唯有基督徒应当审判教义。然而，这些确实乃是祭司与君王的职分。"（《路德全集》圣路易版，第十卷，1590页。）

另一方面，路德也阐明了信徒普遍祭司职分与服侍职分之间的区别。他说："虽然我们众人皆是祭司，我们并不因此就都可以或都应当公开传道、教导或治理。然而，从整个会众中，必须选出若干人，将这一职务交付给他们；凡持守此职务者，现在因这职务的缘故，并非（如其余人一样的）祭司，乃是众人的仆人或执事。若他不能或不愿再传道或服事，他便退回普通众人之中，将职务交给另一人，现在又不过是与每位普通基督徒相同。由此可见，传道的职务——即服侍职分——必须与全体受洗基督徒的普遍祭司职分区分开来。因为这职务不过是公众服务，由整个会众委托给某一人，虽然众人皆共同是祭司。"（《路德全集》圣路易版，第五卷，1037页。）

路德进一步证明行使服侍职分需要特别呼召，说道："既然我们迄今所列举的一切，皆当为全体基督徒所共有（此我们亦已证明），则任何人擅自挺身、将本属我们众人之物据为己有，皆是不合宜的。你可以渴求这一特权并加以运用，只要没有其他同样领受此特权的人。但既然众人皆有此特权，便有必要由一人或若干人（按会众所愿）从中被选拔任命，以众人的名义代表具有同等权利的众人，公开行使这些职务，以免神的子民中产生令人厌恶的混乱，使教会沦为巴别，因为其中一切都当"规规矩矩地按着次序行"，如使徒在哥林多前书14:40所教导的。奉会众命令行使共有权利，与在紧急情况下行使同一权利，有着极大的区别。在会众中，此特权乃公共财产，任何人不可在没有全体会众的呼召与委任的情况下承担；但在紧急情况下，任何人皆可使用。"（《路德全集》圣路易版，第十卷，1589页。）
