# 当爱成为工具：两种属灵成长观

## 一、一种普遍的不安

在基督教圈子里，我们常听到这样的教导：

"真正的改变不来自道德努力，而来自圣灵的更新。" "不要靠自律，要靠恩典。" "福音的能力能够改变人。"

这些话神学上无可挑剔。但许多人在听到时，内心深处会有一种说不清的不适感：好像有什么地方不对劲，却又难以言说。

这种不安往往在实际生活中更明显：

- 教会强调"恩典能改变你"，但你总觉得有种隐形压力："我改变了吗？改变够吗？"
- 父母说"我爱你，神也爱你"，紧接着就是"所以你要..."
- 辅导者说"让圣灵更新你"，但焦点似乎永远在"你有没有变好"

**本文的核心问题是：我们是否不知不觉中，把"改变人"当作了目标，把"爱/恩典/圣灵"当作了达成这目标的手段？**

**当爱成为工具时，它还是爱吗？**

## 二、两种截然不同的框架

### 框架A：功能性/结果导向

**基本逻辑：**

```
诊断：人有问题，需要变好
方法对比：道德自律(无效) vs. 圣灵更新(有效)
目标：人被改变
验证：观察改变是否发生
```

**典型话语：**

- "律法不能改变人，**但**圣灵能"
- "我们需要圣灵的更新，**这样才能**变得圣洁"
- "真正的改变**来自**恩典"

**深层结构：** 神的爱/圣灵(手段) → 改变人的心(过程) → 人变好(目标) → 荣耀神(最终目的)

**关注的核心问题：** "如何让人变好？""哪种方法更有效？""我改变了吗？"

### 框架B：关系性/存在导向

**基本逻辑：**

```
真相：神爱罪人(就是爱这个人本身)
恩典：使罪人成为儿女(身份的改变)
本性：儿女想要像父亲(关系的本质)
呼召：活出你的真实身份
结果：改变自然发生(爱的副产品)
```

**典型话语：**

- "神爱你，不是因为你会变好，而是因为你是祂的孩子"
- "你**已经是**新造的人，活出你的真实身份"
- "改变不是爱的条件，而是爱的果子"

**深层结构：** 神爱人(本身就是目的) → 收养为儿女(本体的礼物) → 儿女想回应(自然的倾向) → 成圣的生活(关系的流露)

**关注的核心问题：** "神如何爱我？""我是谁？""如何回应这份爱？"

## 三、语言的微妙背叛

两种框架的区别，藏在极其微妙的语言差异中。

### 介词的神学

|功能性语言|关系性语言|
|---|---|
|神爱你，**为了**改变你|神爱你，**就是**爱你|
|**借着**恩典，你能成圣|**在**恩典中，你是神的儿女|
|圣灵更新你，**使你能够**...|圣灵使你**成为**...|
|因为被爱，**所以要**...|因为被爱，**所以想要**...|

看起来只是语言细节，但背后是完全不同的神学世界。

### "能够"的陷阱

**常见表述：** "圣灵**能够**改变你" "恩典**能够**使你成圣"

**"能够"暗示什么？** 焦点在能力/效能，衡量的是有效还是无效，心态是评估工具是否好用。

**对比：** "圣灵**已经使**你成为新造的人" "恩典**已经**使你成为神的儿女"

**"已经"强调什么？** 焦点在既成事实，确立身份的确据，心态是活出已经是的身份。

## 四、圣经怎么说？

### 神爱的本质：无条件的给予

**罗马书5:8**

> "惟有基督**在我们还作罪人的时候**为我们死，神的爱就在此向我们显明了。"

**时间顺序揭示了爱的本质：**

不是：我们有悔改的心 → 神爱我们 而是：我们还是罪人 → 神就已经爱了我们

**"还作罪人的时候"这几个字，粉碎了一切功能性神学。**

**约翰一书4:10**

> "不是我们爱神，乃是**神爱我们**，差他的儿子为我们的罪作了挽回祭，**这就是爱了**。"

爱的定义：不是被吸引(因为对方可爱)，不是为交换(期待回报)，而是**主动的、牺牲的给予**。

### 但爱确实带来改变

**约翰一书4:19**

> "我们爱，**因为**神先爱我们。"

**逻辑链：** 神的爱(A) → 我们被爱(B) → 我们回应以爱(C)

**关键：C不是A的目的，C是B的自然流露。**

**以弗所书2:8-10**

> "你们得救是本乎恩，也因着信...我们原是他的工作，**在基督耶稣里造成的**，为要叫我们行善。"

**顺序：**

1. 本乎恩得救(被爱/被接纳)
2. 成为祂的工作(新身份)
3. 为要行善(结果/呼召)

**"为要"不是目的，而是设计：** 不是：神救你**为了**让你行善(交易) 而是：神造你**就是**会行善的新人(本性)

### 儿女的身份

**加拉太书4:6-7**

> "你们既为儿子，神就差他儿子的灵进入你们的心，呼叫：'阿爸！父！'可见，从此以后，你不是奴仆，乃是儿子了。"

**对比：**

|奴仆的心|儿子的心|
|---|---|
|害怕|亲密("阿爸")|
|凭表现赚工价|凭身份享继承|
|焦点：我做得够吗？|焦点：我是天父的儿女！|

**儿女身份本身就是恩典的高峰，不是达成某个目标的中间步骤。**

## 五、两种框架的实际后果

### 在个人属灵生活中

**功能性框架的体验循环：**

```
我得救了 → 现在需要"让圣灵改变我" → 自我检视：我改变了吗？
    ↓                                    ↓
看不到足够改变                        看到一些改变
    ↓                                    ↓
焦虑/怀疑                              骄傲/比较
    ↓                                    ↓
        永不满足的循环
```

**内在对话：** "神的恩典够用吗？我怎么还是老样子..." "也许我需要更多祷告/读经/顺服"

**临床症状：** 属灵完美主义、隐藏的表演、焦虑循环、灵性倦怠

**关系性框架的体验循环：**

```
我被神爱 → 我是祂的孩子 → 我想要像父亲 → 在日常中活出
    ↓                                        ↓
 失败了                                    成长了
    ↓                                        ↓
仍是儿女，回到父怀                        感恩恩典，继续前行
    ↓                                        ↓
            在爱中持续成长
```

**内在对话：** "我是神的孩子，这是确定的" "今天我失败了，但父亲仍爱我，我回到祂那里"

**临床症状：** 真实的自由、持久的动力、真诚的喜乐、可持续性

### 在教养儿女中

**场景：孩子考试不及格**

**功能性回应：** "我们这么爱你，为你付出这么多，**就是希望**你好好学习。你怎么还考这么差？你需要让神改变你的学习态度。"

**传递的信息：** 我们的爱是投资，期待回报；你的价值=你的表现；你让我们失望了

**孩子可能的反应：** 内疚，表面顺从但内心疏离；或者反叛："我永远达不到你们的期望"

**关系性回应：** "我爱你，不管你考多少分。你都是我的孩子，这不会改变。(拥抱)

但我关心你，想帮你找到困难在哪里。因为我知道学习的能力是神给你的礼物，我希望你能享受学习，而不是被它压垮。

我们一起面对，好吗？"

**传递的信息：** 爱是无条件的；关心你的挣扎本身；邀请进入对话

**孩子可能的反应：** 感到被理解和接纳；更愿意敞开真实的挣扎；可能重新思考，因为感受到真诚的关切

## 六、问题的哲学根源

### 康德的绝对命令

18世纪哲学家康德提出：

> **"永远不要把人仅仅当作手段，而要同时当作目的本身。"**

**应用到神学：**

**把人当手段：**

- 我爱这个人 → 为了让他成为某种样子
- 神赐恩典 → 为了改造罪人

**把人当目的：**

- 我爱这个人 → 就是爱他这个人
- 神赐恩典 → 因为祂爱你，要你与祂同在

### 存在 vs. 功能

**功能性价值：** 一个东西的价值 = 它能做什么 （例：锤子的价值在于能钉钉子；如果不能钉钉子了，就没用了）

**存在性价值：** 一个人的价值 = 他是谁 （例：我孩子的价值在于他是我的孩子；即使他什么都做不了，他仍有价值）

**危险的应用：**

当我们说"圣灵能改变人"时， 如果**改变是焦点**， 我们就悄悄把人从**主体**降格为**对象**， 从**目的**降格为**手段**。

## 七、圣经的完整见证

### 神为什么爱我们？

**罗马书5:6-8**

> "因我们还软弱的时候，基督就按所定的日期为罪人死...惟有基督**在我们还作罪人的时候**为我们死，神的爱就在此向我们显明了。"

**时间顺序揭示爱的本质：**

不是：我们有悔改的心 → 神爱我们  
而是：我们还是罪人 → 神就已经爱了我们

**约翰福音17:24**

> "父啊，我在哪里，愿你所赐给我的人也同我在那里"

耶稣的祷告：**让他们与我同在** 不是"让他们变得完美"，而是"**与我同在**" **同在本身就是目的。**

### 儿女的名分：恩典的终点

**加拉太书4:4-7**

> "神就差遣他的儿子...要把律法以下的人赎出来，**叫我们得着儿子的名分**...你不是奴仆，乃是儿子了。"

**救赎的目的：** 不是（首要地）改变我们的行为， 而是：**恢复儿女的身份**

**罗马书8:15-17**

> "你们所受的，不是奴仆的心，仍旧害怕；所受的，乃是**儿子的心**，因此我们呼叫：'阿爸！父！'"

奴仆凭表现赚工价；儿子凭身份享继承。 **儿女身份本身就是恩典的高峰，不是达成某个目标的中间步骤。**

### 改变的位置

**关键：改变在爱的逻辑链中处于什么位置？**

**错误的位置：**

- 改变是爱的条件：你改变了 → 所以神/我爱你（明显错误）
- 改变是爱的目的：神爱你 → 为了改变你（更隐蔽的错误）
- 改变是爱的验证：神爱你 → 你会改变 → 改变证明神的爱（危险）

**正确的位置：**

```
神爱你(A-原因) → 使你成为儿女(B-本体的礼物) → 
儿女想要像父亲(C-新本性) → 在恩典中成长(D-生命的表达) → 
改变发生(E-自然的结果)
```

**E不是A的目的，E是D的结果，而D是C的表达，C是B的本质，B是A的礼物。**

## 八、两种框架的实际后果对比

### 在教养中：场景对比

**场景：青春期孩子说"我不想去教会了"**

**功能性回应：** "我们这么多年带你去教会，为你祷告，就是希望你成为敬虔的人。你怎么能这样？你需要悔改，让神改变你的心。"

**传递的信息：**我们的爱是投资，期待回报；你让我们失望了

**关系性回应：** "孩子，谢谢你诚实告诉我。我想了解发生了什么。（倾听）

我爱你。无论你去不去教会，你都是我的孩子，这不会改变。

但我也想告诉你为什么教会对我很重要——不是因为规则，而是因为我在那里经历了神的爱。我希望你也能经历，不是为了满足我的期望，而是因为这是真正的好。

你知道吗？神爱你，不管你去不去教会。但祂的心意是要你认识祂，经历团契。这不是重担，而是祝福。

我们可以谈谈你的挣扎吗？"

**传递的信息：**爱是无条件的；关心你的挣扎本身；邀请对话，不强加要求

### 在讲道中：结构对比

**经文：马太福音18:21-35（不饶恕的仆人）**

**功能性讲道结构：**

```
一、律法：不饶恕的罪(定罪)
二、福音：神的饶恕(安慰)  
三、命令：所以你们要饶恕
四、能力：靠圣灵，你能做到
```

会众离开时想："我要努力饶恕"（焦点：我的表现）

**关系性讲道结构：**

```
一、认识我们所领受的
   - 主人免了"一千万两"的天文数字债务
   - 你无法偿还，但神说："我免了你！"

二、认识我们是谁  
   - 你不再是债务人，你是儿女
   - 你是慷慨天父的孩子

三、认识我们的挣扎
   - 仆人掐住欠他"十两"的同伴
   - 对比：一千万 vs. 十两
   - 我们也常常忘记自己被免了多少

四、活出儿女的身份
   - 当有人伤害你：记住你被免了一千万
   - 饶恕，不是因为"你应该"（义务）
   - 而是因为你已经是被饶恕的人（身份的流露）

五、恩典中的成长
   - 承认：饶恕很难
   - 应许：当你软弱，父亲仍爱你  
   - 盼望：每次选择饶恕，你在经历自由的喜乐
```

会众离开时想："我被慷慨地爱了，我想分享这爱"（焦点：天父的爱和我的身份）

## 九、深层的神学区分

### 存在先于行动

**神学格言：Agere sequitur esse**（行动跟随存在）

你**做**什么，源于你**是**什么， 不是反过来：你做什么决定你是什么。

**在救恩论中：**

|顺序|功能性错误|关系性正确|
|---|---|---|
|1|悔改(你做)|神的爱(神做)|
|2|改变(你做)|重生(神做)|
|3|因此得救|儿女身份(礼物)|
|4|继续改变(维持)|因此想要顺服(流露)|

**约翰福音1:12-13**

> "凡接待他的...他就赐他们权柄，**作神的儿女**。这等人...乃是**从神生的**。"

1. 从神生（神的行动）
2. 成为儿女（本体的改变）
3. 信祂的名（回应）
4. 接待祂（行动）

### 称义、收养、成圣的关系

**这些不是时间上的阶段，而是同一个现实（联合于基督）的不同面向：**

```
           联合于基督(一次性的事实)
                    |
        ________________________
        |           |           |
      称义        收养        重生
    (法律地位)  (家庭身份)  (新本性)
        |           |           |
        ————————————|————————————
                    |
                  成圣
            (活出新本性)
```

**当你信主的那一刻：**

- 你被称义（法律上无罪）
- 你被收养（关系上是儿女）
- 你被重生（本性上是新造）
- 成圣**开始**（经验上的成长）

**这意味着：** 成圣不是称义后的"第二阶段项目"， 而是儿女身份在生活中的**逐渐实现**。

## 十、最深的悖论

### 当你停止以改变为目标，改变反而发生

**葡萄树的比喻（约翰福音15:4-5）**

> "你们要**常在我里面**...枝子若不常在葡萄树上，自己就不能结果子...因为**离了我**，你们就不能做什么。"

**"常在"重复了7次！**

**焦点：** 不是"如何结果子？" 而是"**常在葡萄树上**"

**枝子不需要：**

- 努力"制造"果子
- 焦虑"我结果了吗？"
- 比较"我的果子比别的枝子多吗？"

**枝子只需要：** 住在树上 → 果子自然就来了

**属灵应用：**

**你的责任：**"常在基督里"（保持联合） **神的责任：**"结果子"（产生改变）

**你的焦点应该在：** 我与基督的关系有多亲密？ **而不是：**我结了多少果子？

**悖论：当你不盯着果子，只专注于住在基督里，果子反而更丰盛。**

### 历史见证

**奥古斯丁的转折：** 早期：努力靠意志力胜过罪 → 失败、挣扎

转折点：听到"拿起来读" 读到罗马书13:14："总要披戴主耶稣基督"

不是"努力变得像基督"，而是"披戴基督"（穿上祂，联合于祂）

**结果：**"你命令贞洁，就赐下你所命令的" 改变，因为**首先接受了**恩典

**德蕾莎修女：**

> "不要过度追求成圣，追求神。成圣是你追求神的副产品。"

## 十一、走出混淆的实践

### 重构你的祷告

**不要这样开始：** "主啊，求你今天改变我，使我更圣洁、更像你..." （焦点：我的缺乏）

**试试这样：** "父啊，感谢你今天又提醒我： 我是你的儿女，这永不改变。 基督在我里面活着，这是现在的真实。

今天，帮助我活出我的真实身份。 不是努力成为我不是的人， 而是活出我已经是的人。" （焦点：我的身份，神的恩典）

### 重构你看自己的方式

**当你失败时：**

**功能性思维：** "我又失败了 → 我没有改变 → 圣灵的工作无效？→ 我真的得救了吗？" （螺旋式下降）

**关系性思维：** "我又失败了 → 但我仍是天父的儿女 → 我回到祂面前 → 祂的爱没有改变 → 在祂的恩典中重新站起" （回到确据）

### 在关系中的语言转换

**对孩子/配偶：**

❌ "我爱你，但你要..." ✓ "我爱你，无论如何。**因为**我爱你，我会引导你，但我的爱不取决于你的表现"

**在教导中：**

❌ "你要努力成圣" ✓ "你已经在基督里，活出你的真实身份"

❌ "圣灵能使你胜过罪" ✓ "圣灵已经使你成为自由的人，你不再是罪的奴仆"

## 十二、面对真实的挣扎

### 当改变很慢或看不见时？

这是最痛苦的问题。

**重新定位问题：**

**不问：**"我改变了吗？"（以结果为中心） **而问：**"我是谁？"（以身份为中心）

**答案：**

"即使我今天又失败了， 我仍然是天父的儿女。

这个挣扎，不定义我。 基督的义，定义我。

我恨恶这个罪，不是因为害怕惩罚， 而是因为它不是我的真实身份。

**改变可能很慢，但方向是确定的**， 因为圣灵的工作是确定的。"

**保罗的例子：**

**罗马书7:24**

> "我真是苦啊！谁能救我脱离这取死的身体呢？"

这是使徒保罗，写了一半新约的保罗，**仍然在挣扎！**

**但第8章开始：**

> "如今，那些在基督耶稣里的就不定罪了"

保罗的挣扎不改变他在基督里的身份。

**腓立比书1:6**

> "我深信那在你们心里**动了善工**的，必成全这工。"

焦点不是我的努力，而是**祂动的善工**。

## 十三、一个不同的完整故事

### 创造：为了团契

三位一体的神，在永恒的爱的团契中完满。 **祂不需要任何东西。**

但爱的本性是给予、分享。

所以神创造人，**不是为了得到什么**（祂已完满）， 而是**为了分享**这团契之爱。

**创世记2:18**

> "那人独居不好"

"不好"不是因为"没用"， 而是因为**不符合人的本质**——人被造是为了关系。

### 堕落：关系的破裂

**创世记3:8-10**

> "那人和他妻子...藏在园里的树木中，躲避耶和华神的面"

悲剧不只是：人违背了命令（功能失败） 悲剧更是：人**躲避神的面**（关系破裂）

罪的本质：不只是"做错事"，更是**"离开关系"**

### 救赎：父亲寻找儿女

**路加福音15章：浪子的比喻**

**小儿子回来时说的话（排练好的）：** "父亲，我不配称为你的儿子，把我当作一个雇工吧！"

**父亲的反应：**

- 打断他的话（没让他说完"雇工"那句）
- 吩咐："把那上好的袍子拿来...戒指戴在他指头上"
- 宣告："因为我这个儿子是死而复活，失而又得的"

**父亲拒绝的：**功能性关系（雇工=凭表现赚工价） **父亲坚持的：**儿子身份（无论如何都是儿子）

**父亲不在乎儿子能不能"工作得好"，** **父亲只在乎儿子回家了。**

这就是福音：不是神评估你能不能改变，而是神奔向你，恢复关系。

### 成圣：儿女长大成人

**不是：**努力达到父亲的标准，这样才能留在家里 **而是：**在父家中长大成人，越来越像父亲

**以弗所书4:13-15**

> "直等到我们...得以长大成人，满有基督长成的身量...在爱中建立自己。"

成长的目标：**认识神的儿子**（关系的深化） 成长的环境：**在爱中**（不是在恐惧、要求、比较中）

**希伯来书12:5-6**

> "主所爱的，他必管教...惟有万灵的**父**管教我们，是**要我们得益处**。"

管教的动机：**因为爱** 管教的目的：得益处（不是达标） 管教的前提：你已经是**儿女**

### 得荣：完全的同在

**启示录21:3**

> "看哪，神的帐幕在人间。**他要与人同住**...神要亲自与他们同在。"

**新天新地的核心：** 不是"终于改造完成了" 而是**"完全的、永恒的团契"**

**启示录22:4**

> "也要**见他的面**"

终极盼望：不是"我终于完美了" 而是"**我要见祂的面，永远与祂同在**"

**同在本身，就是天堂。**

## 十四、回应常见的误解

### 误解1："这是廉价恩典吗？"

**廉价恩典（潘霍华的定义）：** 不要求门徒训练的恩典，没有十字架的恩典

**我们提倡的不是廉价恩典，因为：**

1. **强调身份必然带来行动** 儿女**一定会**想要像父亲，这不是可选的，是**本性**
    
2. **强调十字架的代价** 这份无条件的爱，付出了最高的代价——神儿子的生命
    
3. **强调门徒的呼召** 跟随基督是真实的呼召，但动机不是**赚得身份**，而是**活出身份**
    

**真正的恩典：** 不是降低标准（廉价）， 不是用表现交换（昂贵的功德）， 而是：**白白赐予儿女身份，然后儿女自然想要像父亲**

### 误解2："这会导致放纵吗？"

**罗马书6:1-2**

> "我们可以仍在罪中、叫恩典显多吗？断乎不可！我们在罪上死了的人岂可仍在罪中活着呢？"

**保罗的逻辑：**

不是：因为恩典，所以你**不能**继续犯罪（禁令） 而是：你已经向罪死了，**怎么可能**还活在罪中？（本体的不可能）

**比喻：**

一条鱼对水说："我可以离开你，在陆地上生活吗？" 水回答："你可以试试，但那不是生活，那是死亡。你是鱼，你的本性需要水。"

**同样：** 一个在基督里的人问："我可以继续活在罪中吗？" 答案："你可以试试，但那违背你的新本性。你是神的儿女，你的本性渴望父亲。"

**约翰一书3:9**

> "凡从神生的，就不**继续**犯罪...他也不**能**继续犯罪，因为他是由神生的。"

"不能"不是"不被允许"（外在禁令）， 而是"违背本性"（内在不可能）

就像苹果树"不能"结橙子——不是被禁止，而是本性使然。

### 误解3："这是被动主义吗？"

**被动主义说：**"神会做一切，我只需要等待" **我们说：**"神已经使你成为新造的人，**现在活出来**"

**腓立比书2:12-13**

> "做成你们得救的工夫。因为你们立志行事都是**神在你们心里运行**。"

**"做成"≠"赚得"**

更好的理解："活出你们得救的现实"

**比喻：**

一个王子被父王派到外地学习， 父王说："做一个配得你身份的王子。"

不是说"努力成为王子"（他已经是了） 而是说"活出你王子的身份"

**同样，神对我们说：** "你已经是我的儿女，**活出来**。"

这需要每天的选择、持续的操练、具体的行动， **但根基是：**我已经是儿女（不是在努力成为）。

## 十五、实践的智慧

### 每日的重新定向

**早晨祷告模板：**

```
"父啊，

今天又一次提醒我真理：
我是你的儿女，这不会改变。
基督在我里面活着。
圣灵已经给了我新的本性。

今天帮助我记住我是谁，
活出我已经是的人，
不是努力成为我不是的人。

当我软弱，提醒我：我仍是你的孩子。
当我成长，让我知道：这是你的恩典。"
```

**晚上反思模板：**

```
"父啊，

今天我看到了我的软弱：（具体认罪）
我也看到了你的恩典：（具体感恩）

感谢你，我的身份不建立在今天的表现上，
而建立在基督的义上。

我仍是你的儿女。
明天，帮助我继续活出这身份。"
```

### 自我检测

**问自己这些问题：**

1. 当我想到"神爱我"，紧接着的念头是什么？
    
    - ❌ "所以我要努力..."（义务感）
    - ✓ "我是祂的儿女！"（惊奇与安息）
2. 我如何看待失败？
    
    - ❌ "我又让神失望了，祂的投资失败了"
    - ✓ "我仍是祂的孩子，我要回到父怀"
3. 我的动力来源是：
    
    - ❌ 恐惧（如果不改变...）
    - ✓ 爱（因为被爱，我想回应）
4. 我对孩子/他人的期望，根源是：
    
    - ❌ 我的付出需要回报
    - ✓ 我关心他们的益处

## 结语：一个根本性的选择

我们每个人，每一天，都在两种框架之间做选择：

**框架A问：** "我变好了吗？我做得够吗？我达标了吗？" → 焦点：我的表现 → 结果：焦虑、表演、倦怠

**框架B问：** "我是谁？我被谁爱？我想如何回应？" → 焦点：我的身份和关系 → 结果：自由、真实、喜乐

---

**这不只是神学的细微差别，这关乎你如何经历神，如何活出信仰，如何爱你身边的人。**

当你停止把"改变"当作目标， 当你开始安息在"被爱"的确据中， 你会发现：

**那个你一直努力想成为的人，** **在神眼中，你已经是了——** **你是祂的儿女。**

**现在，只需活出这真实的你。**

改变会发生，但那是在父怀中长大的自然结果， 不是赚得父爱的入场券。

---

**三位一体的神，在永恒的团契中，完满而喜乐。** **祂爱你，不是为了从你这里得到什么，** **而是为了与你分享祂的喜乐。**

**这份爱，本身就是目的。** **你，本身就是目的。**

**当你真正相信这一点时，** **一切都会不同。**
