# 七月十七日

**诗篇吟唱**
8你耶和华是永远至高的。
9耶和华啊，你的仇敌，你的仇敌必然灭亡；一切作孽的人必被驱散。
10你使我的角高举，如野牛的角；你用新鲜的油膏了我。
11我眼睛看见了我敌人所遭的报应；我耳朵听见了那些起来攻击我的恶人所受的报应。
12义人要发旺如棕树，生长如黎巴嫩的香柏树。
13他们种在耶和华的殿中，发旺在我们神的院里。
14他们年老的时候仍要结果子，要满了汁浆而常发青，
15好彰显耶和华是正直的；他是我的磐石，在他毫无不义。
——诗篇92篇8节至15节

**旧约读经**：撒母耳记上1章1节至20节

撒母耳的诞生

1以法莲山地的拉玛琐非有一个人，名叫以利加拿，是苏弗的曾孙，托户的孙子，以利户的儿子，耶罗罕的儿子，是以法莲人。2他有两个妻：一个名叫哈拿，一个名叫毗尼拿。毗尼拿有儿女，哈拿没有儿女。

3这人每年从本城上到示罗，拜耶和华万军之神，在那里，以利的两个儿子何弗尼和非尼哈斯是耶和华的祭司。4每逢以利加拿献祭的日子，他把祭肉分给妻毗尼拿和她所有的儿女；5但给哈拿的是双份，因为他爱哈拿，然而耶和华关闭了她的子宫。6她的对头毗尼拿为激怒哈拿，就不断地激动她，因为耶和华关闭了她的子宫。7年年如此；每逢哈拿上耶和华的殿，毗尼拿就这样激怒她，哈拿就哭泣不吃饭。8她丈夫以利加拿对她说："哈拿啊，你为何哭泣，为何不吃饭，为何心里忧愁？有我不比有十个儿子更好吗？"

9他们在示罗吃喝完了，哈拿就站起来；那时祭司以利坐在耶和华殿的门框旁边的位子上。10哈拿心里愁苦，向耶和华祷告，痛哭。11她许愿说："万军之耶和华啊，你若垂顾你婢女的苦情，眷念我不忘记你的婢女，赐给我一个儿子，我必使他终身归给耶和华，剃头刀必不加在他头上。"

12哈拿在耶和华面前不住地祷告；以利看着她的嘴。13哈拿是在心里说话，只动嘴唇，却听不见她的声音，所以以利以为她喝醉了。14以利对她说："你要醉到几时呢？把你的酒消散了吧！"15哈拿回答说："我主啊，不是这样，我是心里愁苦的妇人，清酒浓酒都没有喝，只是在耶和华面前倾心吐意。16不要把你的婢女看作不正经的女人；我因忧愁过多，哀情太深，所以说话到如今。"17以利回答说："你安然回去吧，愿以色列的神允准你向他所求的！"18哈拿说："愿婢女在你眼前蒙恩。"于是女人走了，吃饭，面上再不带愁容了。

19次日清早，他们起来，在耶和华面前敬拜；然后回去，到了拉玛的家里。以利加拿与妻哈拿同寝，耶和华记念哈拿。20哈拿就怀了孕；日期满足，生了一个儿子，给他起名叫撒母耳，说："这是我从耶和华那里求来的。"

**新约读经**：加拉太书5章1节至26节

基督释放了我们

1基督释放了我们，叫我们得以自由；所以要站立得稳，不要再被奴仆的轭挟制。

2我保罗告诉你们：你们若受割礼，基督就与你们毫无益处了。3我再指着凡受割礼的人确实地说：他是欠著行全律法的债。4你们这要靠律法称义的，是与基督隔绝，从恩典中坠落了。5我们靠着圣灵，凭着信心，等候所盼望的义。6因为在基督耶稣里，受割礼不受割礼全无功效，惟独使人生发仁爱的信心才有功效。

7你们跑得好，有谁拦阻你们不顺从真理呢？8这等劝化不是出于那召你们的。9一点面酵能使全团发起来。10我在主里面相信你们必不怀别样的心；只是那搅扰你们的人，无论是谁，必担当罪责。11弟兄们，我若仍旧传割礼，为何还受逼迫呢？若是这样，那十字架讨厌的地方就没有了。12恨不得那搅乱你们的人把自己割绝了！

13弟兄们，你们蒙召是要得自由，只是不可将你们的自由当作放纵情欲的机会，总要用爱心互相服事。14因为全律法都包在"爱人如己"这一句话之内了。15你们要谨慎，若相咬相吞，只怕要彼此消灭了。

随圣灵而行

16我说，你们当顺着圣灵而行，就不放纵肉体的情欲了。17因为肉体所起的欲望是与圣灵相争的，圣灵所起的欲望是与肉体相争的；这两个是彼此相敌，使你们不能做所愿意做的事。18但你们若被圣灵引导，就不在律法以下。19情欲的事都是显而易见的，就如：淫乱、污秽、邪荡、20拜偶像、邪术、仇恨、争竞、忌恨、恼怒、结党、纷争、异端、21嫉妒、醉酒、荒宴等类；我从前告诉你们，现在又告诉你们，行这样事的人必不能承受神的国。22圣灵所结的果子，就是仁爱、喜乐、和平、忍耐、恩慈、良善、信实、23温柔、节制。这样的事没有律法禁止。24凡属基督耶稣的人，是已经把肉体连肉体的邪情私欲同钉在十字架上了。

25我们若是靠圣灵得生，就当靠圣灵行事。26不要贪图虚名，彼此惹气，互相嫉妒。

**历史与灵修读写**

保罗在《罗马书》6章20、22节所言的自由与奴役，正是此意："你们做罪之奴仆的时候，就不受义的约束。但如今，你们既从罪里得了释放，做了神的奴仆……"让我们用一个图式来表述：

凡从罪中得自由者，已成为义的仆人；凡为罪之奴仆者，则从义中得释放——二者互为倒置……

保罗说："基督使我们因这自由而得自由。"这是属灵的自由，须在灵里得以守护。这并非异教徒所熟知的那种自由——就连异教哲人佩尔修斯也知晓，那种自由远远不够。这是脱离律法的自由，然而其方式却与人间通行的做法截然相反。世人的自由是在律法更改而人心未变的情况下实现的；基督徒的自由则是在人心改变而律法未变的情况下实现的。因此，那曾令自由意志所厌恶的律法，如今因着圣灵将爱浇灌在我们心中（《罗马书》5章5节），便成了令人喜悦之事。保罗教导我们，在这自由中，我们必须坚定屹立、毫不动摇——因为基督为我们成全律法、胜过罪恶，将爱的灵浇灌到所有信他之人的心中，使他们成为义人，成为律法的爱慕者。这并非出于他们自身的功德，乃是基督白白赐予的恩赐，我们也当白白领受。
——马丁·路德

慈悲的神啊，你藉着基督的救赎之死与复活，使我们在自由中得蒙释放。在这自由中，求你教导我们活在受洗时所赐圣灵的果子里，使我们在己身彰显律法的成全，以爱人如己的心事奉邻舍；奉我们的主耶稣基督之名。（1059）
