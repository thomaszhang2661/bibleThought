# 五月二十四日　以斯帖

**诗篇吟唱**

⁴"耶和华啊，求你叫我晓得我身的终结，我的寿数是多少，叫我知道我的生命不长。⁵你使我的年日不多于几掌宽，我一生的年数，在你面前如同无有。人诚然是一口气！⁶人行动实在是虚幻；他们劳碌积蓄，实在是枉然，不知将来有谁收取。⁷主啊，如今我等什么呢？我的指望在乎你。⁸求你救我脱离我一切的过犯，不要使我受愚顽人的羞辱。⁹我因你所行的，哑口无言，不开我的口。¹⁰求你从我身上将你的责打挪去；因你手的打击，我便消灭。¹¹你因人的罪，以责罚惩诫他，你使他所羡慕的被蠹虫吃了；世人诚然是一口气！"

——诗篇三十九篇4—11节

**旧约读经**：传道书一章1—18节

**虚空的虚空**

¹在耶路撒冷作王、大卫的儿子传道者的话。²传道者说："虚空的虚空，虚空的虚空，凡事都是虚空。"³人在日光之下的一切劳碌有什么益处呢？⁴一代过去，一代又来，地却永远长存。⁵日头出来，日头落下，急归所出之地。⁶风往南刮，又向北转，不住地旋转，而且风旋转之后仍归原处。⁷江河都往海里流，海却不满；江河从何处流，仍归还何处。⁸万事令人厌烦，人不能说尽，眼看，看不饱；耳听，听不足。⁹已有的事，后必再有；已行的事，后必再行；日光之下并无新事。¹⁰岂有一件事人能指着说这是新的？哪知，在我们以前的世代早已有了。¹¹已过的世代，无人记念；将来的世代，后来的人也不记念。

**智慧的虚空**

¹²我传道者在耶路撒冷作过以色列的王。¹³我专心用智慧寻求、查究天下所做的一切事，乃知神叫世人所经练的是极重的劳苦。¹⁴我见日光之下所做的一切事，都是虚空，都是捕风。¹⁵弯曲的，不能变直；缺少的，不能足数。¹⁶我心里说："我得了大智慧，胜过我以前在耶路撒冷众人，而且我心多经历智慧和知识的事。"¹⁷我又专心察明智慧、狂妄和愚昧，乃知这也是捕风。¹⁸因为多有智慧，就多有愁烦；加增知识的，就加增忧伤。

**新约读经**：约翰福音六章60—71节

**永生的话**

⁶⁰他的门徒中有许多人听见了，就说："这话甚难，谁能听呢？"⁶¹耶稣心里知道门徒为这话私下议论，就问他们说："这话是叫你们跌倒吗？⁶²倘或你们看见人子升到他原来所在之处，怎么样呢？⁶³叫人活着的乃是灵，肉体是无益的；我对你们所说的话，就是灵，就是生命。⁶⁴只是你们中间有不信的人。"（耶稣从起初就知道谁不信他，谁要卖他。）⁶⁵他又说："所以我对你们说过，若不是蒙我父的恩赐，没有人能到我这里来。"

⁶⁶从此，他门徒中多有退去的，不再和他同行。⁶⁷耶稣就对那十二个门徒说："你们也要去吗？"⁶⁸西门·彼得回答说："主啊，你有永生之道，我们还归从谁呢？⁶⁹我们已经信了，又知道你是神的圣者。"⁷⁰耶稣说："我不是拣选了你们十二个门徒吗？但你们中间有一个是魔鬼。"⁷¹他是指着加略人西门的儿子犹大说的；他本是十二个门徒里的一个，后来要卖耶稣的。

**历史与灵修读写**

穿戴体面的衣服、寻求正当的生计，是生活的必需，并非罪恶。然而，孩子们内心必须认清一件事：在这悲惨的今生，若不追求超过遮体御寒、填饱肚腹所需之外的更多装扮与财物，实在难以过活，甚至难以起步——这是何等可惜的事。因此，孩子们必须被教导，为了更好的事、为了避免更坏的事，要违背自己的心意，去做世界所要求的事。孩子们必须忍受愚昧和那类的恶。

王后以斯帖戴着皇冠，但她对神说："你知道，我戴在头上的，象征我高位的冠冕，并不使我喜悦，我憎恶它如同污秽的月经布；在我独处时，我从不戴它，只在不得不见众人时才戴上。"一个有这种心志的人，穿戴装饰而没有危险；因为他穿戴，却又好像没有穿戴；他跳舞，却又好像没有跳舞；他活得好，却又好像没有活得好。这样的灵魂就是基督神圣隐秘的新妇。但这样的灵魂是稀少的，因为不以华丽的装扮和炫耀为乐，是很难做到的事。

——马丁·路德

**今日祷文**

神啊，你不仅以美貌与优雅，更以信心与智慧恩待你的仆人——王后以斯帖。求你赐恩，使我们也能将你慷慨赐给我们的一切才能，用于荣耀你威严的名、造益你的百姓；藉着你在我们里面的工作，使我们成为受压迫者的倡导者、软弱者的捍卫者，在那为我们代求的大祭司耶稣基督里保守我们的信心；耶稣基督与你和圣灵，同活同治，是独一真神，从今直到永远。阿们。

**以斯帖**

以斯帖是圣经中以她命名之书的女主角。她的希伯来名字是哈大沙，意思是"桃金娘"。她的美丽、魅力与勇气，使她在成为亚哈随鲁王王后时大有助益。凭借这一地位，她得以拯救她的百姓免遭灭族之祸——此祸乃王的首席谋士哈曼所策划（以斯帖记二章19节—四章17节）。以斯帖努力揭发这个阴谋，最终使哈曼吊死在他为末底改——她的叔叔兼监护人——所建造的那副绞架上。此后，王任命末底改为首相，接替哈曼之职。这个故事显明，神如何为他百姓出手干预，拯救他们脱离恶势力；神在此藉着以斯帖保全了旧约时代的子民，弥赛亚将从他们而来。
