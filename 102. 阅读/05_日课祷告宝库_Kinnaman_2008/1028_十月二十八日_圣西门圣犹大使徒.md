# 十月二十八日　圣西门和圣犹大，使徒

**诗篇吟唱**

若不是耶和华建造房屋，建造的人就枉然劳力；若不是耶和华看守城池，看守的人就枉然儆醒。
你们清晨早起，夜晚安歇，吃劳碌得来的饭，本是枉然；惟有耶和华所亲爱的，必叫他安然睡觉。
儿女是耶和华所赐的产业；所怀的胎，是他所给的赏赐。
少年时所生的儿女，好像勇士手中的箭。
箭袋充满的人，便为有福；他们在城门口和仇敌说话的时候，必不至于羞愧。

——诗篇一百二十七篇

**旧约读经**：申命记三十章1—20节

**悔改与赦免**

我所陈明在你面前的这一切祝福和咒诅，都临到你身上，你在耶和华你神驱逐你到的万国中，若放在心上，又归向耶和华你神，照着我今日所吩咐你的话，尽心尽性听从他的话，那时，耶和华你神必怜悯你，使你被掳的人归回，耶和华你神要从万民中将你们召聚回来。你们若被赶散在天涯，耶和华你神也必从那里将你招聚来；必从那里将你取回。耶和华你神必领你们到你列祖所承受的地，也要赐你们承受那地；又必赐福给你，使你比你列祖更多更盛。耶和华你神必为你行割礼，使你心里和你后裔心里受割礼，好叫你尽心尽性爱耶和华你的神，使你可以活着。耶和华你神必将这一切咒诅加在你仇敌，和那些恨你逼迫你的人身上。你必归向耶和华，听从他的话，照着我今日吩咐你的一切诫命行。耶和华你神必使你手所办的一切事，并你身所生的，牲畜所下的，地所产的，都有余；因为耶和华必再喜悦你，降福给你，正如从前喜悦你列祖一样——若是你听从耶和华你神的话，谨守他的诫命和律例，就是这律法书上所写的，并且你是尽心尽性归向耶和华你的神。

**生死的抉择**

因为这诫命，就是我今日所吩咐你的，不是你难以做到的，也不是离你遥远的。这不是在天上，使你以为：谁能为我们上天取下来，使我们听见，可以遵行呢？也不是在海外，使你以为：谁能为我们过海取来，使我们听见，可以遵行呢？这话却离你甚近，就在你口中，在你心里，使你可以遵行。

看哪，我今日将生死、祸福，陈明在你面前。你若遵从耶和华你神的诫命，就是我今日所吩咐你的，爱耶和华你的神，遵行他的道，谨守他的诫命、律例、典章，你就可以存活，人数增多，耶和华你神也必在你所要进去得的地上赐福给你。你若心里偏离，不肯听从，却被引诱去敬拜事奉别神，我今日告诉你们，你们必定灭亡，在你过约旦河、进去得的地上，你的日子必不长久。我今日呼天唤地向你作见证，我将生死、祸福，陈明在你面前，所以你要拣选生命，使你和你的后裔都得存活；且爱耶和华你的神，听从他的话，专靠他；因为他是你的生命，你的日子长久也在乎他；这样，你就可以在耶和华向你列祖亚伯拉罕、以撒、雅各起誓应许所赐的地上居住。

**新约读经**：马太福音十九章1—15节

**论离婚**

耶稣说完了这些话，就离开加利利，来到犹太的境界约旦河外。有许多人跟着他，他就在那里把他们的病人治好了。

有法利赛人来试探耶稣，说：人无论什么缘故，都可以休妻吗？耶稣回答说：那起初造人的，是造男造女，并且说：因此，人要离开父母，与妻子联合，二人成为一体。既然如此，夫妻不再是两个人，乃是一体的了；所以神所配合的，人不可分开。法利赛人说：这样，摩西为什么吩咐给妻子休书，打发她离开呢？耶稣说：摩西因为你们心硬，所以许你们休妻，但起初并不是这样。我告诉你们，凡休妻另娶的，若不是为淫乱的缘故，就是犯奸淫了。

门徒对耶稣说：人和妻子既是这样，倒不如不娶。耶稣说：这话不是人都能领受的，惟独赐给谁，谁才能领受。因为有生来是阉人，也有被人阉的，也有为天国的缘故，自阉的。这话谁能领受，就可以领受。

**让孩子到我这里来**

那时，有人带着小孩子来见耶稣，要耶稣给他们按手祷告，门徒就责备那些人。耶稣说：让小孩子来，不要禁止他们，因为在天国的，正是这样的人。耶稣给他们按了手，就离开那地方去了。

**历史与灵修读写**

《反驳书》（写来反对《奥格斯堡信条》的文件）也说修士更丰富地赚取永生，并引用圣经："凡离开……的人"等（马太福音十九29）。于是这里也为人为的宗教礼仪主张完全性。但这段圣经毫无支持修道院生活之处。基督并不是说离开父母、妻子和兄弟姊妹是一件必须做的工，因为它赚取罪的赦免和永生。事实上，这样的离开是被咒诅的。任何离开父母或妻子，以为凭此工赚取罪的赦免或永生的人，都是在羞辱基督。

有两种离开。其一是没有呼召、没有神命令的离开，基督不赞许这种离开（马太福音十五9）。我们所选择的工是无用的侍奉。当基督谈到离开妻子和孩子时，他清楚地表明他不赞许这种离开。我们知道，神的诫命禁止离开妻子和孩子。其二是神命令的离开，即当权势或暴政迫使我们要么离开、要么否认福音时。这时我们被命令忍受伤害，宁可让财富、妻子、孩子，甚至生命被夺去，也不妥协。基督赞许这种离开，所以他加上了"为福音的缘故"（马可福音十29）。他这样说，是要说明他所说的，不是那些伤害妻子和孩子的人，而是因福音的认信而忍受伤害的人。为着福音的缘故，我们甚至当离弃自己的身体。若有人认为自杀和无神命令地离弃身体是侍奉神，这是荒谬的。同样，无神命令地离弃财产、朋友、妻子和孩子，认为是侍奉神，也是荒谬的。

——《奥格斯堡信条辩护》第二十七（十三）章40—42节

**圣西门和圣犹大，使徒**

在十二使徒的名单中（马太福音十2—4；马可福音三16—19；路加福音六14—16；使徒行传一13），第十和第十一位分别是奋锐党的西门（或称"迦南人"）和犹大（或"雅各的儿子犹大"，而不是加略人犹大），后者也明显地以"达太"为人所知。根据早期基督徒的传统，西门和犹大一同作为宣教士前往波斯，在那里殉道。至少有一部分原因，这两位使徒被纪念在同一天。西门在新约中除了出现在十二使徒名单中之外，没有被提及。因此他被纪念和尊崇，是为了他的职分，由此他在永恒中，如同在他地上的生命和事奉中，以我们的主耶稣基督的名义和代替他而站在我们面前。我们感谢神呼召和差遣西门以及犹大和所有使徒，传讲和教导圣福音，宣告悔改和罪的赦免，并以父、子、圣灵的名施洗（约翰福音四1—2；马太福音十；二十八16—20；路加福音二十四46—49）。

犹大出现在约翰福音（十四22）中，在我们主受难的那夜，他问耶稣：你怎么向门徒显现，却不向世人显现呢？耶稣对这个问题的回答，对这个节日颇具意义："人若爱我，就必遵守我的道，我父也必爱他，并且我们要到他那里去，与他同住"（约翰福音十四23）。犹大和西门肯定以生死彰显了他们对耶稣的爱和对他话语的信心。我们不仅借着他们的榜样在基督徒的信仰和生活中得到坚固，更因主忠实地保守了他对他们所许下的带他们归家与自己同在的应许而深受鼓励。他们在那里永远与他同住，我们也将在某日加入他们。

**路德宗信条建议阅读**

《奥格斯堡信条辩护》第二十四（十二）章1—8节

1他们接受第十五条款之首部，即我们说：我们应当遵守能遵守而不为犯罪，并同时有助于教会中安静及优良次序的教会礼仪。他们全然咒诅第二部，即我们说：设立属人遗传为缓和上帝、赚得恩典、并为罪补赎，乃与福音相反。2我们虽然在信条第二十六条内，较长地讨论了传统的问题，在此我们仍须重复一些事。

3我们原希望，对敌根据其他缘由辩护属人的传统。我们未曾想到，他们会实际地咒诅我们不藉遵守人的传统，配得恩典或罪的赦免。他们既然咒诅了此道理，我们便有一简明的事例。4我们对敌对此明然地模仿犹太人的错误；他们公然以鬼魔的道理代替福音。圣经称传统为“鬼魔的道理”（提前四1）， 即某人教导：宗教礼仪帮助人获得恩典及罪之赦免。此异端蒙蔽福音、基督的祝福，及信之义。5福音教导我们：我们藉信，因基督之故，白白领受罪的赦免，并与上帝和好。另一方面，我们对敌设立这些传统为其他的中保，藉着它们寻获罪的赦免，并缓和上帝的怒气。但是基督明白的说（太十五9）：“他们将人的吩咐，当作道理教导人，所以拜我也是枉然。”

6我们前面曾详细表示：人藉信称义，使他们有一位恩惠的上帝，不是因着行为，而是白白因基督之故。这确是福音的教训，因为保罗明白地教导（弗二8）：“你们得救是本乎恩，也因着信；这并不是出于自己，乃是上帝所赐的，”并且不由于人。他们说，人藉这些属人的遵守，配得罪的赦免。7这岂不算另设立一位称义者和中保代替基督呢？8保罗对加拉太人（五4）说：“你们这要靠律法称义的，是与基督隔绝。”就是说：你们若想藉着遵守律法，配得在上帝面前算为义，那么，基督便与你们无益，因为谁若相信自己藉遵守律法为义，他为何需要基督呢？
