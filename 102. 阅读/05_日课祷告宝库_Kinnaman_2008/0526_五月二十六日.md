# 五月二十六日

**诗篇吟唱**

¹神啊，求你怜恤我，因为人要吞吃我，整天攻击欺压我。²我的仇敌整天要吞吃我，因为攻击我的人甚多，都是骄傲自大的。³我惧怕的时候，要倚靠你。⁴我倚靠神，我要称赞他的话；我倚靠神，必不惧怕；血气之辈能把我怎么样呢？

**旧约读经**：传道书3章1—22节

**万事各有其时**

¹凡事都有定期，天下万务都有定时：²生有时，死有时；栽种有时，拔出所栽种的也有时；³杀戮有时，医治有时；拆毁有时，建造有时；⁴哭有时，笑有时；哀恸有时，跳舞有时；⁵抛掷石头有时，堆聚石头有时；怀抱有时，不怀抱有时；⁶寻找有时，失去有时；保存有时，舍弃有时；⁷撕裂有时，缝补有时；静默有时，言语有时；⁸相爱有时，恨恶有时；争战有时，和好有时。

**神所赐的本分**

⁹这样，做事的人在他的劳碌上有什么益处呢？¹⁰我见神赐给世人所当做的事。¹¹神造万物，各按其时而美好；又将永恒安置在世人心里，然而神从始至终的作为，人不能参透。¹²我知道世人，莫强如终身喜乐行善；¹³并且人人吃喝，在日光之下的一切劳碌中享受益处，这也是神的恩赐。¹⁴我知道神一切所做的都必永存，无所增添，无所减损；神这样行，是要人在他面前存敬畏的心。¹⁵现今的事，早先就有了；将来的事，早已也有了；神使已过的事重新再来。

**尘归尘，土归土**

¹⁶我又见日光之下，在审判之处有奸恶，在公义之处也有奸恶。¹⁷我心里说：公义人和奸恶人，神必审判，因为在那里各事都有定时，各工都有审判。¹⁸我心里说：这是为世人的缘故，是神要试验他们，使他们看出自己不过像兽一样。¹⁹因为世人遭遇的，兽也遭遇，所遭遇的都是一样：这个怎样死，那个也怎样死，气息都是一样，人不胜于兽，都是虚空。²⁰都归一处，都是出于尘土，也都归于尘土。²¹谁知道人的灵是往上升，兽的魂是下入地呢？²²我所见的，以为人莫强如在他自己的事业上喜乐，因为这是他的本分；他以后的事，谁能使他回来见呢？

**新约读经**：约翰福音7章14—31节

¹⁴到了节期中间，耶稣上圣殿去教训人。¹⁵犹太人就希奇，说："这个人没有学过，怎么明白书呢？"¹⁶耶稣说："我的教训不是我自己的，乃是那差我来者的。¹⁷人若立志遵着他的旨意行，就必晓得这教训或是出于神，或是我凭着自己说的。¹⁸凭着自己说的，是求自己的荣耀；惟有求那差他来者的荣耀，这人是真实的，在他心里没有不义。¹⁹摩西岂不是传律法给你们吗？你们却没有人守律法。你们为什么想要杀我呢？"²⁰众人回答说："你是被鬼附着了！谁想要杀你呢？"²¹耶稣说："我做了一件事，你们都以为希奇。²²摩西传割礼给你们（其实不是摩西传的，乃是列祖传的），你们因此在安息日给人行割礼。²³若在安息日受割礼，免得违背摩西的律法，我在安息日叫一个人全然好了，你们就向我生气吗？²⁴不可按外貌断定是非，总要按公平断定是非。"

**这人莫非是基督吗？**

²⁵耶路撒冷人中有的说："这不是他们想要杀的人吗？²⁶你看，他还明明地讲道，他们也不向他说什么；难道官长真知道这是基督吗？²⁷然而，我们知道这个人是哪里来的；只是基督来的时候，没有人知道他是哪里来的。"²⁸耶稣在殿里教训人，大声说："你们也知道我，也知道我是哪里来的；我来，并不是由于自己，但那差我来的是真实的，你们却不认识他。²⁹我认识他，因为我是从他来的，他也是差我来的。"³⁰他们就想要拿他，只是没有人下手拿他，因为他的时候还没有到。³¹众人中间有好些人信了他，说："基督来的时候，他所行的神迹，岂能比这人所行的更多吗？"

**历史与灵修读写**

圣经中有许多地方告诉我们，在安息日不可做任何工作。我们当安息，停止一切招致汗水与劳苦的事务。因为他在《出埃及记》中说："六年你要耕种你的地，收藏土产；但第七年要叫地歇息，不耕不种。"【出埃及记23:10–11。另参利未记25:2–4】。他所释放的，并非那不知劳作为何物的土地，这律法也不是为土地而设的。它是给那些拥有这地的人的。他使地得享安息，为的是叫他们不在其上劳作。这一切，以及许多其他方式，都是预表我们与基督同享的节期——在这节期中，那些在敬畏神中生活的人，将奔向那在圣洁中完全自由的境界，跑向圣灵那最丰盛的恩典。这在摩西的诫命本身中已清楚可见。经上写着："你弟兄中若有一个希伯来男人或希伯来女人被卖给你，他服事你六年，第七年你要放他自由离开你。"【申命记15:12】。我们起初是罪的奴仆，以某种方式，因以恶为乐而将自己卖给了魔鬼。但如今，我们藉着信心在基督里被称义，必将昇入真实而圣洁的安息日遵守之中，披戴那藉恩典而来的自由，以神的美善得荣耀。

——亚历山大的区利罗

**圣诗**

永恒之主，你的智慧洞见，
探透人生一切苦难；
你知我们的哀恸，你听我们的叹息——
求你施怜悯，拭干我们泪眼。

从一切凶险，你带出大善；
十字架下，我们安然站立。
如今人生之路，我们只见昏暗，
终有一日，我们将要面对面地相遇。

——《上帝所定皆美善》（路德宗崇拜书 762:3）

**今日祷文**

神啊，你曾藉着赐下圣灵的光，教导你忠信子民的心。求你在我们这世代，藉着同一圣灵，使我们在万事上有正确的理解，且永远欢欣在他圣洁的安慰中；藉着你的儿子，我们的主耶稣基督，他与你和圣灵，同活同治，是独一真神，从今直到永远。阿们。
