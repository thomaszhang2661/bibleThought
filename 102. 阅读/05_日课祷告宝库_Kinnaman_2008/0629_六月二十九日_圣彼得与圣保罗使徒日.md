# 六月二十九日　圣彼得与圣保罗使徒日

**诗篇吟唱**

我的心哪，你要称颂耶和华！
凡在我里面的，也要称颂他的圣名！
我的心哪，你要称颂耶和华！
不可忘记他的一切恩惠！
他赦免你的一切罪孽，
医治你的一切疾病；
他救赎你的命脱离死亡，
以仁爱和慈悲为你的冠冕；
他用美物使你所愿的得以知足，
以致你如鹰返老还童。
耶和华施行公义，
为一切受屈的人伸冤。
他将自己的法则指示摩西，
将自己的作为告诉以色列人。
耶和华有怜悯，有恩典，
不轻易发怒，且有丰盛的慈爱。
他不长久责备，
也不永远怀怒。
他没有按我们的罪过待我们，
也没有照我们的罪孽报应我们。
天离地何等高，
他的慈爱向敬畏他的人也是何等大！
东离西有多远，
他叫我们的过犯离我们也有多远！

——诗篇一〇三篇一—十二节

**旧约读经**：约书亚记四章一—二十四节

**约旦河中的十二块纪念石**

以色列众人全部过了约旦河之后，耶和华吩咐约书亚说："你们当从百姓中选出十二人，每支派一人，吩咐他们说：'你们从这里，从约旦河中、从祭司脚站的地方，取出十二块石头，带过去，放在你们今夜住宿的地方。'"约书亚就召了那十二个人，就是他从以色列人中所预备的，每支派一人；约书亚对他们说："你们走在耶和华你们　神的约柜前，进入约旦河中，你们各人照以色列人支派的数目，把石头扛在肩上，这要在你们中间作为证据。日后，你们的子孙问你们说：'这些石头是什么意思？'你们就告诉他们：以色列人过这约旦河，当时耶和华你们的约柜的水断绝了；这些石头要作以色列人永远的纪念。'"

以色列人照耶和华所吩咐约书亚的话，就这样行了，按着以色列人支派的数目，从约旦河中取了十二块石头，正如耶和华所吩咐约书亚的；他们把石头带过去，到他们住宿的地方，就把石头放在那里。约书亚又在约旦河中立了十二块石头，在抬约柜的祭司脚站的地方；直到今日，那石头还在那里。抬约柜的祭司在约旦河中站立，等到耶和华吩咐约书亚传告百姓的事都办完了，是照摩西所吩咐约书亚一切的话。百姓就急速过去了。众百姓都过完了，耶和华的约柜和祭司也在百姓面前过去。流便人、迦得人和玛拿西半支派的人，都照摩西所吩咐他们的，带着武器在以色列人前面过去，约有四万人，都准备好打仗的，在耶利哥的平原，在耶和华面前过去。那日，耶和华使约书亚在以色列众人眼前尊大，以色列人敬畏他，像从前敬畏摩西一样，终约书亚的一生都是这样。

耶和华吩咐约书亚说："你吩咐抬约柜的祭司从约旦河里上来。"约书亚就吩咐祭司说："从约旦河里上来！"抬耶和华约柜的祭司从约旦河里上来；祭司的脚刚踏上旱地，约旦河的水就还原，仍旧涨过两岸。

正月初十日，百姓从约旦河里上来，就在耶利哥东边的吉甲安营。约书亚将他们从约旦河中取来的那十二块石头立在吉甲，对以色列人说："日后，你们的子孙问他们的父亲说：'这些石头是什么意思？'你们就告诉他们说：'以色列人过这约旦河是走干地。因为耶和华你们的　神使约旦河的水在你们前面干了，等你们过来，正像耶和华你们的　神从前在我们前面使红海干了，等我们过来一样；为要使地上万民都知道耶和华的手是大能的，也为要使你们永远敬畏耶和华你们的　神。'"

**新约读经**：使徒行传九章二十三—四十三节

**扫罗从大马士革逃脱**

过了好些日子，犹太人商议要杀扫罗，但扫罗知道了他们所设的计谋。他们昼夜把守城门，要杀他。他的门徒就在夜间用筐子把他从城墙上缒下去。

**扫罗在耶路撒冷**

扫罗到了耶路撒冷，想与门徒相交；但大家都怕他，不信他是门徒。惟有巴拿巴接待他，领他去见使徒，把他在路上怎么看见主，主怎么向他说话，他在大马士革怎么奉耶稣的名放胆传道，都述说出来。于是扫罗在耶路撒冷和门徒自由来往，奉主的名放胆传道，又和说希腊话的犹太人讲论辩驳；他们却想法子要杀他。弟兄们知道了，就送他下凯撒利亚，打发他往大数去。

那时，犹太、加利利、撒马利亚各处的教会都得平安，被建立；凡事敬畏主，蒙圣灵的安慰，人数就增多了。

**医治以尼雅**

彼得周游各处，到了住在吕大的圣徒那里，遇见一个人，名叫以尼雅，他害瘫痪病，已经卧床八年。彼得对他说："以尼雅，耶稣基督医治你了，起来，铺好你的床！"他立刻起来了。住在吕大和沙仑的人都看见了他，就归服主。

**多加从死里复活**

在约帕有一个门徒，名叫大比大——翻希腊话，就是多加；他满心做好事，多施周济。当时他患病死了，有人把他洗了，停在楼上。吕大与约帕相近；门徒听见彼得在那里，就打发两个人去见他，央求他说："快来，不要耽延。"彼得就起身和他们同去；到了，便有人领他上楼；众寡妇都站在彼得旁边哭，拿多加与他们同在时所做的里衣外衣给他看。彼得叫她们都出去，就跪下祷告，转身对着那身体说："大比大，起来！"她就睁开眼睛，见了彼得，便坐起来。彼得伸手扶她起来，叫众圣徒和寡妇进来，把多加活活地交给他们。这事传遍了约帕，就有许多人信了主。此后，彼得在约帕住了多日，寄居在一个硝皮匠西门的家里。

**历史与灵修读写**

圣餐设立言词固有、简明、自然的含义教导我们：基督自己在圣餐的庆典中，以其神性与人性亲自与我们同在，并来到我们这里，为要抓住我们（腓三12），尽可能亲密地将我们与他联结。这带来最甜蜜的安慰。因为基督——既是　神又是人——必须抓住我们，使他与我们之间得以联合。然而我们被罪担所压、被软弱所困，尚且不能进入天上的密室（西二18），穿越到荣耀中的他那里。因此，他亲自来到我们这里，以那使他成为我们弟兄的本性抓住我们。又因我们今生的软弱无法承受他威严的荣耀（太七12及以后；徒九3及以后），所以他的身体与宝血就在饼与酒之下临在、分施并被领受。他也不愿我们在天门之前徘徊流连，不确定应在天上哪处寻找具有人性的基督，或能否找见他；乃是在圣餐中，他亲自临在于外在的庆典，并借有形的记号指明他愿以身体与宝血临在之处，我们便可在那里安全地寻找他，必定地找到他；因为他在那里亲自借着服侍的职分，将他的身体与宝血分施给领受者。这些最甜蜜、最必要的安慰，若是基督身体与宝血实质性的临在、分施与领受从圣餐中被移除，便会被彻底夺去。

——马丁·肯尼茨

**圣诗**

我们赞美你，因着圣彼得；
我们赞美你，因着圣保罗；
他们教导犹太人与外邦人，
基督就是一切。
他们献身于十字架与刀剑，
见证你国度的降临；
哦，　神啊，这两位使徒，
经由殉道到达生命。

——《路德宗崇拜书》五一八首第十九节

**圣彼得与圣保罗使徒日**

圣彼得与圣保罗的节日，可能是所有圣徒纪念日中历史最悠久的（约可追溯至三世纪中叶）。早期传统认为，这两位新约教会的柱石，在尼禄逼迫期间同日在罗马殉道。除了这次对他们死亡的联合纪念之外，两位使徒还分别设有纪念日：彼得的纪念日为一月十八日，纪念他承认耶稣是基督（马太福音十六13—16）；保罗的纪念日为一月二十五日，纪念他的悔改（使徒行传九1—19）。

新约对两位使徒均有大量记载。彼得从耶稣事工伊始便跟随他，并在门徒中担任领袖。尽管他信心坚定，圣经也记录了他的一些失败，例如责备耶稣（马太福音十六21—23）以及三次不认主（马太福音二十六69—75）。耶稣升天之后，彼得继续在教会中担当领袖（使徒行传一15；二14；十五7）。

保罗，也称扫罗，是一位虔诚的犹太人，最初以逼迫教会为业。经历了奇妙的悔改——复活的基督亲自向他显现——之后，保罗成为宣讲　神恩典的大能传道人。在他的三次宣教旅程中（使徒行传十三—十四；十六—十八；十八—二十一），保罗走遍了今日土耳其与希腊各地。新约关于他生平的记述，以保罗在罗马被软禁作结（使徒行传二十八16），不过传统认为他其后又去了西班牙，然后重返罗马。
