# 六月五日　美因茨的卜尼法斯，向日耳曼人传教的宣教士

**诗篇吟唱**

因为他必救拔困苦人呼求他的，拯救穷乏人，和没有人帮助的人。他必怜恤贫寒和穷乏的人，拯救穷乏人的性命。他要救赎他们脱离欺压和强暴，他们的血在他眼中看为宝贵。他们要活着，并且将示巴的金子献给他，人还要为他常常祷告，终日称颂他。在地上必有丰盛的粮食，在山顶上也必有粮食摇动，如黎巴嫩的果子；城里的居民必兴旺如地上的草。他的名必存到永远，他的名必在日光之前，被人传流；人必因他蒙福，万族必称他有福！

耶和华神——以色列的神——是独行奇事的，是应当称颂的！他荣耀的名也当称颂，直到永远；愿他的荣耀充满全地！阿们，阿们！

——诗篇七十二篇12—19节

**旧约读经**：箴言一章8—33节

**罪人的引诱**

我儿，要听你父亲的训诲，不可离弃你母亲的法则；因为这要作为你头上的华冠，你项上的金链。我儿，恶人若引诱你，你不可随从。他们若说："你与我们同去；我们要埋伏流人之血，要蹲伏害无辜之人。我们好像阴间，把他们活活地吞下，他们如同下坑的人，被我们囫囵吞了；我们必得各样宝贝，将被掳来的物装满我们的房屋；你与我们大家共分，我们共用一个囊袋"；我儿，不要与他们同行一道；禁止你脚走他们的路。因为他们的脚奔跑到恶事上，急速流人的血。好鸟眼前张开网罗，枉然！这些人埋伏，要害自己的命；蹲伏，要取自己的性命。凡贪恋财利的人，所行之道都是如此，而且要丧失自己的性命。

**智慧的呼声**

智慧在街市上呼喊，在宽阔处发声，在热闹街头喊叫，在城门口、在城中发出言语，说："你们愚昧人喜爱愚昧，亵慢人喜欢亵慢，愚顽人恨恶知识，要到几时呢？你们当因我的责备回转；我要将我的灵浇灌你们，将我的话指示你们。我呼唤你们，你们却推却；我伸手，无人理会；你们忽视我一切的劝戒，不肯受我的责备。你们遭难，我就发笑；你们恐惧，我就嗤笑；你们的恐惧临到，好像狂风；你们的灾难来到，如同旋风；你们遭遇忧愁苦难的时候，那时他们必呼求我，我却不答应；必恳切寻找我，却寻不到。因为他们恨恶知识，不喜爱敬畏耶和华，不肯受我的劝戒，藐视我一切的责备，所以必吃自结之果，充满自设之谋。愚昧人背道，必杀自身；愚顽人安逸，必害自己。惟有听从我的，必安然居住，得享安静，不怕灾祸。"

**新约读经**：约翰福音十一章17—37节

**我是复活，我是生命**

耶稣到了，就知道拉撒路在坟墓里已经四天了。伯大尼离耶路撒冷不远，约有六里路。有许多犹太人来看马大和马利亚，要为她们的兄弟安慰她们。马大听见耶稣来了，就出去迎接他；马利亚却仍然坐在家里。马大对耶稣说："主啊，你若早在这里，我兄弟必不死。就是现在，我也知道你无论向神求什么，神必赐给你。"耶稣说："你兄弟必然复活。"马大说："我知道在末日复活的时候，他必复活。"耶稣对她说："复活在我，生命也在我；信我的人，虽然死了，也必复活；凡活着信我的人，必永远不死。你信这话吗？"马大说："主啊，是的，我信你是基督，是神的儿子，就是那要临到世界的。"

**耶稣哭泣**

马大说了这话，就回去暗暗地叫她妹子马利亚说："夫子来了，叫你。"马利亚听见，就急忙起来，出去见耶稣。那时耶稣还没有进入村子，仍在马大迎接他的地方。那些同马利亚在家里安慰她的犹太人，见她急忙起来出去，就跟着她，以为她要往坟墓那里去哭。马利亚到了耶稣那里，看见他，就俯伏在他脚前，说："主啊，你若早在这里，我兄弟必不死。"耶稣看见她哭，并看见与她同来的犹太人也哭，就心里悲叹，又甚忧愁，便说："你们把他安放在哪里？"他们回答说："请主来看。"耶稣哭了。犹太人就说："你看他爱这人何等深！"其中有人说："他既然开了瞎子的眼睛，岂不能叫这人不死吗？"

**历史与灵修读写**

看哪，你的救赎主用道成肉身的香膏涂抹了你瞎眼的双目，使你这原本不能仰望神荣耀、神那隐秘威严的人，能够以人的形象来仰望神；仰望他，便认识他；认识他，便爱他；爱他，便尽一切力量竭力寻求他的荣耀。他成为肉身，是为了将你召回灵里的事；他成为你变化的参与者，是为了使你成为他不变的参与者。他屈尊降卑到你的卑微，是为了将你提升到他崇高的荣耀……在伯大尼——意为"顺从之家"——他叫拉撒路从死里复活，显明凡藉着良善意志的殷勤努力，向世界死去、安息在顺从怀中的人，都要被他复活得永生。当他将自己的身体和血赐给门徒，在神秘的晚餐中时，他谦卑地洗了他们的脚，教导我们那圣洁的奥秘当以纯洁的行为和虔诚谦卑的心灵来领受。当他将要因圣洁复活的荣耀而被高举时，他却忍受了出卖者的嘲弄、侮辱的残酷、十字架的羞耻、苦胆的苦涩，以及最终的死亡，从而警示他的仆人们：那些渴望在死后得着荣耀的人，必须在今生忍受困苦劳役和恶人的压迫，不仅毫无怨言，更要以爱心、渴望，并欢然迎接——在今世，这是为着那永恒赏赐的缘故，是艰难的。

——坎特伯雷的安瑟伦

**圣诗**

耶稣已来！看看，死亡的囚锁被挣断！死亡的堡垒之墙，如今倒塌、消失。
看他突破，发出如雷的声音！他从我们的罪债和恐惧中释放我们，
将我们从羞耻中提升到荣耀的地位。
耶稣已来！听，神雷霆般的滚动！

——《路德宗崇拜书》533:2

**今日祷文**

全能的神，你呼召卜尼法斯在日耳曼地作见证并殉道，藉着他的劳苦与苦难，你兴起了一批属你自己的百姓。求你将圣灵浇灌在四方各地你的教会上，使众多人的服侍与牺牲，得以荣耀你的圣名，扩展你的国度；奉我们的主耶稣基督，他与你和圣灵同活同治，是独一真神，从今时直到永远。（1035）

**美因茨的卜尼法斯，向日耳曼人传教的宣教士**

卜尼法斯约生于七世纪末的英格兰。他虽在英格兰受教育、成为修士并被按立为长老，却受他人榜样的激励，立志成为宣教士。公元719年，他奉教皇委任，前往日耳曼工作，从此全身心投入于在黑森、图林根和巴伐利亚植立教会、组织教会、整顿修道院。成为总主教后，卜尼法斯于公元743年被指派至美因茨主教区。十年后，他辞去职位，往荷兰从事宣教工作。公元754年6月5日，他在等候一群受洗归信者来接受坚信礼时，与同伴一道被一群异教徒杀害。卜尼法斯被称为日耳曼人的宗徒与宣教士。
