# 八月二十四日　圣巴多罗买使徒

**诗篇吟唱**

耶和华是我的亮光，是我的拯救，我还怕谁呢？耶和华是我性命的保障，我还惧谁呢？

耶和华啊，我用声音呼喊，求你垂听，并且怜恤我，应允我。

你说："你们当寻求我的面。"那时我心向你说："耶和华啊，你的面我正要寻求。"

求你不要向我掩面，不要发怒赶开你的仆人。你向来是帮助我的，求你不要丢掉我，也不要离弃我，我得救的神啊！

我的父母离弃了我，耶和华却收留我。

耶和华啊，求你将你的道教训我，因为有仇敌窥探我，所以求你引导我走平坦的路。

——诗篇二十七篇1、7—11节

**旧约读经**：列王纪上五章1—18节

**建殿的准备**

推罗王希兰听见人膏所罗门接续他父亲作王，就差遣臣仆往所罗门那里去，因为希兰素来爱戴大卫。所罗门差人去见希兰，说："你知道我父亲大卫因为四围的战争，不能为耶和华他神的名建造殿宇，直等耶和华使他的仇敌都服在他脚下。但现在耶和华我的神使我四境平安，没有仇敌，也没有灾祸临近。我想为耶和华我神的名建造殿宇，正如耶和华对我父亲大卫所说：'我必使你儿子接续你坐你的位，他必为我名建造殿宇。'所以你可以吩咐人为我从利巴嫩砍伐香柏树，我的仆人必与你的仆人同工，你的仆人雇价，我必照你所要的给你，因你知道我们中间没有人善于砍伐树木，像西顿人一样。"

希兰听见所罗门的话就大大喜欢，说："耶和华今日是应当称颂的，因他赐给大卫一个有智慧的儿子，治理这众多的民。"希兰打发人去见所罗门，说："你所打发来告诉我的话，我都听见了。论到香柏木和松木，我必照你所愿意的而行。我的仆人必将这些木料从利巴嫩运到海里，我也必在海里扎成筏子，运到你所指定我的地方，在那里拆开，你便可以取去。你也要满足我所愿的，就是给我的家室食物。"这样，希兰将所罗门所要的香柏木和松木都给了他。所罗门给希兰二万歌珥小麦，作他家室的食物，又给二万歌珥精制的油。所罗门每年如此给希兰。耶和华照着所应许的赐智慧给所罗门，希兰与所罗门相和，两人立了约。

所罗门王从以色列全地征调服苦的人，共有三万。他差人轮流往利巴嫩去，每月一万人；他们在利巴嫩住一个月，在本家住两个月。亚多尼兰督管这些服苦的人。所罗门还有七万扛抬的，八万在山上凿石的，并有三千三百督工的，管理做工的人。王吩咐人凿出大石，就是贵重的石头，为立殿的根基，凿成方石。所罗门的匠人和希兰的匠人，并迦巴勒人都凿好了，备好了木料和石头，要建造这殿。

**新约读经**：哥林多后书一章23节—二章17节

**保罗更改行程的缘由**

我以神为见证，叫我的灵魂作见证——我再没有往哥林多去，是为要宽容你们。我们不是辖管你们的信心，乃是帮助你们的快乐，因为你们凭信心站立得住。我自己定了主意，再到你们那里去，必不叫你们忧愁。若是我叫你们忧愁，除了我叫那忧愁的人之外，还有谁能叫我快乐呢？我曾把这事写给你们，是恐怕我到了的时候，应该叫我快乐的那些人，反倒叫我忧愁。我也深信，你们众人都以我的快乐为自己的快乐。我先前心里难过，痛苦，多多流泪，写信给你们，不是叫你们忧愁，乃是叫你们知道我格外疼爱你们。

**饶恕犯罪的人**

若有叫人忧愁的，他不但叫我忧愁，也是叫你们众人有几分忧愁（我不愿说太重了）。这样的人受了众人的责罚，也就够了，倒不如赦免他，安慰他，免得他忧愁太过，甚至沉沦了。所以我劝你们，要向他显出坚定不移的爱心来。我先前也写信给你们，要试验你们，看你们凡事顺从不顺从。你们赦免谁，我也赦免谁。我若有所赦免，是在基督面前为你们赦免的，免得撒但趁机胜过我们，因我们并非不晓得他的诡计。

**在基督里得胜**

我为基督的福音到了特罗亚，主也给我开了门，那时因为没有遇见兄弟提多，我心里不安，便辞别那里的人往马其顿去了。

感谢神！常率领我们在基督里夸胜，并借着我们在各处显扬那因认识基督而有的香气。因为我们在神面前，无论在得救的人身上，或灭亡的人身上，都有基督馨香之气。在这等人，就是一个从死到死的香气；在那等人，就是一个从生到生的香气。这些事谁能当得起呢？我们不像那许多人，为利混乱神的道，乃是由于诚实，由于神，在神面前凭着基督讲道。

**历史与灵修读写**

以使徒保罗为例，思想牧养灵魂是何等重要的事……他侍奉的丰富多样，他的温柔慈爱，同时他的严肃认真，以及两者如何交融调和——使他的温柔不至软弱，他的严厉不至激怒人心……他为一些人感恩，责备另一些人。有些人是他的喜乐和冠冕，有些人他则指责其愚蠢。那些走在正道上的人，他与之同行、共同热忱；那些走偏了的人，他出言纠正。有时他将人逐出教会，有时他坚定地表达他的爱。有时他哀恸忧伤，有时他欢欣喜乐。有时他以奶哺育人，有时他讲明奥秘真理。有时他降卑自己，有时他将人提升至他自己的高度。有时他以杖相威胁，有时他带来温柔的灵。有时他对骄傲的人刚强，有时他对卑微的人谦和。他时而是使徒中最微小的，时而又显出基督借着他说话的明证；时而他渴望离去，如祭物浇奠，时而他又认为为了他们的缘故留在肉身更有必要。因为他不寻求自己的益处，乃是寻求他在基督里藉福音所生的儿女的益处。这正是他一切属灵权柄的目标——在任何事上都将自己的利益置于次位，以成全他人的益处。

——纳西盎的额我略

**圣巴多罗买使徒**

圣巴多罗买（又名拿但业，见约翰福音）是最早跟从耶稣的十二使徒之一。他的家乡在加利利的迦拿（约翰福音二十一章2节），正是耶稣行第一个神迹之地。他由腓力邀请成为十二门徒之一，腓力告诉他，他们找到了弥赛亚，就是拿撒勒人耶稣（约翰福音一章45节）。巴多罗买起初因耶稣来自拿撒勒而有所迟疑，但很快便被一句清晰而坚定的信仰宣认所取代："你是神的儿子，你是以色列的王！"（约翰福音一章49节）。他与其他门徒同在（约翰福音二十一章1—13节），有幸见到复活的主，与他交谈并一同吃饭。据早期教父记载，巴多罗买将福音带到了亚美尼亚，并在那里遭受活剥之刑，为主殉道。

**路德宗信条建议阅读**

《奥格斯堡信条辩护》第五（三）条，第160—166段（第281—287段）

281路十一41的经文也按任意窜改方式被引用：“只要把里面的施舍给人，凡物于你们就都洁净了。”我们的对敌必定耳聋了。我们一再地说基督的福音须加于律法的宣讲内，即为基督的缘故，上帝喜悦信徒的善行。并且他们在各处排除基督，教导我们以律法的善行赚得称义。查考全经文可知道需要信心。282 基督斥责法利赛人，因为他们认为藉着屡次守洁净的礼，即在上帝面前称了义。某位教皇——我不确知是那一位——正这样说：若用水加盐撒在人身上“能使人们成圣洁净，”而此话的解释说：是洗净轻罪。法利赛人的意见也是如此。基督弃绝这些意见，立两倍的洁净代替虚假的洁净，一是内心的，一是外在的。祂曾命令洁净内心，后加上关于外面洁净：“由于剩下的，凡事为你们洁净了。”

283我们对敌误解此普遍的质词“凡”。基督加这结论与两个子句：你们若里面洁净了，并且若施舍，那时凡事就洁净了。祂的意思是说：当在上帝所命令的行为内，求外面的洁净，不在于属人的传统，像当时的洁净礼节，或我们今时日日以水洒，即修道员之礼衣，分辨食物等华而不实的行动。我们对敌曲解此经文之意思，巧辩地将普遍质词改为一单句：“当你赈济了，凡物于你们就都洁净了。” 284尚且彼得（徒十五9 ）说：人的心藉信洁净了。硏究全经文现出〔这道理〕与其他圣经相合；因为人心洁净后，若加外表赈济（所有爱的行为），那时人们完全洁净了，无论内面外面。然而，为什么他们不呈上基督的全议题呢？祂的劝勉中有许多部分，有些命令信仰，有些则要求善行。诚实读者不得选出行为的命令，忽略属信的经句。

285最后我们提醒读者：我们对敌甚恶劣地指导虔诚良心，他们教导人行为赚得罪的赦免，因为藉着善工寻求赦罪的良心，不能确知其善工使上帝满意。那时良心常受痛苦，时时发明别的善工和礼拜，直到全然失望。保罗在罗四5下描写此进行，且证明义的应许不在乎我们的善事，因我们总不能确知有了一位恩惠的上帝。律法时常控告人。如此看来，应许必是徒然不确的。他结论：不是行为，乃是信接受所应许的赦罪与信之义。这是保罗真正的意思。这道将至大安慰提供与忠诚良心，并照明基督的荣耀，基督确定被赐予我们，好藉着祂有恩典、义、与和平。

286至此，我们重阅对敌引用来反对我们的经句，辩论信不称人为义，却藉我们的行为赚得恩典与罪的赦免。这些经文与我们的立场无冲突。我们的对敌恶毒地曲解圣经，以合乎他们自己的意见。他们以窜改方式引用许多经句。他们省略圣经论信最明晰的证据，选择论及行为的经句，连这些也误传。对各处经上所说的话，他们加上属人的意见。他们教训律法的方式，是隐藏基督的福音。希望我们已将这一切都表示出来，使虔诚人的良心得到满意。

287对敌之全系〔道理〕由于人理性或律法的教训得来，不由于福音。他们教导两种称义的样式，一种是基于理性，第二种基于律法，两种皆不基于福音或基督的应许。
