# 九月十九日

**诗篇吟唱**

你们要称谢耶和华，因他本为善，
他的慈爱永远长存。
你们要称谢万神之神，
他的慈爱永远长存。
你们要称谢万主之主，
他的慈爱永远长存；
惟独他行大奇事，
他的慈爱永远长存；
他用智慧造诸天，
他的慈爱永远长存；
他铺地在诸水之上，
他的慈爱永远长存；
他造了大光，
他的慈爱永远长存；
日头管白日，
他的慈爱永远长存；
月亮星宿管黑夜，
他的慈爱永远长存；

——诗篇136篇1—9节

**旧约读经**：尼希米记2章11—20节；4章1—6节

[**尼希米察看耶路撒冷的城墙**]

11我到了耶路撒冷，在那里住了三天。12我夜间起来，有几个人跟从我，但我没有告诉任何人，就是我的　神感动我要为耶路撒冷所做的事；除了我所骑的牲口以外，也没有别的牲口。13当夜我出了谷门，往龙井去，到了粪厂门，察看耶路撒冷的城墙，见城墙拆毁，城门被火焚烧了。14我往前过了泉门，到了王池，但所骑的牲口没有地方过去。15于是我夜间沿着溪边上去，察看城墙，又转回，从谷门进来，就回去了。16官长们不知道我往哪里去，做什么事；我也没有将这事告诉犹大人、祭司、贵胄、官长，和其余将做这工的人。
17后来我对他们说："我们所遭遇的困难，耶路撒冷怎样荒废，城门怎样被火焚烧，你们都看见了。来，我们重建耶路撒冷的城墙，免得再受凌辱！"18我告诉他们我　神的手怎样帮助我，并王对我所说的话。他们就说："我们起来建造吧！"于是他们振作心，做这美好的工作。19但参巴拉、亚扪人多比雅和阿拉伯人基善听见了，就嗤笑我们，藐视我们，说："你们做什么呢？要背叛王吗？"20我回答他们说："天上的　神必使我们亨通；我们是他的仆人，我们要起来建造。你们却在耶路撒冷无分、无权、无纪念！"

**对抗工程的阻挠**

4:1参巴拉听见我们重建城墙，就发怒，大大恼恨，嗤笑犹大人，2对他的弟兄和撒马利亚的军兵说："这些软弱的犹大人做什么呢？他们能建造吗？能献祭吗？能一日成功吗？他们能从碎石堆中复活那烧过的石头吗？"3亚扪人多比雅在旁边，说："他们所建造的石墙，就是狐狸跳上去也必拆倒了！"4我们的　神啊，求你垂听，因为我们被藐视了！愿你使他们的羞辱归到他们自己的头上，使他们在掳到之地成了掠物。5不要遮掩他们的罪孽，不要使他们的过错从你面前涂抹，因为他们在建造者面前惹了你的怒气。
6这样，我们修造城墙，城墙就都连络了，直到一半，因为百姓有心做工。

**新约读经**：提摩太前书2章1—15节

**为众人祈祷**

1我劝你，第一要为万人恳求、祷告、代求、感谢；2为君王和一切在位的，也该如此，使我们可以敬虔、端正、平安无事地度日。3这是好的，在我们救主　神面前可蒙悦纳。4他愿意万人得救，明白真道。5因为只有一位　神，在　神和人中间，只有一位中保，就是降世为人的基督耶稣；6他舍自己作万人的赎价，到了时候，这事必证明出来。7我为此奉派做传道者和使徒（我说真话，并不说谎），做外邦人的师傅，教导他们信心和真道。
8我愿男人无忿怒、无争论地，举起圣洁的手，随处祷告；9又愿女人的装饰，不是外面的编头发、戴金饰、穿美衣，10而是要有廉耻和自守，以端庄为装饰，自称是敬虔女人的所当有的，就是用善行。11女人要沉静学道，凡事顺服。12我不许女人讲道，不许她管辖男人，只要沉静。13因为先造的是亚当，后造的是夏娃；14且不是亚当被引诱，乃是女人被引诱，陷在罪里。15然而，女人若能存信心、爱心，又圣洁、自守，就必在生育上得救。

**历史与灵修读写**

我不知道如何在分毫之间改变我过去一贯所教导的称义之道。即：借着信心——如彼得所说——我们有了新而洁净的心（《使徒行传》15:9—11），而　神愿意且确实为了我们的中保基督之缘故，将我们视为完全公义与圣洁的（《提摩太前书》2:5）。虽然肉体中的罪尚未完全被除去或死去（《罗马书》7:18），然而他却不会惩罚或记念它。

这样的信心、更新与罪的赦免之后，必有善行随之而来（《以弗所书》2:8—9）。其中凡仍属罪或不完全的，为了基督的缘故，都不会被算作罪或缺陷（《诗篇》32:1—2；《罗马书》4:7—8）。整个人——无论是其人格还是其行为——都被宣告为公义与圣洁，这完全出于倾注于我们、借基督遮盖我们的纯全恩典与怜悯。因此，若将善行与恩典和怜悯割裂来看，我们便无法在许多功德与行为上自夸。正如经上所写："那夸口的，当指着主夸口"（《哥林多前书》1:31）；就是夸自己有一位满有恩典的　神。因为有了这一点，一切都妥当了。我们还要说，若善行不随之而来，那信心便是虚假的，并非真正的信心。

——《施马加登信条》第三部分第十三条1—4节

我们焦切地等候你。
