# 五月二十七日

**诗篇吟唱**

⁴我有一件事曾求耶和华，我仍要寻求：
就是一生一世住在耶和华的殿中，
瞻仰他的荣美，在他的殿里求问。
⁵因为我遭遇患难，他必暗暗地保佑我；
在他亭子里藏着我，把我藏在帐幕的隐密处，
将我高举在磐石上。
⁶我现在得以抬起头来，高过四面的仇敌，
我要在他的帐幕里献欢呼声的祭，
我要唱歌，用诗歌颂赞耶和华。
⁷耶和华啊，我用声音呼吁，
求你听允我，并怜恤我，应允我。
⁸你说"你们当寻求我的面"，
我心向你说："耶和华啊，你的面我正要寻求。"
⁹不要向我掩面，
不要因怒气推开你的仆人，
你向来是帮助我的，求你不要丢掉我，也不要离弃我，
我得救的　神啊。
¹⁰我的父母离弃我，
耶和华必收留我。

——诗篇二十七篇4—10节

**旧约读经**：传道书四章1—16节

**日光之下的压迫**

¹我又转念，见日光之下所行的一切欺压。看哪，被欺压的人流泪，且无人安慰；欺压他们的人有权势，也无人安慰他们。²我就赞叹那早已死去的死人，胜过那还活着的活人。³并且，我以为那未曾生的，就是未见过日光之下恶事的，比这两等人更强。⁴我又见人为一切的劳碌和各样灵巧的工作，就被邻舍嫉妒。这也是虚空，也是捕风。

⁵愚昧人抱着手，吃自己的肉。⁶满手安静，强如两手劳碌捕风。

⁷我又转念，见日光之下有一件虚空的事：⁸有人孤单无二，无子无兄，竟没有穷尽的劳碌，眼目也不以钱财为足。他说："我劳碌，使自己缺乏，到底是为谁呢？"这也是虚空，是极重的劳苦。⁹两个人总比一个人好，因为二人劳碌同得美好的果效。¹⁰若是跌倒，这人可以扶起他的同伴；若是孤身跌倒，没有别人扶起他来，这人就有祸了。¹¹再者，二人同睡就都暖和；一人独睡怎能暖和呢？¹²有人攻胜孤身一人，若有二人便能敌挡他；三股合成的绳子不容易折断。

¹³贫穷而有智慧的少年人，胜过年老而愚昧不肯纳谏的王。¹⁴因为那少年人从监牢中出来，得登王位，虽然在所治之国中生来是贫穷的。¹⁵我见日光之下一切行动的活人，都随从那第二位，就是接续王位的少年人。¹⁶他所治理的众人，就是无穷的众人，在他以后的人也不喜悦他。这真是虚空，也是捕风。

**新约读经**：约翰福音七章32—53节

**差役受命拿耶稣**

³²法利赛人听见众人为耶稣纷纷议论这些事，祭司长和法利赛人就打发差役去拿他。³³耶稣说："我还有不多的时候与你们同在，以后就回到差我来的那里去。³⁴你们要找我，却找不到我；我所在的地方，你们不能到。"³⁵犹太人彼此说："这人要往哪里去，叫我们找不到他呢？难道他要往散住希利尼中的犹太人那里去，教训希利尼人吗？³⁶他说'你们要找我，却找不到我；我所在的地方，你们不能到'，这话是什么意思呢？"

**活水的江河**

³⁷节期的末日，就是最大之日，耶稣站着高声说："人若渴了，可以到我这里来喝。³⁸信我的人，就如经上所说，从他腹中要流出活水的江河来。"³⁹耶稣这话是指着信他之人要受圣灵说的。那时还没有赐下圣灵来，因为耶稣尚未得着荣耀。

**众人中的分裂**

⁴⁰众人听见这话，有的说："这真是那先知！"⁴¹有的说："这是基督。"但也有的说："基督岂是从加利利出来的吗？⁴²经上岂不是说，基督是大卫的后裔，从大卫本乡伯利恒出来的吗？"⁴³众人因着耶稣起了纷争。⁴⁴其中有人要拿他，只是没有人下手拿他。

⁴⁵差役回到祭司长和法利赛人那里，他们对差役说："你们为什么没有带他来呢？"⁴⁶差役回答说："从来没有像他这样说话的！"⁴⁷法利赛人说："你们也受了迷惑吗？⁴⁸官长或是法利赛人中有信他的吗？⁴⁹但这些不明白律法的百姓是被咒诅的。"⁵⁰其中有尼哥底母，就是从前去见耶稣的那个人，他说：⁵¹"不先听本人的口供，不知道他所做的事，难道我们的律法就定他的罪吗？"⁵²他们回答说："你也是出于加利利吗？你且去查考，就可知道加利利没有出过先知。"

⁵³于是各人都回家去了。

**历史与灵修读写**

因此，律法并不与　神的应许相悖——原因有二：其一，应许并不倚赖律法，而是倚赖　神的信实；其二，律法在其最高、最重要的用途上，是使人降卑，而降卑则使人叹息、哀痛，转而寻求中保的扶持。律法使他的恩典与怜悯分外甘甜——正如诗篇一百零九篇二十一节所说："你的慈爱甘甜"——使他的恩赐成为无可言说的宝贵。如此，律法使我们预备好来就基督。从未尝过苦味的人，不会记得甜味；饥饿才是最好的厨师。正如干旱的大地渴望雨水，律法使烦扰的心灵渴望基督。对这样的心灵而言，基督是何等甘甜；对他们而言，他是喜乐、安慰与生命。唯有如此，基督和他的工作才能被正确理解。

因此，律法最好的用途，在于能用它使人产生谦卑，并对基督的渴慕。他本人所要求的，正是渴慕的灵魂，并以最动人的方式呼召他们来就他，他说（马太福音十一章二十八节）："凡劳苦担重担的人，可以到我这里来，我就使你们得安息。"因此，他乐于滋润、浇灌这干旱之地。他不将水倒在肥沃、丰盛、毫无干渴的土地上。他的赐福是无可估量的，因此他只将其赐给有需要的人；他向贫穷的人传好信息（路加福音四章十八节），将水赐给干渴的人。他在约翰福音七章三十七节说："人若渴了，可以到我这里来。""他治愈伤心的人"（诗篇一四七篇三节）——就是说，他安慰、拯救那些被律法折磨困扰的人。因此，律法并不与　神的应许相悖。

——马丁·路德

**圣诗**

望那活水的江河，涌自永恒之爱，
丰沛供应儿女，除去一切匮乏之惧。
谁能在如此江河旁枯竭？
恩典恒久解渴——
如同那赐恩之主，
世代永不改变。

——《路德宗崇拜书》648:2
