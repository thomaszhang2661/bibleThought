# 十月二十七日

**诗篇吟唱**

愿耶和华在你遭难的日子应允你；愿雅各神的名保护你；
愿他从圣所给你救助，从锡安坚固你；
愿他记念你一切的供献，悦纳你的燔祭；
愿他照你的心意赐给你，使你一切所谋的都成全。
我们要因你的救恩夸胜，要在我们神的名下竖立旌旗；
耶和华必成全你一切的祈求。
现在我知道耶和华救护他受膏的了，必从他的圣天上应允他，用他右手的大能，赐他救护。
有人靠车，有人靠马，但我们要提到耶和华我们神的名。
他们跪倒仆倒，我们却起来，立得正直。
耶和华啊，求你救王，我们呼求你的时候，求你应允我们。

——诗篇二十篇

**旧约读经**：申命记二十九章1—29节

**摩押之约的更新**

这是耶和华在摩押地吩咐摩西与以色列人立的约的话，是在他于何烈山与他们所立的约之外另立的。

摩西招了以色列众人来，对他们说：耶和华在埃及地，在法老和他一切臣仆，并他全地面前所行的，你们都亲眼看见了；就是那些大试验，你亲眼看见的神迹和大奇事。但从今日以前，耶和华没有给你们能明白的心，能看见的眼，能听见的耳。我在旷野引导你们四十年，你们身上的衣服没有磨破，脚上的鞋也没有穿坏；你们没有吃饼，也没有喝清酒或浓酒，好叫你们知道他是耶和华你们的神。你们到了这地方，希实本王西宏和巴珊王噩出来与我们交战，我们就击杀了他们，夺了他们的地，将这地给了吕便支派、迦得支派、并玛拿西半支派的人为业。所以你们要谨守这约的话，照着去行，使你们所办的一切事都可以顺利。

你们今日都站在耶和华你们神的面前：就是你们各支派的领袖、长老、官长，以及以色列众人，你们的妻子儿女，并你营中寄居的，就是给你们劈柴的、挑水的，要你们进入耶和华你神所立的约，就是他今日与你所立的起誓的约，要以你为他的子民，他作你的神，正如他向你起誓所应许的，也照他向你列祖亚伯拉罕、以撒、雅各所起的誓。我不只是与你们立这起誓的约，也是与今日没有站在这里和我们同在的人立这约。

你们知道我们曾住在埃及地，也知道我们从那里经过的那些国民，你们看见了他们中间那些可憎的偶像，就是木、石、金、银的。恐怕你们中间——无论是男是女，是族长是支派长——有人心里转离耶和华我们的神，去侍奉那些国民的神；又恐怕你们中间有苦根生长，结有毒的苦果；他若听见这起誓的约上的话，就在心里为自己祝福说：我虽然心里顽梗，却仍然平安。这样，就把湿的和干的一同灭绝。耶和华必不肯饶恕这样的人，耶和华的怒气和愤恨必向他发作，如烟冒出，将这书上所写的一切咒诅都加在他身上，耶和华要从天下涂抹他的名，又按着约书上所写一切咒诅，将他从以色列众支派中分别出来，受祸患。后来的人，就是你们的儿女，从远方而来的外邦人，看见耶和华所降给那地的灾殃，并他发泄在那地的忿怒，就是各种疾病，并且那地土烧尽，硫磺、盐、火烧，没有播种，没有生长，也没有草长，与耶和华在忿怒、恼恨中所倾覆的所多玛、蛾摩拉、押玛、洗扁相似，诸国的人，就都要问说：耶和华为什么这样待这地呢？这样的烈怒是什么意思呢？人必回答说：因为他们离弃了耶和华他们列祖的神，就是领他们出埃及地之后与他们所立的约；去侍奉事奉他们素来不认识的神，不是他分给他们的；所以耶和华的怒气向这地发作，将这书上所写的一切咒诅都降在这地上；耶和华向他们大发怒气，就将他们拔出本地，忿怒恼恨，丢在别的地上，正如今日一样。

隐秘的事，是属耶和华我们神的；唯有明显的事，是永远属我们和我们子孙的，好叫我们遵行这律法上的一切话。

**新约读经**：马太福音十八章21—35节

**不饶恕人的仆人的比喻**

那时，彼得进前来，问耶稣说：主啊，我弟兄得罪我，我当饶恕他几次呢？到七次可以吗？耶稣说：我不是说七次，乃是到七十个七次。

天国好比一个王要和他仆人算账。才算的时候，有人带了一个欠一千万两银子的来。因为他没有什么偿还之物，主人吩咐把他和他妻子儿女，并一切所有的都卖了偿还。那仆人就俯伏拜他，说：主啊，宽容我，将来我都还清。那仆人的主人，就动了慈心，把他释放了，并且免了他的债。那仆人出来，遇见他的一个同伴，欠他十两银子，便揪住他，掐住他的喉咙，说：你把所欠的还我！他的同伴就俯伏央求他，说：宽容我吧，将来我必还清。他不肯，去把他下在监里，等他还了所欠的债。众同伴看见了，非常忧愁，去把所做的事都告诉了主人。于是主人叫了他来，对他说：你这恶奴才！你央求我，我就把你所欠的都免了；你不应当怜悯你的同伴，像我怜悯你吗？主人大怒，把他交给掌刑的，等他还清了所欠的债。你们各人若不从心里饶恕你的弟兄，我天父也要这样待你们了。

**历史与灵修读写**

若弥撒是一个应许，如我们所说的，那么接近它的途径，不在于任何的工、能力或自身的功德，而是只在于信心。因为在有许诺之神的话语之处，必然要有接受之人的信心。由此可见，我们救恩的开始，是一种信心，它紧紧抓住许诺之神的话语——他不需要我们任何的努力，以自由和白白的怜悯主动向我们赐下他应许的话语。"他发出他的话语，医治了他们"，而不是"他接受了我们的工，就医治了我们"。首先有神的话语。随其而来的是信心；继信心之后是爱；然后爱做每一件善事，因为它不行恶，甚至是律法的总结（罗十三10）。除了借着信心，人没有其他途径能来到神面前或与他交往。这就是说，救恩的作者不是人、凭着他任何的工，而是神、借着他的应许；一切都依赖、被支撑和保守于他那能力的话语中（来一3），他借着这话语生了我们，使我们成为他所造物的初熟果子（雅一18）。

——马丁·路德
