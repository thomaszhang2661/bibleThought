# 七月三十一日　亚利马太的约瑟

**诗篇吟唱**
14万军之神啊，求你转回！从天上垂看，眷顾这棵葡萄树，15就是你右手所栽的根株，你为自己所坚固的子孙。
16这株已被火烧灭，被人砍伐；愿他们因你脸上的责备而灭亡！
17愿你的手扶持你右边的人，就是你为自己所坚固的人子！
18这样，我们便不退后离开你；求你救活我们，我们就要求告你的名。
19万军之主神啊，求你使我们回转！使你的脸发光，我们便得救！
——诗篇八十篇14节至19节

**旧约读经**：撒母耳记上16章1节至23节

**大卫受膏为王**

1耶和华对撒母耳说："我既厌弃扫罗作以色列的王，你为他悲伤要到几时呢？你将膏油盛满了角，我差你往伯利恒人耶西那里去，因为我在他众子之中，预备了一个王。"2撒母耳说："我怎能去呢？扫罗若听见，必要杀我。"耶和华说："你带一只小母牛去，就说：'我来是要向耶和华献祭。'3你要请耶西来吃祭肉，我必指示你所当做的事，你就要为我膏我所指定的人。"4撒母耳照耶和华所说的去行，来到伯利恒，城中的长老都战战兢兢地来迎接他，说："你平安来到吗？"5他说："平安。我来是要向耶和华献祭，你们当自洁，来与我一同吃祭肉。"他就使耶西和他众子自洁，请他们来吃祭肉。

6他们来的时候，撒母耳看见以利押，心里说："耶和华的受膏者必定在他面前。"7耶和华却对撒母耳说："不要看他的外貌和他身材高大，我不拣选他，因为耶和华不像人看人：人是看外貌，耶和华是看内心。"8耶西叫亚比拿达从撒母耳面前走过，撒母耳说："这人耶和华也没有拣选。"9耶西又叫沙玛从撒母耳面前走过，撒母耳说："这人耶和华也没有拣选。"10耶西叫他七个儿子都从撒母耳面前走过，撒母耳对耶西说："这些耶和华都没有拣选。"11撒母耳问耶西说："你的儿子都在这里了吗？"他说："还有个小的，他正在放羊。"撒母耳对耶西说："你打发人去叫他来，他若不来，我们不坐席。"12耶西就打发人去叫了他来，他面色红润，双目清秀，容貌俊美。耶和华说："起来，膏他，就是他了。"13撒母耳就用那角油膏大卫，耶和华的灵从这日起，大大感动大卫。撒母耳起来，回到拉玛去了。

**大卫为扫罗效力**

14耶和华的灵离开扫罗，有恶魔从耶和华那里来扰乱他。15扫罗的臣仆对他说："现在有恶魔从神那里来扰乱你，16我们的主可以吩咐面前的臣仆寻找一个善于弹琴的人，当神赐下恶魔的时候，他便弹琴，你就好了。"17扫罗吩咐臣仆说："你们为我寻找一个善于弹琴的，带他到我这里来。"18臣仆中有一个回答说："我曾见到伯利恒人耶西的一个儿子善于弹琴，是大能的勇士，是战士，说话有智慧，是俊美的人，耶和华也与他同在。"19扫罗就差遣使者到耶西那里说："你把你的儿子大卫，就是那放羊的，打发到我这里来。"20耶西就用驴驮着饼和一皮袋酒，并一只山羊羔，交给他儿子大卫送到扫罗那里。21大卫到了扫罗那里，侍立在他面前，扫罗甚喜爱他，大卫就做了他的拿兵器的人。22扫罗差人到耶西那里说："请你让大卫侍立在我面前，因为他在我面前蒙恩了。"23每逢神赐下恶魔的时候，大卫就拿琴，用手弹奏，扫罗便舒畅爽快，恶魔就离开他了。

**新约读经**：使徒行传25章13节至27节

**保罗在亚基帕和百尼基面前**

13过了些日子，亚基帕王和百尼基来到凯撒利亚，问腓斯都安。14在那里住了许多天，腓斯都将保罗的事告诉王，说："这里有一个人，是腓力斯留在监里的，15我在耶路撒冷的时候，祭司长和犹太长老告他的状，求我定他的罪。16我回答他们说：'无论什么人，被告还没有和原告对质，没有得着申辩所告之事的机会，就定他的罪，这不是罗马人的规矩。'17及至他们来到这里，我没有耽延，第二天就坐堂，吩咐把那人带上来。18原告站着，告出他的罪来，但是所告的，并没有我所猜测的那些凶恶的事；19不过是关于他们自己的宗教和一个人名叫耶稣的事，这人已经死了，保罗却说他是活着的。20我既不能审断这些事，就问他可愿意上耶路撒冷，在那里受审这些事不？21但保罗请求把他留下，等皇上审问他，我就吩咐把他收押，等候打发他到凯撒那里去。"22亚基帕对腓斯都说："我也愿意亲自听这人说话。"腓斯都说："明天你就可以听他了。"

23第二天，亚基帕和百尼基大张威势而来，同着众千夫长和城里尊贵的人进了公厅；腓斯都吩咐一声，就把保罗带进来。24腓斯都说："亚基帕王和在座的诸位啊，你们看这人，就是一切犹太人在耶路撒冷和这里，求告我的，高声喊着说他不该再活着。25但我查明他没有犯什么该死的罪，他自己也请求归皇上审断，所以我定意把他解去。26但论到这人，我没有确实的事可以奏明主上，因此把他带到你们面前，特意带到你亚基帕王面前，为要在查问之后，有所陈奏。27因为解送囚犯，不指明他的罪案，在我看来是不合理的事。"

**历史与灵修读写**

**为一切好诗歌集所作序——音乐女神自述**

音乐女神如是言：

大地之上诸般喜乐，
无一能与吾相媲美。
吾以悦耳之鸣响，
以众声和谐之歌唱，
赐予世人最深的安慰。

歌声响起之处，
邪恶的念头无处藏身；
嫉妒、仇恨与忿怒，
皆随吾而烟消云散。
贪婪、忧虑与沉重的孤寂，
再不能压迫人心。
人人皆可在欢喜中自由，
因这样的喜乐，毫无罪愆。

然而神在吾中所寻得的喜悦，
远超过世间一切欢乐。
凭借我那灿烂的力量，
魔鬼收敛其罪恶、凶残与阴谋。

大卫王的行事印证了这一点——
他以悠扬的琴声，
平息了扫罗王的怒火，
因而逃脱了那杀身之祸。
昔日以利沙先知，
藉弹琴而得圣灵感动，
以此传讲神圣的真道，
引领谦卑信心的人归向神。

这一年中最美好的时光属于我，
当百鸟争鸣，歌声四起。
天地之间充满歌声，
和谐悦耳，韵律清扬。
百鸟之王——夜莺，
以那迷人欢快的歌喉，
使人心欢欣鼓舞；
愿人们永远感谢她。

然而，首先要感谢的，
是我们的主神，
是祂用话语创造了她，
使她成为祂所钟爱的歌者，
成为音乐的女主人。
她日夜不停地为主歌唱，
献上赞美之声；
我也将我的旋律献给祂，
在永恒中归荣耀于祂。

——马丁·路德

**亚利马太的约瑟**

这位约瑟见于四福音书的记载，来自犹太山地一个名为亚利马太的小村庄。他是公会（即耶路撒冷犹太宗教议会）中德高望重的成员。他想必家境殷实，因为他在耶稣受难之地附近的园中拥有一座自己尚未使用的坟墓（马太福音二十七章60节）。约瑟是一个殷切等候神国降临的人，他在耶稣死后前往彼拉多处，求领耶稣的身体（马可福音十五章43节）。他与尼哥底母一同将遗体取下，安放在墓中（约翰福音十九章38至39节）。他们公开表达的虔诚敬爱，与那些离弃耶稣而逃散的门徒之畏缩，形成了鲜明的对比。
