# 九月二十九日　米迦勒与众天使节

**诗篇吟唱**

7耶和华的律法完全，能使人苏醒；
耶和华的法度确定，能使愚人有智慧。
8耶和华的训词正直，能快悦人心；
耶和华的命令纯洁，能使人眼目明亮。
9耶和华的道理洁净，存到永远；
耶和华的典章真实，全然公义。
10都比金子可羡慕，且比极多的精金可羡慕；
比蜂房下滴的蜜更甜。
11况且你的仆人因此受了警戒，
守此典章就有大赏。
12谁能知道自己的错失呢？
愿你赦免我隐而未现的过错。
13求你拦阻仆人不犯任意妄为的罪，
不容这罪辖制我，我便完全，
免去大罪了。
14耶和华我的磐石，我的救赎主啊，
愿我口中的言语、心里的意念
在你面前蒙悦纳。

——诗篇十九篇7—14节

**旧约读经**：申命记1章19—36节

**以色列人拒绝进入迦南地**

19"我们从何烈山起行，经过你们所见那大而可怕的旷野，往亚摩利人的山地去，正如耶和华我们神所吩咐我们的；我们到了加低斯巴尼亚。20我对你们说：'你们已经来到耶和华我们神所赐给我们的亚摩利人山地。21看哪，耶和华你的神已将那地摆在你面前，你要照耶和华你列祖神所说的话上去得那地，不要惧怕，也不要惊惶。'22你们都来就近我，说：'我们要先打发人去，为我们窥探那地，将我们上去应走何路、可进哪些城，都带信息回来。'23这话我以为好，就从你们中间按十二支派各取一人。24他们起身上了山地，到以实各谷，窥探那地；25他们手拿那地的果子下来见我们，说：'耶和华我们神所赐给我们的那地是美地。'

26"你们不肯上去，竟违背了耶和华你们神的命令；27你们在帐棚里发怨言说：'耶和华因为恨我们，所以将我们从埃及地领出来，要把我们交在亚摩利人手中，除灭我们。28我们上哪里去呢？我们的弟兄使我们的心消化，说那民比我们又大又高，城邑又广大又坚固，高得顶天；并且我们在那里看见亚衲族的人。'29我就对你们说：'不要惊恐，也不要怕他们。30在你们前面行的耶和华你们的神，必为你们争战，正如他在埃及在你们眼前所行的一样；31你们在旷野所行的路上也曾见耶和华你们的神抚养你们，如同人抚养儿子一样，直等你们来到这地方。'32你们在这事上还是不信耶和华你们的神；33他在路上在你们前面行，为你们寻找安营的地方，夜间在火柱里，日间在云柱里，指示你们所当行的路。'

**以色列人悖逆所受的惩罚**

34"耶和华听见你们的话，就发怒，起誓说：35'这恶世代的人中，没有一人得见我誓给你们列祖的那美地，36除了耶孚尼的儿子迦勒，他必得见，因为他专心跟从我，我必把他所踏之地赐给他和他的子孙。'"

**新约读经**：马太福音5章21—48节

**论愤怒**

21"你们听见有话对古人说：'不可杀人'；又说：'凡杀人的，难免受审判。'22只是我告诉你们：凡向弟兄动怒的，难免受审判；凡骂弟兄是拉加的，难免公会的审断；凡骂弟兄是魔利的，难免地狱的火。23所以，你在祭坛上献礼物的时候，若想起弟兄向你怀怨，24就把礼物留在坛前，先去同弟兄和好，然后来献礼物。25你同告你的对头还在路上，就赶紧与他和息，恐怕他把你送到审判官，审判官交付衙役，你就下在监里了。26我实在告诉你：若有一文钱没有还清，你断不能从那里出来。

**论奸淫**

27"你们听见有话说：'不可奸淫。'28只是我告诉你们：凡看见妇女就动淫念的，这人心里已经与她犯奸淫了。29若是你的右眼叫你跌倒，就剜出来丢掉，宁可失去百体中的一体，不叫全身丢在地狱里。30若是右手叫你跌倒，就砍下来丢掉，宁可失去百体中的一体，不叫全身下入地狱。

**论休妻**

31"又有话说：'人若休妻，就当给她休书。'32只是我告诉你们：凡休妻的，若不是为淫乱的缘故，就是叫她作淫妇了；人若娶这被休的妇人，也是犯奸淫了。

**论起誓**

33"你们又听见有话对古人说：'不可背誓，所起的誓总要向主谨守。'34只是我告诉你们：什么誓都不可起，不可指着天起誓，因为天是神的宝座；35不可指着地起誓，因为地是他的脚凳；也不可指着耶路撒冷起誓，因为耶路撒冷是大君的京城；36又不可指着你的头起誓，因为你不能使一根头发变黑变白了。37你们的话，是就说是，不是就说不是；若再多说，就是出于那恶者。

**论报复**

38"你们听见有话说：'以眼还眼，以牙还牙。'39只是我告诉你们：不要与恶人作对。有人打你的右脸，连左脸也转过来由他打；40有人想要告你，要拿你的里衣，连外衣也由他拿去；41有人强逼你走一里路，你就同他走二里；42有求你的，就给他；有向你借贷的，不可推辞。

**论爱仇敌**

43"你们听见有话说：'当爱你的邻舍，恨你的仇敌。'44只是我告诉你们：要爱你们的仇敌，为那逼迫你们的人祷告；45这样，就可以做你们天父的儿子，因为他叫日头照好人，也照歹人；降雨给义人，也给不义的人。46你们若单爱那爱你们的人，有什么赏赐呢？就是税吏不也是这样行吗？47你们若单请你弟兄的安，比人有什么长处呢？就是外邦人不也是这样行吗？48所以，你们要完全，像你们的天父完全一样。"

**历史与灵修读写**

论罪的揭示：摩西的帕子（哥林多后书三章12—16节）遮蔽着众人的眼目，只要他们仅仅听见律法的单纯讲道，而不闻基督，情况便是如此。因此，他们不能从律法中学会正确认识自己的罪。他们或是成为厚颜无耻的假冒为善者——像法利赛人那样〈因自以为义而膨胀〉（马太福音二十三章），或是像犹大那样陷入绝望（马太福音二十七章3—5节）。因此，基督亲手接过律法，照灵意解释律法（马太福音五章21—48节；罗马书七章14节）。如此，神的忿怒便从天上向一切罪人显明（罗马书一章18节），使他们看见那忿怒是何等之大。如此，他们被带回到律法面前，这才第一次从律法中正确认识自己的罪——这是摩西凭借律法从未能够逼出来的认识。

照此而论，宣讲神之子基督的受苦与死亡，乃是对神之忿怒严肃而令人战栗的宣告与宣示。藉着这样的讲道，人才被正确地引入律法——在摩西的帕子从他们身上除去之后。此时他们才第一次正确理解，神在他的律法中要求我们何等重大的事，而这一切我们都无力遵守。因此，他们知道要在基督里寻求我们一切的义。……只要这一切（即基督的受苦与死亡）宣告神的忿怒并使人战兢，它就还不是严格意义上的福音讲道，仍是摩西与律法的讲道，因此是基督的外来工作。藉着这教训，基督到达他本有的职分，就是宣讲恩典、安慰人心、赐人生命，这才是正确意义上的福音讲道。

——《协和信条》摘要第五条8—10节

天使对人持何等态度？主基督用一句话揭示了这一点：他称天使为"他们的天使"，就是小子们的天使、儿女与一切信徒的仆人……他们以多种方式向每一位基督徒尽这服事。当我们还是孩童时，神便将天使派给我们，正如基督在圣福音中所告诉我们的。当我们年长，走上自己的道路，就是行走于我们蒙召之路上时，神也将天使派给我们（诗篇九十一篇11、12节）。我们入睡时，天使守望保护我们，以防魔鬼侵害。我们死亡时，天使将我们的灵魂带到亚伯拉罕的怀里（路加福音十六章22节）。他们对我们的保护贯穿整个人生……为何天使虽远比我们尊贵崇高，却甘心事奉信徒？（一）因为他们在善中得以坚立，因此甘心完全地顺从神的旨意。神的旨意和命令就是要他们服事我们（希伯来书一章14节）。诸天的大军——日月星辰——按神为人所赐的秩序运行，更何况圣天使的天军岂不更当维持其秩序？（二）因为我们的本性在基督里被升举在一切天使与大天使之上（以弗所书一章20、21节；希伯来书一章4节）。因此，天使不拒绝服事我们人类，是为尊崇基督所取之人性。正如整个家族因一桩婚姻而得荣耀，神子与人性的联合同样使人类恢复了荣耀（马太福音二十二章2节）。那么，天使服事我们有什么奇怪的呢？因为天使的主、神之子亲自来到地上，正是为了服事我们。（三）因为爱在天使中是纯全的，所以他们欢喜服事我们，正如那位主——他本身就是爱（约翰一书四章8节），天使是按其形像被造的——所宣告的："我要喜悦待他们好"（耶利米书三十二章41节）。（四）最后，因为我们有一天将与天使同在天上，与他们同声赞美神，所以天使乐于在今世服事我们。

——约翰·格哈德

## 米迦勒与众天使节

大天使圣米迦勒的名字意为"谁能与神相比？"米迦勒见于但以理书（十二章1节），亦见于犹大书（第9节）和启示录（十二章7节）。但以理书将米迦勒描绘为以色列属天的守护者，带领众军与邪恶势力争战。在启示录中，米迦勒与他的天使与撒但及其邪灵争战，将他们从天上逐出，取得胜利。他们的胜利是由基督藉自己的死与复活所得对撒但之胜利成为可能的——天上一个声音宣告："我神的救恩、能力、国度，并他基督的权柄，现在都来到了"（启示录十二章10节）。米迦勒常与加百列和拉斐尔并提，他们是环绕神宝座的其他主天使或大天使。按照传统，米迦勒被视为教会的守护者与保护者，尤其是在信徒临终时刻对他们的保护者。
