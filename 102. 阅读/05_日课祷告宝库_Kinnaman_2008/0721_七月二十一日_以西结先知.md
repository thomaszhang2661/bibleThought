# 七月二十一日　以西结先知

**诗篇吟唱**

2谁能传说耶和华大能的作为，或能宣扬他一切的美德？
3凡遵守公平、时常行义的，这人便为有福！
4耶和华啊，你用恩惠待你的百姓的时候，求你纪念我，搭救我的时候求你帮助我，
5使我得见你选民的福乐，与你的国一同欢喜，与你的产业一同夸耀。

**旧约读经**：撒母耳记上4章1节至22节

非利士人夺取约柜

1撒母耳的话传遍以色列。

那时，以色列人出去与非利士人打仗，在以便以谢安营，非利士人在亚弗安营。2非利士人摆阵攻击以色列人，两军交战，以色列人败在非利士人面前。非利士人在战场上杀了他们约有四千人。3百姓回到营中，以色列的长老说：「耶和华今日为何使我们败在非利士人面前呢？我们要将约柜从示罗抬到我们这里来，耶和华的约柜好在我们中间，救我们脱离仇敌的手。」4于是百姓打发人到示罗，从那里将坐在二基路伯上的万军之耶和华的约柜抬来；以利的两个儿子何弗尼、非尼哈也在那里与神的约柜同在。

5耶和华的约柜到了营中，以色列众人就大声喊呼，地也震动了。6非利士人听见喊呼的声音，就说：「这大声喊呼是什么意思呢？」他们知道耶和华的约柜到了营中，7非利士人就惧怕，说：「有神到了营中了。」又说：「我们有祸了！因为以前没有这样的事。8我们有祸了！谁能救我们脱离这大能神的手？他从前在旷野用各样的灾殃击打埃及人。9非利士人哪，你们要刚强，要像勇士，免得你们做了希伯来人的仆人，就如他们做了你们的仆人一样。你们要刚强，要去打仗。」

10非利士人就打仗，以色列人败了，各人逃回帐棚去。被杀的人甚多，以色列兵被杀的有三万步兵。11神的约柜被掳去，以利的两个儿子何弗尼、非尼哈也都死了。

以利的死

12有一个便雅悯人从阵上逃跑，当日到了示罗，衣服撕裂，头蒙尘土。13他来到的时候，以利正坐在道旁的椅子上，心中战战兢兢，为神的约柜挂虑。那人进城报信，合城的人都哀号。14以利听见哀号的声音，就说：「这喧嚷是什么意思？」那人急忙前来告诉以利。15那时以利已经九十八岁了，眼目发直，不能观看。16那人对以利说：「我是从阵上来的，今日我从阵上逃跑来到这里。」以利说：「我儿，事情怎样？」17报信的人回答说：「以色列人在非利士人面前逃跑，百姓中间死亡甚多，你的两个儿子何弗尼、非尼哈也死了，并且神的约柜被掳去了。」18他一提到神的约柜，以利就从椅子上往后跌倒，在城门旁折断颈项而死，因为他年老体重。他作以色列的士师四十年。

19他的儿媳，就是非尼哈的妻，已经身怀六甲，快要临盆。她听见神的约柜被掳、公公和丈夫都已死了，就忽然屈身生产，因为她的疼痛猛然临到她身上。20她将要死的时候，旁边伺候的妇人对她说：「不要惧怕，你生了个儿子了。」她却不回答，也不放在心上。21她给孩子起名叫伊迦博，说：「荣耀离开以色列了！」这是因为神的约柜被掳，又因她的公公和丈夫都死了。22她又说：「神的约柜被掳去了，荣耀离开以色列了！」

**新约读经**：使徒行传16章23节至40节

保罗和西拉在狱中

23官差打了他们许多棍，便将他们下在监里，嘱咐禁卒严紧看守。24禁卒受了这吩咐，就把他们下在内监里，两脚上了木狗。

腓立比狱卒归信

25约在半夜，保罗和西拉祷告，唱诗赞美神，众囚犯也侧耳而听。26忽然地大震动，甚至监狱的地基都摇动了，监门立刻全开，众囚犯的锁链也都松开了。27禁卒一醒，看见监门全开，以为囚犯已经逃走，就拔出刀来，要自杀。28保罗大声呼叫说：「不要伤害自己！我们都在这里。」29禁卒叫人拿灯来，就跳进去，战战兢兢地俯伏在保罗、西拉面前；30又领他们出来，说：「二位先生，我当怎样行才可以得救？」31他们说：「当信主耶稣，你和你一家都必得救。」32他们就把主的道讲给他和他全家的人听。33当夜就在那时候，禁卒把他们带去，洗他们的伤；他和属乎他的人立时都受了洗。34于是禁卒领他们上自己家里，给他们摆上饭。他和全家因为信了神，都很喜乐。

35到了天亮，官长打发差役说：「释放那两个人吧。」36禁卒把这话告诉保罗说：「官长打发人来释放你们，如今可以出去，平平安安地走吧。」37保罗却说：「我们是罗马人，并没有定罪，他们就当众打了我们，又把我们下在监里；现在要私下打发我们出去吗？这是不行的！叫他们自己来领我们出去吧。」38差役把这话回禀官长，官长听见他们是罗马人就害怕，39便来劝他们，领他们出去，请他们离开那城。40他们出了监，进入吕底亚家里，见了弟兄们，劝慰他们一番，就走了。

**历史与灵修读写**

以西结是布西之子，出自亚伦一脉，身兼祭司与先知二职。在世界史纪元3350年，即耶路撒冷陷落前六年，他在巴比伦领受神的感示，安慰并坚固那些在五年前受耶利米劝勉、自愿随约雅敬王（即耶哥尼雅）投降尼布甲尼撒王并移居巴比伦的犹太人。这些人眼见耶路撒冷的毁灭延迟了整整五年，便以为耶利米的预言毫无根据，且懊悔自己当初甘心离弃故土。因此，以西结预言耶路撒冷必然倾覆，纵使刑罚暂时推迟。与此同时，他也应许百姓将从巴比伦得释放归回（第11章）。在世界史纪元3358年，即耶哥尼雅被掳第三年，以西结藉奇特的礼式与神圣的见证，被呼召进入他的职事（第1—3章）。那四轮与活物的异象（第1章与第10章），象征基督的战车——他借着福音的执事、圣灵的气息与使徒的火舌，被运行至普天之下……在世界史纪元3378年，即耶哥尼雅被掳第二十五年，以西结自第四十章直至末章，描述属灵耶路撒冷的建造，亦即基督教会圣职的建立。
——大卫·基特罗斯

**以西结先知**

以西结是布西之子，身为祭司，蒙神呼召在巴比伦被掳期间作流亡者的先知（以西结书1:3）。公元前597年，巴比伦王尼布甲尼撒率军将犹大王及耶路撒冷数千精英——包括以西结在内——掳至巴比伦（列王纪下24:8—16）。以西结的祭司背景深刻地铸就了他的预言风格：神的圣洁与圣殿的意象在其信息中占有举足轻重的地位（如以西结书第9—10章与第40—48章）。自公元前593年至公元前586年耶路撒冷与圣殿被毁，以西结宣告神对耶路撒冷、对巴比伦流亡者，以及对以色列周边七国的审判不可避免（以西结书第1—32章）。耶路撒冷必将陷落，流亡者也不会迅速归回，这是他们犯罪所应受的公义报应。当以西结得知耶路撒冷与圣殿已被毁的消息后，其信息便转为安慰与盼望：神藉他应许，他的百姓将在将来的弥赛亚国度中经历复兴、更新与复苏（以西结书第33—48章）。以西结预言中许多奇异的象征意象，后来被圣约翰的启示录所援引。
