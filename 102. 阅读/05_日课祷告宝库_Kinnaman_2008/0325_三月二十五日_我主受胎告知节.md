# 三月二十五日　我主受胎告知节

**历史与灵修读写**

历代圣祖、族长与先知，无不渴望亲眼见他，却未能如愿。其中有人藉着异象，在预表与朦胧中瞥见了他；另有一些人，则蒙恩得闻那藉着云彩降临的神圣声音，并蒙赐观见圣天使的荣光。然而，唯独对圣洁的童贞玛利亚，大天使加百列以炫目的光华显现，向她带来欢喜的问候："你好，蒙大恩的女子！"于是她接纳了道，不久之后，藉着身体自然的孕育过程，她生下了那颗珍贵的珍珠。那么，来吧，亲爱的弟兄们，我们也一同颂唱那由受灵感的大卫竖琴所教导我们的歌，说："主啊，兴起，进入你的安息之所，你与你圣所之约柜一同。"因为这位圣童贞女确实是一座约柜，内外皆以金子包裹，其中承载了至圣所全部的宝库。"主啊，兴起，进入你的安息之所。"主啊，兴起，从父的怀抱中出来，为要拯救那堕落的始祖族类……玛利亚将那位坐在基路伯之上、受万万天使颂赞的主，安放在马槽之中。神的道躺在那供非理性牲口使用的马槽里，为要将真正的理性与悟性赐予那些凭自由意志选择了非理性的人类。那供牛马进食的马槽中，安置了天上的粮，为要以属灵的粮食供养那些活得如同地上走兽的人们。甚至连客店里也没有地方容纳他。那位以言语建立天地的主，找不到安身之处；"他本来富足，却为我们的缘故成了贫穷"，并为着拯救我们的本性，甘愿选择了极度的卑微，在他固有的良善中，向我们倾覆了自己。他在父的怀抱中，成全了上帝奥秘工作中那无法言说的整个大工；他又在山洞里，躺在母亲的怀抱中，安歇于马槽之内。天使的诗班围绕着他，高歌天上的荣耀与地上的平安。他在天上坐在父的右边；他在马槽里歇息，仿佛坐在基路伯之上。那马槽，确实是他的基路伯宝座，是他的王座所在。圣中之圣，地上独一荣耀，圣中更圣者，乃是基督我神所安歇的马槽。愿荣耀、尊贵与权能归于他，并归于无瑕的父，以及那全然圣洁、赐生命的圣灵，从今时直到永永远远。阿们。

——大能行者额我略

**我主受胎告知节**

大天使加百列向玛利亚显现，宣告上帝已赐她恩典，并将藉着她成就弥赛亚的降生。天使说："圣灵要临到你身上，至高者的能力要荫庇你"（路加福音1:35），玛利亚便由此怀了耶稣。这同一位圣灵，昔日运行在水面上，带出了创造（创世记1:2），如今也要"覆庇"玛利亚子宫中的水，以成就创造主的救赎者之降孕。圣灵临到玛利亚，她"藉着耳朵"怀了耶稣——如马丁·路德所言。那位被怀孕的，被称为圣者、上帝的儿子。这一刻，便是我们主道成肉身的时刻。受胎告知节定于三月二十五日，乃因古代教会相信十字架受难发生于该日。在古代，人们将一人的受孕之日与其死亡之日相连。因此，在受胎告知节中，教会将耶稣的道成肉身与他所成就的救赎合为一体，共同纪念。
