# 十月二十六日　菲利普·尼古拉、约翰·海尔曼和保罗·格哈德，圣诗作家

**诗篇吟唱**

耶和华啊，求你帮助，因为虔诚人断绝了，世人中间的忠信人没有了。
人人都对邻舍说谎，他们说话，嘴唇虚假，心口不一。
凡虚假的嘴唇，夸大话的舌头，耶和华必要剪除。
他们说：我们必用舌头得胜，我们的嘴唇帮助我们，谁能作我们的主呢？
耶和华说：因为困苦人受欺压，贫乏人叹息，我现在要起来，把他安置在他所切望的稳妥之地。
耶和华的言语，是纯净的言语，如同银子在泥炉中炼过七次。
耶和华啊，你必保守他们；你必保护这世代的人，直到永远。
恶人在四围游行，下流人在世人中被尊崇。

——诗篇十二篇

**旧约读经**：申命记二十八章1—22节

**顺服的祝福**

你若留意听从耶和华你神的话，谨守遵行他的一切诫命，就是我今日所吩咐你的，耶和华你的神必使你超乎天下万国之上。你若听从耶和华你神的话，这以下的福必追随你，临到你身上：你在城里必有福，在田间也必有福；你身所生的，地所产的，牲畜所下的，以及牛犊、羊羔，都必有福；你的筐子和你的搓面盆都必有福；你出也有福，入也有福。

耶和华必使那起来攻击你的仇敌在你面前被杀，他们从一条路来攻击你，必从七条路逃跑。耶和华必命令福气临到你，使你的仓房充满，并赐福给你手所办的一切事，耶和华你神所赐你的地，也必赐福给你。你若遵守耶和华你神的诫命，行他的道，他必照所应许你的，立你作为属自己的圣民；天下万民见你归在耶和华的名下，都要惧怕你。耶和华必使你的财物有余，就是你身所生的，你牲畜所下的，并你地所产的，都有余，在耶和华向你列祖起誓应许给你的地上。耶和华必为你开天上的府库，按时降雨在你的地上，赐福给你手所办的一切事，你必借给许多国民，却不致向他们借贷。耶和华必使你作首不作尾，但居上不居下，你若听从耶和华你神的诫命，就是我今日所吩咐你的，谨守遵行，不偏左右，也不随从左右任何事的神，这就必实现了。

**悖逆的咒诅**

你若不听从耶和华你神的话，不谨守遵行他的一切诫命和律例，就是我今日所吩咐你的，这以下的咒诅都必临到你身上，追赶你：你在城里必受咒诅，在田间也必受咒诅；你的筐子和搓面盆都必受咒诅；你身所生的和地所产的，以及牛犊、羊羔，都必受咒诅；你出也受咒诅，入也受咒诅。

耶和华因你所行恶事离弃了他，必使咒诅、扰乱、责罚临到你手所办的一切事上，直到你被灭绝，速速地灭亡。耶和华必使瘟疫贴近你，直到他将你从你所进去得的地上灭绝了。耶和华必用痨病、发热、发炎、极热、刀剑、霉烂、霉病攻击你，这些都必追赶你，直到你灭亡。

**新约读经**：马太福音十八章1—20节

**谁是最大的**

那时，门徒进前来，问耶稣说：天国里谁是最大的？耶稣叫一个小孩子来，使他站在他们当中，说：我实在告诉你们，你们若不回转，变成小孩子的样式，断不得进天国。所以，凡自己谦卑像这小孩子的，他在天国里就是最大的。凡为我的名接待一个像这小孩子的，就是接待我。

凡使这信我的一个小子跌倒的，倒不如把大磨石拴在这人的颈项上，沉在深海里。

**跌倒的因由**

这世界有祸了，因为将人绊倒！绊倒人的事是免不了的，但那绊倒人的有祸了！倘若你一只手，或是一只脚，叫你跌倒，就砍下来丢掉。你缺一只手，或是一只脚，进入永生，强如有两手两脚被丢在永火里。倘若你一只眼叫你跌倒，就剜出来丢掉，你只有一只眼进入永生，强如有两只眼被丢在地狱的火里。

**失羊的比喻**

你们要小心，不可轻看这小子里的一个；我告诉你们，他们的使者在天上，常见我天父的面。你们以为如何？若是一个人有一百只羊，一只走迷了路，他岂不撇下这九十九只，往山里去找那只迷路的羊吗？若是找着了，我实在告诉你们，他为这一只羊欢喜，比为那没有迷路的九十九只欢喜还大呢！你们在天上的父也是这样，不愿意这小子里失丧一个。

**弟兄得罪你**

你的弟兄若得罪你，你就去，趁着只有他和你在一处的时候，指出他的错来；他若听你，你便得了你的弟兄。他若不听，你就另带一两个人同去，要凭两三个人的口作见证，句句都可定准。若是不听他们，就告诉教会；若是不听教会，就看他像外邦人和税吏一样。我实在告诉你们，凡你们在地上所捆绑的，在天上也要捆绑；凡你们在地上所释放的，在天上也要释放。我又告诉你们，若是你们中间有两个人在地上同心合意地求什么事，我在天上的父，必为他们成全。因为无论在哪里，有两三个人奉我的名聚会，那里就有我在他们中间。

**历史与灵修读写**

改革宗和其他教派的错误教导：传道人没有权力代替神赦免罪，只当宣告一般性的罪的赦免。与此相反，路德宗教会按照神的话语（约翰福音二十32；哥林多后书二10；撒母耳记下十二13；马太福音三6；十八17—20）教导：传道人能够并当奉基督的命令、替代基督的位置，赦免那寻求赦免之人的罪，基督徒应当认为"他的罪由此在天上神面前得到了赦免"。因为赦罪"不是在场之人的声音，而是神在此赦罪的话语"。我们路德宗人主要是为了这令人安慰的赦罪而保留了私人认罪，路德曾说，宁愿舍去整个世界也不愿失去这认罪。

——法兰西斯·皮伯

**今日祷文**

全能的神，使徒保罗教导我们要以诗篇、颂歌和灵歌称颂你。我们今日为那些给你的教会带来伟大圣诗的人感谢你，尤其为你的仆人菲利普·尼古拉、约翰·海尔曼和保罗·格哈德。愿你的教会永不缺少圣诗作家，借着他们的文字和音乐向你献上颂赞。求你充满我们渴望颂赞和感谢你无量之善的愿望；借着我们的主耶稣基督，他与你和圣灵同活同治，一神，直到永远。（1102）

**菲利普·尼古拉、约翰·海尔曼和保罗·格哈德，圣诗作家**

菲利普·尼古拉（1556—1608）是德国一位牧师，他任职期间遭遇了大瘟疫，在六个月内夺走了他一千三百位会众的生命。在那段充满压力与悲痛的时日里，除了英勇的牧养事奉之外，他写下了"醒来吧，夜飞的声音响起"和"哦，明晨之星，何等美丽光辉"的歌词，这两首圣诗分别被称为路德宗圣咏的国王和皇后。约翰·海尔曼（1585—1647）也是一位德国牧师，他既身患疾病，又饱受三十年战争（1618—48）的摧残。他的圣诗歌词以温柔和深厚的感情著称。保罗·格哈德（1607—76）是另一位经历了三十年战争恐怖的路德宗牧师。到1668年，他已因拒绝妥协路德宗信念而失去柏林的牧师职位，并忍受了五个子女中四个孩子和妻子的相继离世。然而他仍设法写出了133首圣诗，所有这些圣诗都反映出他坚定的信仰。他与马丁·路德并列，被认为是路德宗最出色的圣诗作家之一。
