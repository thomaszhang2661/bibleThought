# 复活六周六 (Saturday—Easter 6)

**诗篇吟唱**

97我何等爱慕你的律法，
我终日思想这律法。
98你的命令常存在我心，
使我比仇敌有更多的智慧。
99我比我的教师更通达，
因我思想你的法度。
100我比年老的人更明白，
因我遵守了你的训词。
101我禁止我的脚走一切的恶路，
为要遵行你的话。
102我没有偏离你的典章，
因为你教训了我。
103你的话语滋味何等甘甜，
比蜂蜜更甜在我口中！
104借着你的训词，我得了聪明，
所以我恨恶一切虚假的道路。——诗篇119:97—104

附加诗篇：诗篇122

**旧约读经：** 民数记 13:1—3，17—33

**探子被差往迦南**

1耶和华晓谕摩西说：2"你打发人去窥探我所赐给以色列人的迦南地，他们祖宗每支派中，各打发一个首领去。"3摩西就照耶和华的吩咐，从巴兰旷野打发他们去，都是以色列人中的族长。……

17摩西打发他们去窥探迦南地，对他们说："你们从南地上去，爬上山地，18看那地如何，其中所住的民是强是弱，是多是少，19所住之地是好是歹，所住之处是营盘是坚城，20地是肥沃是瘦瘠，其中有树木没有。你们要壮胆，将那地的果子带些来。"那时正是葡萄初熟的时候。

21他们上去窥探那地，从寻的旷野直到利合，就是哈马口。22他们从南地上去，到了希伯仑。在那里有亚纳族人亚希幔、示筛、挞买。希伯仑城建造，比埃及的锁安早七年。23他们到了以实各谷，从那里砍了一枝葡萄树枝，上面有一挂葡萄，两个人用杠抬着，又带了些石榴和无花果。24那地方叫以实各谷，是因以色列人在那里剪了那挂葡萄。

**探子报告**

25四十天后，他们窥探那地回来，25到了巴兰旷野的加低斯，见摩西、亚伦，并以色列全会众，报告摩西和众人，把那地的果子给他们看。27他们对摩西说："我们到了你所打发我们去的那地，果然是流奶与蜜之地，这就是那地的果子。28然而住那地的民强壮，城邑也坚固宽大，并且我们在那里看见了亚纳族的人。29亚玛力人住在南地；赫人、耶布斯人、亚摩利人住在山地；迦南人住在海边并约旦河旁。"

30迦勒在摩西面前安抚百姓，说："我们立刻上去得那地吧，我们足能得胜！"31但那些和他同去的人说："我们不能上去攻击那民，因为他们比我们强壮。"32探子中有人在以色列人面前诽谤所窥探之地，说："我们所窥探经过之地，是吞吃居民之地；我们在那里所看见的人民，都身量高大。33我们在那里看见亚纳族人——他们是伟人的后裔。我们在自己眼中看自己如同蚱蜢一样，在他们眼中也是如此。"

**新约读经：** 路加福音 18:1—17

**坚持不懈的寡妇比喻**

1耶稣设一个比喻，是要人常常祷告，不可灰心。2他说："某城里有一个官，不惧怕神，也不尊重世人。3那城里有个寡妇常来见他，说：'我有一个对头，求你给我伸冤。'4他多次不肯，后来心里说：'我虽不惧怕神，也不尊重世人，5只因这寡妇常来烦扰我，我就给她伸冤吧，免得她常来缠磨我。'"6主说："你们听这不义之官所说的话。7神的选民昼夜呼吁他，他纵然为他们忍了多时，岂不终究给他们伸冤吗？8我告诉你们，要快快地给他们伸冤了。然而，人子来的时候，遇得见世上有信德吗？"

**法利赛人和税吏**

9耶稣向那些仗着自己是义人，藐视别人的，设了这个比喻：10"有两个人上殿里去祷告：一个是法利赛人，一个是税吏。11法利赛人站着，自言自语地祷告说：'神啊，我感谢你，我不像别人勒索、不义、奸淫，也不像这个税吏。12我一个礼拜禁食两次，凡我所得的都捐上十分之一。'13那税吏远远地站着，连举目望天也不敢，只捶着胸说：'神啊，开恩可怜我这个罪人！'14我告诉你们，这人回家去比那人更算为义。因为，凡自高的，必降为卑；自卑的，必升为高。"

**让小孩子到我这里来**

15有人抱着自己的婴孩来见耶稣，要他摸他们；门徒看见就责备那些人。16耶稣却叫他们来，说："让小孩子到我这里来，不要禁止他们，因为在神国的，正是这样的人。17我实在告诉你们，凡不像小孩子接受神国的，断不能进去。"

**历史与灵修读写**

保罗在给哥林多教会的书信中写道："犹太人要神迹，希腊人寻求智慧；我们却是传钉十字架的基督——在犹太人为绊脚石，在外邦人为愚拙，但在那蒙召的，无论是犹太人还是希腊人，基督总为神的能力，神的智慧。"……（保罗）认为只讲基督为神还不够，还必须加上"被钉十字架"。他如此说，是为了坦率扎实地传讲那位被称为"受难者"的基督，乃是神的智慧。……他在传讲主的福音时，并不以提到基督的十字架为耻。尽管对犹太人而言，听到神降生、神取肉身、神受苦、神被钉十架是绊脚石；对外邦人而言是愚拙，他却没有因犹太人的冒犯而削弱自己虔敬言词的力度，也没有因他人的不信与愚昧而减损信心的活力。他公开地、持续地、大胆地宣告：那曾被母亲所生、被人杀害、被矛刺穿、被十字架伸展的那一位，乃是"神的能力与智慧"，"在犹太人为绊脚石，在外邦人为愚拙"。然而那对某些人是绊脚石和愚拙的，对另一些人却是神的能力与智慧。——约翰·卡西安

**圣诗**

主耶稣，愿你宝血如河，
洗净我灵一切罪恶。
主啊，莫将你的客人赶走，
赐我因信称义，
平安回到你面前：
神啊，怜悯我这罪人！
——《全知全能的主》（路德宗崇拜诗集613，第3节）

**当日祷文**

全能永在的主神，你命令我们祷告，又应许垂听我们。求你施恩，使你的圣灵在万事上引导并治理我们的心，使我们以坚定的信心持守对你圣名的认信；借着你的儿子，我们的主耶稣基督，他与你和圣灵同活同治，一神，直到永远。(C82)
