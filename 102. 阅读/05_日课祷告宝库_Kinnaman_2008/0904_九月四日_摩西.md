# 九月四日　摩西

**诗篇吟唱**

耶和华啊，求你转回！你的仆人要等到几时呢？求你使我们早早饱得你的慈爱，好叫我们一生一世欢呼喜乐。
求你照着你使我们受苦的日子，和我们遭难的年岁，叫我们喜乐。
愿你的作为向你仆人显现，愿你的荣耀向他们子孙显明。
愿主我们神的荣美归于我们身上；愿你坚立我们手所做的工；我们手所做的工，愿你坚立。

——诗篇九十篇十三—十七节

**旧约读经**：列王纪下二章一—十八节

**以利亚被接升天**

耶和华要用旋风接以利亚升天的时候，以利亚与以利沙从吉甲前行。以利亚对以利沙说："耶和华差遣我往伯特利去，你可以在这里等候。"以利沙说："我指着永生的耶和华，又敢在你面前起誓，我必不离开你。"于是二人下到伯特利。在伯特利的先知门徒出来见以利沙，对他说："你知道耶和华今日要把你的师傅从你头上取去吗？"以利沙说："我知道，你们不要作声。"

以利亚对他说："耶和华差遣我往耶利哥去，你可以在这里等候。"以利沙说："我指着永生的耶和华，又敢在你面前起誓，我必不离开你。"于是二人到了耶利哥。在耶利哥的先知门徒就近以利沙，对他说："你知道耶和华今日要把你的师傅从你头上取去吗？"以利沙说："我知道，你们不要作声。"

以利亚对他说："耶和华差遣我往约旦河去，你可以在这里等候。"以利沙说："我指着永生的耶和华，又敢在你面前起誓，我必不离开你。"于是二人同往。先知门徒中有五十个人去，远远地站在他们对面；二人在约旦河边站住。以利亚将自己的外衣卷起来，用以打水，水就左右分开，二人走干地而过。

过去之后，以利亚对以利沙说："我未曾被接去离开你之先，你要我为你做什么，只管求我。"以利沙说："愿感动你的灵加倍地感动我。"以利亚说："你所求的难得；虽然如此，我被接去离开你的时候，你若看见我，就必得着；你若看不见，就得不着了。"他们正走着说话，忽有火车火马将二人隔开，以利亚就乘旋风升天去了。以利沙看见，就呼叫说："我父啊！我父啊！以色列的战车马兵啊！"以后不再见他了。以利沙就抓住自己的衣服，撕为两片，又拾起以利亚身上掉下来的外衣，回去站在约旦河边，用以利亚身上掉下来的外衣打水，说："耶和华——以利亚的神——在哪里呢？"他打水之后，水也左右分开，以利沙就过去了。

**以利沙承继以利亚**

在耶利哥的先知门徒从对面看见他，说："感动以利亚的灵降在以利沙身上了。"他们就来迎接他，在他面前俯伏于地，对他说："你的仆人中有五十个壮士，请让他们去寻找你的主人，或者耶和华的灵将他提起来，丢在某山或某谷了。"以利沙说："不要差遣他们去。"他们再三催逼他，他就难以推辞，说："你们去吧。"他们就差了五十个人去，寻找了三天，找不着他，回来见以利沙——他还住在耶利哥——他对他们说："我岂没有告诉你们不要去吗？"

**新约读经**：以弗所书四章一—二十四节

**在基督身体里合而为一**

我为主被囚的劝你们：既然蒙召，行事为人就当与蒙召的恩相称，凡事谦虚、温柔、忍耐，用爱心互相宽容，用和平彼此联络，竭力保守圣灵所赐合而为一的心。身体只有一个，圣灵只有一个，正如你们蒙召，同有一个指望；一主，一信，一洗，一神，就是众人的父，超乎众人之上，贯乎众人之中，也住在众人之内。

我们各人蒙恩，都是照基督所量给各人的恩赐。所以经上说："他升上高天的时候，掳掠了仇敌，将各样的恩赐赏给人。"既说升上，岂不是也降到地下吗？那降下的，就是远升诸天之上的，为要充满万有。他所赐的，有使徒，有先知，有传福音的，有牧师和教师，为要成全圣徒，各尽其职，建立基督的身体，直等到我们众人在真道上同归于一，认识神的儿子，得以长大成人，满有基督长成的身量；使我们不再作小孩子，中了人的诡计和欺骗的法术，被一切异教之风摇动，飘来飘去，就随从各样的异端；惟用爱心说诚实话，凡事长进，连于元首基督。

**新生活**

所以我说，且在主里确实地说，你们行事不要再像外邦人，存虚妄的心行事。他们心地昏昧，与神所赐的生命隔绝了，都因自己无知，心里刚硬；良心既然丧尽，就放纵私欲，贪行种种的污秽。你们学了基督，却不是这样；如果你们听过他的道，领了他的教，学了他的真理，就要脱去你们从前行为上的旧人，这旧人是因私欲的迷惑渐渐变坏的；又要将你们的心志改换一新，并且穿上新人；这新人是照着神的形象造的，有真理的仁义和圣洁。

**历史与灵修读写**

亲爱的弟兄们，我用不亚于蒙福使徒彼得的热切之词来称呼你们："你们是蒙拣选的族类，是有君尊的祭司，是圣洁的国度，是属神的子民"——你们建立在磐石基督这坚不可摧的根基上，借着他对我们肉身的真实承担，与我们的救主主耶稣联合——你们要坚守信仰，就是那在许多见证人面前你们所承认的，也是你们藉着水和圣灵重生、受恩膏得救、并领受永生印记的信仰；但"若有人传福音给你们，与你们所领受的不同，他就应当被咒诅"；拒绝将邪恶的寓言置于最清楚的真理之上；凡你们所读到或听到违背大公使徒信经之规范的，都要断然判断为致死的、魔鬼的……他们披上敬虔贞洁的外衣，却以此掩盖其行为的污秽，从不虔不义之心的深处射出箭来，伤害纯朴的人……一个坚固的堡垒，就是健全的信仰、真实的信仰，其中无物可加，也无物可减，因为若不是唯一的，便不是信仰，正如使徒所言："一主，一信，一洗，一神，就是众人的父，超乎众人之上，贯乎众人之中，也住在众人之内。"亲爱的弟兄们，要心志坚定地持守这合一，在其中"追求圣洁"。在其中遵行主的诫命，因为"人非有信，就不能得神的喜悦"；没有信仰，便没有什么是圣洁的，没有什么是纯洁的，没有什么是有生命的——"义人必因信得生"，被魔鬼欺骗而失去信仰的人，活着也如同死去，因为正如义是藉着信得来的，同样，永生也是藉着真实的信仰得来的，正如我们的主和救主所说：永生就是认识你，独一的真神，并且认识你所差来的耶稣基督。愿他使你们进步、坚守到底——他与父和圣灵同活同王，直到永远。阿们。

——大利奥

**圣诗**

求你帮助我们更多事奉你，以纯洁谦卑的心；愿你神圣的道，那光明，在圣洁的光辉中持续照耀，使我们悔改显明，在信心中常常成长；消灭罪恶的权势，以及一切烦扰之事。求你使我们成为忠心的基督徒。

——《主耶稣基督，教会的头》（路德宗崇拜书 647:3）

**摩西**

摩西生于埃及，是约瑟领其父雅各和众兄弟来埃及避饥荒之后数代人的后裔。雅各的后代已被埃及人奴役，并被命令杀死所有男婴。摩西出生时，他的母亲将他放在篮子里，放入尼罗河中漂流。他被法老的女儿发现，被收为养子抚养成人（出二：一—十）。四十岁时，摩西杀了一个埃及监工，逃往米甸地，在那里做了四十年牧人。后来耶和华呼召他回到埃及，对法老说："容我的百姓去，使他们可以在旷野向我守节"（出五：一）。法老最终屈服，在以色列民守了第一次逾越节之后，摩西领他们出了埃及。在红海边，埃及大军被消灭，以色列民得以在干地上安然过去（出十二—十五章）。在西奈山，他们领受了律法，建造了会幕（出十九—四十章）。但因悖逆，他们不得不在旷野漂流四十年。摩西本人虽未被允许进入应许之地，但神让他从远处眺望了那片土地（申三十四章）。在新约中，摩西被称为律法的颁布者与先知。圣经前五卷书被认为出自他手。
