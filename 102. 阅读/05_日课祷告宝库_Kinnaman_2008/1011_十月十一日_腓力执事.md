# 十月十一日　腓力执事

**诗篇吟唱**

1耶和华是我的牧者，我必不至缺乏。
2祂使我躺卧在青草地上，领我在可安歇的水边。
3祂使我的灵魂苏醒，为自己名的缘故，引导我走义路。
4我虽然行过死荫的幽谷，也不怕遭害，因为你与我同在；你的杖、你的竿都安慰我。
5你在我敌人面前为我摆设筵席；你用油膏了我的头，使我的福杯满溢。
6我一生一世必有恩惠慈爱随着我；我且要住在耶和华的殿中，直到永远。
——诗篇二十三篇

**旧约读经**：申命记九章二十三节—十章二十二节

**金牛犊事件**

23"耶和华从加低斯巴尼亚打发你们去，说：'你们上去，得我所赐给你们的地。'那时你们违背了耶和华你们神的命令，不信服祂，不听从祂的话。24你们从我认识你们的那日起，就常常悖逆耶和华了。

25"耶和华说要灭绝你们，所以我在耶和华面前俯伏了那四十昼夜。26我祷告耶和华说：'主耶和华啊，求你不要灭绝你的百姓——就是你的产业，你用大能救赎的、用大力从埃及领出来的。27求你记念你的仆人亚伯拉罕、以撒、雅各，不顾这百姓的顽梗、奸恶与罪过，28免得你领我们出来的那地之人说："耶和华不能将他们领进祂所应许之地，又因恨他们，所以领他们出去，要在旷野杀他们。"29其实他们是你的百姓，你的产业，是你用大能和伸出来的膀臂领出来的。'

**新石版**

10:1"那时耶和华对我说：'你要凿出两块石版，与原先的一样，又要做一个木柜，然后上山到我这里来。2我要在这版上写那两块初版上所写的话，就是你打碎了的；你要把这版放在柜里。'3于是我用皂荚木造了一个柜，凿出两块石版与原先的一样，手拿两块石版上了山。4耶和华照先前所写的，将那十条诫命写在版上，就是祂在山上从火中传给你们的话；耶和华就把版交给我了。5我转身下山，将版放在我所造的柜里，耶和华怎么吩咐我，柜就在那里了。"

6（以色列人从比尼亚干的比录起行，到了摩西拉；亚伦死在那里，也葬在那里，他儿子以利亚撒接续他供祭司职分。7他们从那里起行，到了谷歌大，又从谷歌大到了约巴他；那地方有许多水溪。8那时耶和华分派利未支派抬耶和华立约的柜，又侍立在耶和华面前事奉祂，奉祂的名祝福，直到今日。9因此利未人在弟兄中没有分，也没有产业；耶和华你的神所应许他的，耶和华是他的产业。）

10"我在山上住了四十昼夜，与头一次相同，耶和华这次也垂听了我的祷告，不肯灭绝你们。11耶和华对我说：'你起来，率领百姓前行，使他们进去，得我向他们列祖起誓应许赐给他们的地。'

**割去心里的包皮**

12"以色列啊，现在耶和华你神向你所要的是什么呢？只要你敬畏耶和华你的神，遵行祂的道，爱祂，尽心尽性事奉祂，13并且遵守祂的诫命律例，就是我今日所吩咐你的，为要叫你得福。14看哪，天和天上的天，地和地上所有的，都属耶和华你的神。15耶和华专爱你们的列祖，从万民中拣选他们的后裔，就是你们，直到今日。16所以你们要将心里的污秽除掉，不可再硬着颈项。17因为耶和华你们的神，祂是万神之神、万主之主，至大的神，大有能力，令人畏惧，不以貌取人，也不受贿赂，18为孤儿和寡妇伸冤，又怜爱寄居的，赐给他衣食。19所以你们要怜爱寄居的，因为你们在埃及地也作过寄居的。20你要敬畏耶和华你的神，事奉祂，紧靠着祂，当指着祂的名起誓。21祂是你所当颂赞的；祂是你的神，为你行了你亲眼所见的大事，可畏的事。22你列祖下埃及的时候共有七十人；现在耶和华你的神使你的人数多如天上的星。"

**斥责不悔改的城市**

20耶稣就在各城中责备人，因为他们不悔改，那些异能大半是在那里行的。21"哥拉汛哪，你有祸了！伯赛大啊，你有祸了！因为在你们中间所行的异能，若行在推罗、西顿，他们早已披麻蒙灰悔改了。22但我告诉你们，当审判的日子，推罗、西顿所受的，比你们还容易受呢！23迦百农啊，你已经升到天上，将来必坠落阴间；因为在你那里所行的异能，若行在所多玛，它还可以存到今日。24但我告诉你们，当审判的日子，所多玛所受的，比你还容易受呢！"

**到我这里来，我必使你们得安息**

25那时，耶稣说："父啊，天地的主，我感谢你！因为你将这些事向聪明通达人就藏起来，向婴孩就显出来。26父啊，是的，因为你本意如此。27我父将一切所有的交给我了；除了父，没有人知道子；除了子和子所愿意指示的，没有人知道父。28凡劳苦担重担的人，可以到我这里来，我就使你们得安息。29我心里柔和谦卑，你们当负我的轭，学我的样式；这样，你们心里就必得享安息。30因为我的轭是容易的，我的担子是轻省的。"

**历史与灵修读写**

我们活在世上，就免不了受苦与受试探。约伯记上写道："人在世上岂无争战？"每个人都必须防备试探，儆醒祷告，免得魔鬼趁机欺哄他——魔鬼从不安睡，乃四处游行，寻找可吞噬的人。无论何人，无论多完全、多圣洁，都难免受到试探；人不可能完全脱离试探。

然而试探虽令人苦恼、严峻，对人却往往大有益处，因为人在其中得以谦卑、洁净，并受教训。众圣徒历经许多试炼，从中获益；凡不能抵挡者，则成为弃绝之人，终至沉沦。无论何等圣洁的处境，无论多么隐秘的所在，试探与磨难终究会临到。人活一日，就有一日的试探；因为试探从我们自身而来——我们生在罪中。一个试探或磨难一过，另一个又来；我们必总有当受之苦，因为我们已失去了原初恩福的状态。……

一切试探的根源，在于心志动摇、对神信心不足：正如无舵的船被波浪东荡西飘，粗心大意、优柔寡断的人，便以种种方式受到试探。火锤炼铁，试探则坚固义人。我们往往不知自己能承受什么，唯有试探显明我们究竟是什么样的人。……

当我们受试探时，不可灰心绝望，而当越发热切地祷告神，求祂施恩帮助我们；因为按照保罗的话，祂必与试探同开出路，使我们能承担得住。
——托马斯·肯培

**圣诗**

我听见耶稣声音说：
"来到我这里安歇；
疲惫的人啊，俯伏下来，
将头枕在我胸怀。"
我就这样来到耶稣跟前，
疲乏、困倦、愁苦满怀；
在祂里面我寻得安息之所，
祂使我心里充满喜乐。
——《我听见耶稣声音》（路德宗崇拜书 699:1）

**今日祷文**

全能永恒的神，我们感谢你，因你呼召你的仆人腓力执事向撒玛利亚与埃塞俄比亚的众民传扬福音。求你在这地与万地兴起你国度的传信使者，使你的教会能宣扬我们救主耶稣基督无可测量的丰富恩典；祂与你和圣灵同活同王，从今时直到永远。（1098）

**腓力执事**

腓力又称传福音者（徒二十一章八节），是被选立协助十二使徒及迅速成长之初期教会的七人之一，负责监管向贫苦者分发食物的事务（徒六章一—六节）。司提反殉道后，腓力在撒玛利亚传扬福音，并引领行邪术的西门成为相信基督的信徒（徒八章四—十三节）。他在促成埃塞俄比亚太监归信一事上也功不可没（徒八章二十六—三十九节）；借着这位太监，腓力间接地将耶稣的好消息带给了非洲大陆上的众民。在该撒利亚城，他曾接待使徒保罗在他家住了数日；那时保罗正途经此地，赶赴耶路撒冷的最后一程（徒二十一章八—十五节）。
