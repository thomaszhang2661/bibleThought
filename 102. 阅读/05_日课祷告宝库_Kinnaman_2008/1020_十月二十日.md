# 十月二十日

**诗篇吟唱**

耶和华啊，我要全心称谢你；在诸神面前歌颂你。
我要向你的圣殿下拜，为你的慈爱和诚实称赞你的名；因你使你的话显为大，过于你所有的名。
我呼求的日子，你就应允我，鼓励我，使我心里有能力。
耶和华啊，地上的君王都要称谢你，因他们听见了你口中的话。
他们要歌颂耶和华的作为，因为耶和华大有荣耀。
耶和华虽高，仍看顾低微的人；他却从远处看出骄傲人。
我虽然行在患难中，你必将我救活；我的仇敌发怒，你必伸手抵挡他们，你的右手也必救我。
耶和华必为我成全他的旨意；耶和华啊，你的慈爱永远长存，求你不要离弃你手所造的。

——诗篇一百三十八篇

**旧约读经**：申命记十九章1—20节

**逃城的律法**

耶和华你神剪除了那些国民，就是耶和华你神所赐你为业之地的国民，你从他们那里得了地，住在他们的城邑并房屋里。在耶和华你神赐你为业的地上，你要分出三座城。你要测量那地，将全地分为三段，使杀人的都可以逃到那里去。

逃城的律例是这样：凡误杀了人，素无仇恨，可以逃到那里存活——就如人与邻舍同入树林砍伐树木，手拿斧子一砍，斧头脱柄，飞落在邻舍身上，以致于死——这人逃到那些城的一座城，便可以存活，免得报血仇的，心中火热追赶，因路远追上，把他杀死，其实他不该死，因为他素来与邻舍没有仇恨。所以我吩咐你分出三座城来。

你若遵守我今日所吩咐你的一切诫命，照着去行，就是爱耶和华你的神，常走他的道，耶和华你的神就在应许你列祖之外，再扩张你的境界，——耶和华若按所起的誓，将那全地都给了你——你还要在那三座城之外，再添三座城，免得流无辜人的血，使那罪归于你。

若有人恨他邻舍，埋伏，起来击打他，打伤要命，逃入那些城的一座城，本城的长老就要打发人，将他从那里带回来，交在报血仇人的手中，把他治死。你眼不可怜惜他，却要从以色列中除掉无辜人的血，使你得福。

**地界的律法**

你的神耶和华赐你承受的地，前人所立的地界，你不可挪移。

**见证人的律法**

无论何事，无论何种罪，一个人的口不可以定罪，要凭两三个人口作见证，才可以定罪。若有恶意的见证人起来，见证某人作了什么恶，这两个争竞的人，要站在耶和华面前，并当时的祭司和审判官面前。审判官要细细地查究；那见证人若是作假见证，以假见证陷害弟兄，你们就要待他如同他想要待弟兄的一样，把那恶从你们中间除掉。其余的人听见都要害怕，不可再在你们中间行那样的恶事。

**新约读经**：马太福音十五章1—20节

**传统与诫命**

那时，有法利赛人和文士从耶路撒冷来见耶稣，说：你的门徒为什么犯古人的遗传呢？因为吃饭的时候，他们不洗手。耶稣回答说：你们为什么因着你们的遗传，犯神的诫命呢？神说：当孝敬父母；又说：咒骂父母的，必治死他。你们倒说：无论何人对父母说：我所当奉给你的，已经作了供献，他就可以不孝敬父母。这就是你们借着遗传，废了神的诫命。假冒为善的人哪，以赛亚指着你们说的预言，是不错的。他说：这百姓用嘴唇尊敬我，心却远离我；他们将人的吩咐，当作道理教导人，所以拜我也是枉然。

**什么使人污秽**

耶稣叫了众人来，对他们说：你们听，也要明白。入口的不能污秽人，出口的乃能污秽人。于是门徒进前来对他说：法利赛人听见这话，不服，你知道吗？耶稣回答说：凡我天父所没有栽种的，必要拔出来。任凭他们吧！他们是瞎眼领路的；若是瞎子领瞎子，两个人都要掉在坑里。彼得说：请将这比喻讲给我们听。耶稣说：你们到如今还不明白吗？岂不知凡入口的，是运到肚子里，又落到茅厕里吗？惟独出口的，是从心里发出来的，这才污秽人。因为从心里发出来的，有恶念、凶杀、奸淫、苟合、偷盗、妄证、谤讟，这都是污秽人的；至于不洗手吃饭，那却不污秽人。

**历史与灵修读写**

我们的教会曾教导：我们不能借遵守人的传统来赚取恩典或称义。我们不应以为这些遵守是必要的敬拜行为。在此，我们添加圣经的见证。基督为那些未遵守通常传统的使徒辩护（马太福音十五3）。这涉及的事情并不违法，而是既未被命令也未被禁止，类似于律法中的洁净礼仪。他在马太福音十五章9节说："他们将人的吩咐，当作道理教导人，所以拜我也是枉然。"因此，他不要求无用的人为服侍。紧接着他又说："入口的不能污秽人，出口的乃能污秽人"（马太福音十五11）。保罗在罗马书十四章17节也说："神的国不在乎吃喝，只在乎公义、和平，并圣灵中的喜乐"；又在歌罗西书二章16节说："所以，不拘在饮食上，或节期、月朔、安息日，都不可让人论断你们。"又说："你们若是与基督同死，脱离了世上的小学，为什么仍像在世俗中活着，服从那不可拿、不可尝、不可摸等类的规条呢"（歌罗西书二20—21）。彼得说："现在为什么试探神，要把我们祖宗和我们所不能负的轭，放在门徒的颈项上呢？我们得救，乃是因主耶稣的恩，和他们一样，这是我们所信的"（使徒行传十五10—11）。彼得在此禁止用许多礼仪——无论来自摩西还是其他——来加重人的良心负担。保罗在提摩太前书四章1—3节称禁止吃肉为魔鬼的道理。制定或推行这些行为，以为能藉此赚取恩典，或以为没有这种神事服侍基督教就不能存在，这与福音是相悖的。

——《奥格斯堡信条》第二十六章21—29节

**圣诗**

主啊，在我心中创造一颗新心，使我甘愿遵行你的话语。
愿你所喜悦的，成为我的渴望，用新生命激励我的灵魂。

——《更新我，永恒之光》（路德宗崇拜书 704:3）
