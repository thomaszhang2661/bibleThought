# 三月三十一日　约瑟列祖（节日）

**历史与灵修读写**

我愿向诸位讲述一则富有教益、大可获益的故事。所讲者何？

昔日，一场迫害骤然兴起，一场猛烈的战争向教会袭来，有两人被捕。其中一人，准备好承受任何苦难；另一人，虽下定决心以坚忍之志受死于刀斧之下，却对其他刑讯惧怕颤抖、退缩畏避。那么，事情结果如何呢？当审判官升堂就坐，他命令那个准备承受任何苦难的人以斩首处死；而对另一人，他却命人将其吊起施以拷打，且不止一次两次，而是辗转城城，屡加折磨。

此事何以被允许？乃是为使那人能借着酷刑恢复其所忽略的心志品质，使他能抖落一切怯懦，不再惧怕任何苦难。约瑟亦然——他本想逃出监狱，却被留在其中。你听他所说的话："我实在是从希伯来人之地被拐带来的；请你在王面前提到我。"正因此事，他被留在那里，好使他学会不将盼望与信心寄托于人，乃要将一切都托付于上帝。

既知此事，我们就当感谢上帝，并当行一切于我们有益之事，使我们借着我们的主耶稣基督获得将来的美善——愿荣耀归于圣父，并归于圣灵，从今时直到永远，世世无尽。阿们。

——金口约翰·屈梭多模（John Chrysostom）

**约瑟列祖**

约瑟是列祖雅各（二月五日）与拉结之子。他是父亲最钟爱的儿子，因此招致兄长们的妒恨；兄长们将他卖为奴隶，带往埃及，并告诉父亲他已身亡（创世记37章）。在埃及，约瑟成为军官波提乏家中的总管仆人。因约瑟拒绝与主人之妻行淫，他遭受不白之冤，被诬以图谋强暴之罪，随即被投入监狱（创世记39章）。数年之后，他为法老解梦，法老遂将约瑟从狱中释放，立他掌管全国事务。当他的兄长们从迦南来到埃及寻粮，他们未曾认出约瑟。他终于向他们表明自己的身份，宽恕了他们，并邀请兄长们与父亲一同迁居埃及。约瑟因其道德上的高洁正直（创世记39章）以及他甘愿宽恕兄长的胸怀（创世记45章与50章），被人们长久铭记与尊崇。
