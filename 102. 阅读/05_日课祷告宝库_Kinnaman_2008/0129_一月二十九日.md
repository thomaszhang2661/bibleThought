# 一月二十九日

**诗篇吟唱**

你们要称谢耶和华，因祂本为善，祂的慈爱永远长存。
愿耶和华的赎民说这话，就是祂从敌人手中所救赎的，从各地所聚集来的，从东从西，从南从北。
有人在旷野荒凉之地漂流，找不着可住的城邑；
他们又饥又渴，心里发昏。
于是他们在苦难中呼求耶和华，祂从他们的祸患中搭救他们；
祂领他们行走直路，使他们到可居住的城邑。
愿他们因耶和华的慈爱和祂向世人所行的奇事称谢祂；
因为祂使心里渴慕的人得以知足，使心里饥饿的人得饱美物。

——诗篇一百零七篇一至九节

**旧约读经**：撒迦利亚书八章一至二十三节

**锡安将来的平安与昌盛**

万军之耶和华的话临到我说："万军之耶和华如此说：我为锡安心里极其火热，我为她发烈怒。"耶和华如此说："我已归回锡安，要住在耶路撒冷中；耶路撒冷必称为忠信的城，万军之耶和华的山必称为圣山。"万军之耶和华如此说："必有年老的男女坐在耶路撒冷街上，各人因年纪老迈手拄拐杖，城中的街道必满有男孩女孩在其中玩耍。"万军之耶和华如此说："这事在这民剩余的人眼中，看为希奇，在我眼中，岂也看为希奇吗？这是万军之耶和华说的。"万军之耶和华如此说："我要从东方和西方救我的子民，领他们来居住在耶路撒冷中，他们要做我的子民，我要做他们的神，以诚实和公义待他们。"

万军之耶和华如此说："现今你们当手脚有力，就是你们这些听见这话的，是先知们在立万军之耶和华殿的根基，要建造圣殿那日所说的话。在这事以前，人没有工价，牲畜也没有工价；因敌人的缘故，出入没有平安；那时我使人各攻击其邻舍。但现在我对于这民剩余的人，必不像从前。这是万军之耶和华说的。因为必有平安的种：藤必结果子，地必有出产，天必有甘露；我必赐这一切给这民剩余的人承受。犹大家和以色列家啊，你们从前在列国中怎样成了咒诅，照样，我要救你们，叫你们成为福气，不要惧怕，手要有力量。"

万军之耶和华如此说："我怎样想加祸与你们的列祖惹我发怒，并不后悔，万军之耶和华如此说，照样，我在这些日子又想赐福与耶路撒冷和犹大家，不要惧怕。你们所当行的事乃是这样：各人与邻舍说诚实话，在你们的城门中，按真实和睦的事判断；不可心里谋害邻舍，也不可喜爱起假誓，因为这些事都为我所恨恶。"这是耶和华说的。

万军之耶和华的话临到我说："万军之耶和华如此说：四月的禁食，五月的禁食，七月的禁食，十月的禁食，必变为犹大家欢喜快乐的事，并为欢乐的节期，所以要喜爱诚实与和平。"

万军之耶和华如此说："将来必有列国的人民和多城的居民来到，这城的居民必到那城，说：'我们要快去恳求耶和华的恩，寻求万军之耶和华；我自己也去。'必有许多国民和强盛的邦国来耶路撒冷，寻求万军之耶和华，恳求耶和华的恩。"万军之耶和华如此说："在那些日子，必有十个人从列国诸族，拉着一个犹大人的衣裳，说：'我们要与你们同去，因为我们听见神与你们同在了。'"

**新约读经**：提摩太后书一章一至十八节

**问安**

奉神旨意，照着在基督耶稣里生命的应许，做耶稣基督使徒的保罗，写信给亲爱的儿子提摩太：愿恩惠、怜悯、平安从父神和我们主基督耶稣归于你。

**守住所托付的**

我感谢神——就是我照着祖先的传统，以清洁的良心所事奉的神——祈求的时候，不住地在祷告中想到你，记念你的眼泪，切切地想见你，好叫我满心欢喜。想到你心里无伪之信，这信是先在你外祖母罗以和你母亲友尼基心里的，我深信也在你的心里。为此我提醒你，使神藉我按手所给你的恩赐，再如火挑旺起来，因为神赐给我们，不是胆怯的心，乃是刚强、仁爱、谨守的心。

你不要以给我们的主作见证为耻，也不要以我这为主被囚的为耻；总要按神的能力，与我为福音同受苦难。神救了我们，以圣召召我们，不是按我们的行为，乃是按祂的旨意和恩典；这恩典是万古之先，在基督耶稣里赐给我们的，但如今藉着我们救主基督耶稣的显现才表明出来了。祂已经把死废去，藉着福音，将不能朽坏的生命彰显出来。我为这福音奉派作传道者，做使徒，做教师，也因这缘故受这些苦难。然而我不以为耻，因为知道我所信的是谁，也深信祂能保全我所交付祂的，直到那日。你从我听的那纯正话语的规模，要用在基督耶稣里的信心和爱心持守，并且要藉着那住在我们里面的圣灵，把那美好的托付看守好了。

你知道亚细亚的人都离弃了我，其中有腓吉路和黑摩其尼。愿主怜悯阿尼色弗一家的人，因他屡次使我畅快，不以我的锁链为耻，反倒到了罗马，急切地找我，并且找到了——愿主使他在那日得主的怜悯！——他在以弗所怎样多方地服侍我，是你明明知道的。

**历史与灵修读写**

这是对先前应许的证实，其意乃是说："我现在所应许你们的，在你们看来似乎不可能，因为你们人数甚少，事情进展如此迟缓。但我的话是真实的，因为我愿意也能够使你们的人数增多，就是这样：我要从东方和西方，从被掳和流散之处，把我的子民带到这里，使他们住在耶路撒冷。他们要做我的子民，我要做他们的神，以诚实和公义待他们；就是说，他们要以真实的信心和生命事奉我，而我也要向他们显明我是他们真实的神，以慈悯和帮助待他们，不再以我从前的愤怒和刑罚对待他们，正如从前他们不愿以诚实和公义做我的子民的时候一样。"

这里要特别留意"诚实和公义"这两个词。因为神不喜欢虚伪之人，也不愿意做恶人的神。那些人确实希望祂是一位赐予富贵荣华之神，就是使他们在世人眼中富裕荣耀——至于虔诚与公义，任其自然。不，祂不是玛门，也不是腹之神。祂愿意在诚实和公义中做神，也愿意有一个爱诚实和公义的子民。然而，世俗的财货确实也要来临，这将在后面论及。"诚实"是指百姓在神面前是公义的，不是虚伪的，他们以真实的信心和诚意事奉神；"公义"在这里是指他们对待邻舍的爱与怜悯。整个圣洁子民的生命，都取决于这两条诫命。

——马丁·路德

**路德宗信条建议阅读**

大问答第五部分，第三十一至四十一段

31所以，若说：基督的身体和血在主晚餐内未曾为我们舍而流出来，故此在圣礼内，我们不能享有罪的赦免便为荒谬话。 虽然〔基督〕在十字架上成了这工，它仍不能以他法临到我们，必须藉圣道〔接受之〕。若不以口传扬上帝的道宣告它，我们焉能知道这工作业已完成，并提供给我们呢？除了恒心相信圣经和福音，他们怎知有赦免，怎能明白而将它归己呢？32全福音及信经条款：“我信一圣基督教会……罪得赦免，”皆包含在此圣礼内面，并藉着圣道提供给我们。那么，我们为何要让人从圣礼中将此珍宝剔除出来呢？我们的对敌〔慈运理派与重洗者〕仍须承认这〔为你舍，为你流〕话正是我们在福音内各处可听到的。他们所说这些在圣礼内的话毫无价值之言毫无理由，正如他们不敢说在圣礼之外，全福音或上帝的道皆无价值。

33至此我们曾由圣礼的本质及其结果与利益之观点讨论它。我们仍要考虑谁领受这能力与利益。简言之，像我们以上关于洗礼及其他许多地方所说，回答：是相信那些话所讲和其要施予之恩赐的人，因为这些话不是对石头和木块讲说，而是对那些听这些话的人讲的，基督对那些人说；“拿着吃，”等。34并因祂应许提供罪的赦免，不能在信心之外领受它。祂亲自在圣道内要求这信仰，说：“为你们舍，而为你们流出来的，”好像祂在说：“这是我施与它的原因，并要求你们吃喝，使你们能拿着它，当自己所有的享受它。” 35谁若认为这些话是对他所说，且相信字句中所讲的都是真的，才享有这话中所说明的。但那不相信的人则一无所有，因为他让这恩惠的祝福白白地向他提供，拒绝享受它。这珍宝打开了，安放在每个人的大门口，而且摆在每个人的桌子上，你也有责任拿它，并坚信它正像那些话所告诉你的一样。

36所以，这乃是基督徒需要配领此圣礼的预备。既然这些话中完全提供了此珍宝，人只能以心把握它归己有。此种恩赐与永远珍宝不能凭手抓牢。37禁食及祷告等，可能仅是一种外表预备与孩童的训练，使人的身体对基督身体与血，能有合宜恭敬的态度。但在圣礼内，且同着圣礼所赐的，人不能以身体抓住而归己。乃是认识并渴望此珍宝的人心之信仰信靠所为。

38此圣礼基本之所有普通教训已充分讲明。若有更多要说明的话，只有等其他的机会了。

39最后，我们已有了圣礼的正确解释与道理，也急需向人劝勉恳求在基督徒中间日日举行分递此大珍宝，不致让它被人忽略不重视。我的意思是：愿作基督徒的人，应自己预备屡次领受此可尊圣礼。40因我们看到有人对于圣礼成了无精打采懒惰的样子。许多听福音的人，当现今教皇的胡言被除去，我们脱离了他的压制和威权，一年、二年、三年，或过了很多年不领圣礼，好似他们是很坚强的基督徒，可不需要领圣餐。41有的人阻止自己领受它，因我们曾教导：除非人感觉饥渴躯使他领圣餐，就不要去领。有些人借口称它是一种自由事情，并非必需，只要他们相信便已足够。如此大多数信徒甚至变的十分野蛮，终究要藐视圣礼及上帝的道。
