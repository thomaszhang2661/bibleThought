# 一月二十八日

**诗篇吟唱**

耶和华啊，求祢帮助，因虔诚人断绝了；世人中间的忠信人没有了。
人人对邻舍说谎，他们说话，嘴唇油滑，心口不一。
耶和华必剪除一切油滑的嘴唇和说大话的舌头——就是那些说："我们要以口取胜，我们有嘴，谁能治服我们呢？"
耶和华说："因为困苦人受欺压，贫穷人哀哭，我现在要起来，把他放在他所切慕的安全之地。"
耶和华的言语，是纯净的言语，如同坩埚里经过熔炼精制七次的银子。
耶和华啊，祢必保护他们；祢必保佑他们这一代，免受恶人的害。
凶恶之人到处游行，卑鄙的事在世人中间被称扬。

——诗篇十二篇

**旧约读经**：撒迦利亚书六章一节至七章十四节

**四车异象**

我又举目观看，见有四辆战车从两山中间出来；那山是铜山。第一辆车套着红马，第二辆车套着黑马，第三辆车套着白马，第四辆车套着有斑点的马，都是壮马。我问与我说话的天使说："主啊，这是什么意思？"天使回答我说："这是四方的风，刚从天上的主那里站立，要出去巡察大地。那套黑马的车往北方去；白马随后而去；有斑点的马往南方去。"那些壮马出来，急着要往各地去巡察大地；天使说："你们去巡察大地吧。"它们就巡察了大地。他向我呼喊说："看哪，往北方去的马已经使我的灵在北方安息了。"

**冠冕与圣殿**

耶和华的话来到我这里说："要从被掳归回的人中收取礼物，就是从希太、托比雅、耶大雅那里；当日你去到西番雅之子约西亚的家里。要把金和银收下，制造冠冕，戴在约撒达的儿子大祭司约书亚的头上，对他说：'万军之耶和华如此说：看哪，那名叫大卫苗裔的人，他要从他的地方生长出来，建造耶和华的殿。他要建造耶和华的殿，他要担负尊荣，坐在宝座上掌权，在宝座上作祭司，两职之间必有协商的关系。'"那冠冕要放在耶和华的殿里，作为希冷、托比雅、耶大雅和西番雅之子恩的纪念。

"远方的人必来建造耶和华的殿；你们就知道是万军之耶和华差遣我来到你们这里了。你们若殷勤听从耶和华你们神的话，这事就必成就。"

**公义与慈悯的呼召**

大流士王第四年的九月，就是基斯流月初四日，耶和华的话来到撒迦利亚这里。原来伯特利的人打发沙利色和利坚米勒及随从的人，恳求耶和华的恩典，并问祭司和先知说："我历年来在五月间哭泣禁食，现在还当这样行吗？"万军之耶和华的话就来到我这里说："你告诉这地的众民和祭司说，你们七十年来在五月、七月禁食悲哀，岂是丝毫向我禁食吗？你们吃喝，不是为自己吃、为自己喝吗？当耶路撒冷有居民、安然居住，周围有城邑，南地和高原有人居住的时候，耶和华藉从前的先知所宣告的话，不是与这话相同吗？"

耶和华的话临到撒迦利亚说："万军之耶和华如此说：你们当按真实审判，各人以慈爱怜悯弟兄，不可欺压寡妇、孤儿、寄居的，和贫穷人，也不可各人心里谋害弟兄。"他们却不肯听从，扭转肩头，塞耳不听，使心硬如金刚石，不听律法和万军之耶和华用灵藉从前的先知所说的话。故此万军之耶和华大发烈怒。万军之耶和华说："我曾呼唤，他们不听；他们呼求，我也不听。我必以旋风吹散他们到素不认识的各国中。"这样，这地在他们后边荒凉，无人来往，把那美好的地变为荒场了。

**新约读经**：罗马书十六章十七至二十七节

**最后的嘱咐与问安**

弟兄们，那些离间你们、叫你们跌倒、背乎所学之道的人，我劝你们要留意躲避他们。因为这样的人不服侍我们的主基督，只服侍自己的肚腹，用花言巧语诱惑那些无知人的心。你们顺服的名声传于众人，所以我为你们欢喜；但我愿意你们在善上聪明，在恶上愚拙。赐平安的神快要将撒但践踏在你们脚下。愿我主耶稣基督的恩常和你们同在。

我的同工提摩太，还有路求、耶孙、所西巴德，我的亲属，都问你们安。

我是特求，替保罗写这信，在主里面向你们问安。

那款待我和全教会的该犹，也向你们问安。城里的财主以拉都和弟兄括土，都向你们问安。

**荣耀颂**

惟有神，能照我所传的福音和所讲的耶稣基督，并照永古隐藏、现今显明的奥秘，坚固你们，这奥秘如今藉先知书显明出来，又藉永生神的命令，传于万国的民，使他们信服真道；愿荣耀因耶稣基督归与独一全智的神，直到永远。阿们。

**历史与灵修读写**

或许有人会问："既然圣经的正典已经完备，就其本身而言已经充分，甚至绰绰有余，为何还需要加上教会诠释的权威呢？"原因在于：由于圣经的深奥，并非所有人都以同一方式接受它；有人以这种方式理解其话语，另一人则以另一种方式理解，以至于圣经仿佛可以有多少解释者，就有多少种解释……

此外，在大公教会本身之内，必须尽一切可能谨守那已被各处、历来、众人所信奉的信仰。因为"大公的"（catholic）这一称呼，按其名称本身和事物的理性，确实且最严格地意味着涵盖一切普世的事物。我们若遵循普遍性、古老性、一致性，便能守住这一规则。若我们承认整个世界性教会所信奉的那一信仰为真，则我们遵守了普遍性；若我们在任何方面都不偏离那些明显为我们圣洁的先祖和教父们所公认持守的诠释，则我们遵守了古老性；若我们在古老性本身之内，坚守一切或至少几乎所有的祭司和博士们一致同意的界定与裁决，则我们同样遵守了一致性。

那么，若教会的一小部分已将自己与普世信仰的团契割断，一位大公基督徒该怎么办？毫无疑问，他当如此行：宁取整体的健全，不取一个腐败的患病肢体的不健全。若某种新的毒疾不只是试图感染教会一个微不足道的部分，而是整个教会呢？那时他就当留心坚守古代的信仰，因为在今日，古代的信仰绝不可能被任何新奇事物的欺骗所引诱。

——卢兰的文桑

**圣诗**

医治祢儿女争战的疯狂；
俯伏我们的骄傲于祢的掌管之下；
羞愧我们任性、自私的喜乐，
物质丰盛，灵魂贫乏。
赐我们智慧，赐我们勇气，
免使我们错失祢国的目标，
免使我们错失祢国的目标。

——《恩典之神、荣耀之神》（路德宗崇拜书 850:3）
