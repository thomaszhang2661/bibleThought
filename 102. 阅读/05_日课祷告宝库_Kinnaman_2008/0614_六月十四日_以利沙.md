# 六月十四日　以利沙

**诗篇吟唱**

全地当向上帝欢呼！
要歌颂他名的荣耀，
用荣耀赞美他！

要对上帝说："你的作为何等可畏！
你的能力极大，
你的仇敌要向你投降屈服。"

全地要敬拜你，歌颂你，
要歌颂你的名。

你们来看上帝所行的作为，
他向世人所行的是何等可畏的事！

他使海变为旱地，
众人徒步走过大河。
我们在那里因他欢喜快乐。
他用大能统治直到永远，
他的眼目鉴察列国；
叛逆的人不可自高。

众民哪，要称颂我们的上帝，
使人听颂赞他的声音。

——诗篇六十六篇一—八节

**旧约读经**：箴言十四章一—二十七节

**所罗门的箴言**

智慧的妇人建立家室，
愚妄的妇人亲手拆毁。

行事正直的人敬畏耶和华，
行事乖僻的人藐视他。

愚妄人口中尽是骄傲的杖，
智慧人的嘴必保护自己。

没有牛，槽头干净；
牛力生财，出产增多。

诚实见证人不说谎话，
假见证人吐出谎言。

亵慢人寻智慧，却寻不着；
聪明人易得知识。

当离愚昧人而去，
因为在他嘴边你寻不着知识。

通达人的智慧在乎明白己路，
愚昧人的愚妄乃是诡诈。

愚妄人因赎愆祭而嘲弄，
正直人彼此蒙悦纳。

心中的苦楚自己知道，
心中的喜乐外人无从干涉。

奸恶人的房屋必倾倒，
正直人的帐棚必兴旺。

有一条路，人以为正，
但其终点却是死亡之路。

人在嬉笑中心中也有愁苦，
欢乐之后仍归忧愁。

心中背道的必得满足于自己所行，
善人也必因自己所行得满足。

愚蒙人信一切话；
通达人步步谨慎。

智慧人惧怕罪恶，就远离它；
愚妄人却狂傲自恃，并不谨慎。

轻易动怒的人行事愚妄；
设谋作恶的人被人恨恶。

愚蒙人承受愚妄，
通达人得知识为冠冕。

坏人俯伏在好人面前，
恶人俯伏在义人门口。

穷人连邻舍也厌弃他，
富足人的朋友却极多。

藐视邻舍的，这人有罪；
怜悯穷人的，这人有福。

谋恶的岂非走入迷途么？
谋善的有慈爱与诚实。

诸般勤劳都有益处，
嘴上空谈只致贫乏。

智慧人的财富是他的冠冕，
愚昧人的愚妄还是愚妄。

作真实见证的能救人命，
吐出谎言的行事诡诈。

敬畏耶和华使人有坚固的倚靠，
他的儿女也有了避难所。

敬畏耶和华是生命的泉源，
可使人远离死亡的网罗。

**新约读经**：约翰福音十五章一—十一节

**我是真葡萄树**

"我是真葡萄树，我父是栽培的人。凡属我却不结果子的枝子，他就剪去；凡结果子的，他就修理干净，使枝子结果子更多。现在你们因我讲给你们的道，已经干净了。你们要常在我里面，我也常在你们里面。枝子若不常在葡萄树上，自己就不能结果子；你们若不常在我里面，也是这样。我是葡萄树，你们是枝子。常在我里面的，我也常在他里面，这人就多结果子；因为离了我，你们就不能做什么。人若不常在我里面，就像枝子丢在外面枯干，人拾起来，扔在火里烧了。你们若常在我里面，我的话也常在你们里面，凡你们所愿意的，祈求，就给你们成就。你们多结果子，我父就因此得荣耀，你们也就是我的门徒了。我爱你们，正如父爱我一样；你们要常在我的爱里。你们若遵守我的命令，就常在我的爱里，正如我遵守了我父的命令，常在他的爱里。这些事我已经对你们说了，是要叫我的喜乐存在你们心里，并叫你们的喜乐可以满足。"

**历史与灵修读写**

以利沙是一位治理教会、推动教义研习、并多次向以色列诸王提供政治谋略的先知。他在上帝子民中服事逾八十年……他来自亚伯米何拉，从耕田之地被呼召担任先知之职（列王纪上十九章）。他是先知以利亚忠心勤勉的门徒与继承人。当以利亚即将升天之时，以利沙求得他双倍的灵，即在上帝子民中以宏大喜乐之心治理教会与学校所必需的恩赐（列王纪下二章）……他以源源不断的油瓶援助一位受债主逼迫的孤苦寡妇（列王纪下四章），并使书念妇人之子复活（列王纪下四章）。他在耶利哥将盐撒入水中，使苦水变甜（列王纪下二章）；借此象征，他同时表明：教义与美德的根本是苦涩的，然而当上帝赐福，智慧之盐调味之后，其果实便极为甘甜。在饥荒之时，他在吉甲用田间的野草和野葫芦这些微薄食材供养百位学者，并将话语与神圣应许的面粉倒入其中，消去其苦涩（列王纪下四章）。他洁净了叙利亚人乃缦的大麻风，并以同样的灾祸惩罚了他那贪婪说谎的仆人基哈西（列王纪下五章）。他使掉入约旦河中的铁斧头浮在水面（列王纪下六章）……最终，他在病中预言以色列王约阿施的胜利，随后离世（列王纪下十三章）。当一具被强盗杀害的旅人尸体被投入以利沙的坟墓时，那人竟死而复生（便西拉书四十八章）。

——大卫·基特勒乌斯（David Chytraeus）

耶稣仍为我流血；
他死，为使我居高天；
他活，使我永不死。
枝条与葡萄树相连，

**今日祷文**

主上帝，天父，你借着先知以利沙，延续了教导子民真道的先知传统，并借着神迹彰显你在受造界中的同在，以医治其破碎。求你赐恩于你的教会，使她在你的儿子、我们的主耶稣基督身上，看见那末世终极之先知——他的教导与神迹，借着福音与圣礼这医治的药方，在你的教会中持续延伸。奉耶稣基督、我们主的名祈求。（1038）

## 以利沙

以利沙是以萨迦支派沙法之子，约于主前八四九至七八六年间担任北国以色列的上帝先知。当他亲眼目睹导师以利亚被接升天，便承接了先知之职，拾起前任的外衣。与以利亚相似，以利沙在政治事务中亦扮演积极角色。他也行了许多神迹，例如医治叙利亚军队长官乃缦的大麻风（列王纪下五章），以及使书念妇人之子复活（列王纪下四章八—三十七节）。作为巴力崇拜的坚决反对者，以利沙以行动活出了他名字的含义——"我的上帝是救恩"。
