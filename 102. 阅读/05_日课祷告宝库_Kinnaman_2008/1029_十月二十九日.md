# 十月二十九日

**诗篇吟唱**

神啊，求你怜悯我，因为人践踏我；终日有人攻打压制我。
我的仇敌终日践踏我；因为攻击我的人甚多，他们狂傲地起来攻打我。
我惧怕的时候，要倚靠你。
我倚靠神，我要颂赞他的话；我倚靠神，必不惧怕；血气之辈能把我怎么样呢？
他们终日曲解我的话，他们一切的心思都是要害我。
他们聚集，埋伏等候；他们窥探我的脚踪，等候要害我的命。
他们岂能因罪孽逃脱吗？神啊，求你在怒中使众民倾倒！
你曾记载我几次流离；求你把我的眼泪装在你的皮袋里，这不都记在你册子上吗？
我呼求的日子，我的仇敌都要转身退后；我知道神帮助我。
我倚靠神，我要颂赞他的话；我倚靠耶和华，我要颂赞他的话——
我倚靠神，必不惧怕；人能把我怎么样呢？
神啊，我把许愿还给你；我要将感谢祭献给你，
因为你救了我的命不致死亡，你岂不是救我的脚不致跌倒，使我在生命光中行在神面前吗？

——诗篇五十六篇

**旧约读经**：申命记三十一章1—29节

**约书亚接续摩西**

摩西将这话讲完了，对以色列众人说：我今日年已一百二十岁，不能照常出入了，并且耶和华已经说过：你必不得过这约旦河。耶和华你的神，他自己要在你面前过去，要将这些国民从你面前灭绝，你就得了他们的地；约书亚必在你面前过去，是耶和华所说的。耶和华要待他们，如他待他所灭绝的亚摩利王西宏和噩，并他们的地一样。耶和华将他们交在你们手中，你们要照我所吩咐你们的一切命令待他们。你们当刚强壮胆，不要害怕，也不要畏惧他们，因为耶和华你的神是与你同去的，他必不撇下你，也不丢弃你。

摩西将约书亚召来，在以色列众人眼前对他说：你当刚强壮胆；因为你要和这百姓进入耶和华向他们列祖起誓应许赐给他们的地，你要使他们承受那地为业。耶和华在你前面行；他必与你同在，他不会离开你，也不会丢弃你；不要惧怕，也不要发愁。

**宣读律法**

摩西将这律法写出来，交给抬耶和华约柜的利未人祭司和以色列众长老，吩咐他们说：每逢七年的末一年，就是豁免年，在住棚节，以色列众人来到耶和华你神所选择的地方朝见他的时候，你要在以色列众人面前，朗读这律法。要将百姓——男女孩子，并城里寄居的外人——聚集，使他们都可以听，都可以学习，好敬畏耶和华你们的神，谨守遵行这律法的一切话，也叫他们不知道这律法的儿女，都可以听，学习敬畏耶和华你们的神，你们过约旦河，进去得地的日子，这话都必常在他们心里。

**约书亚蒙召领导以色列**

耶和华对摩西说：你死的日子临近了，你要召约书亚，你们同到会幕，让我嘱咐他。摩西和约书亚去了，站在会幕里。耶和华在会幕中云柱内显现，云柱停在会幕门口。

耶和华对摩西说：你快要与你列祖同睡了；这百姓起来，在他们所要进入的地上，随从那地之外邦人的神行邪淫，离弃我，违背我与他们所立的约。那时我的怒气必向他们发作，我要离弃他们，将脸向他们掩藏，他们必遭吞灭；许多祸患、急难，必临到他们，那时他们必说：这些祸患临到我们，岂不是因为神不在我们中间吗？那日，他们所行的一切恶，我必将脸向他们掩藏。

现在你们要写这首歌，教导以色列人，传授给他们，使这歌为我作见证。因为我将他们领入我向他们列祖所起誓应许之流奶与蜜之地，他们吃得饱足，身体丰盛，就转向别神侍奉，藐视我，违背我的约；他们遭遇许多祸患急难的时候，这歌必在他们后裔口中作见证，因为这歌必不从他们后裔的口中断绝。我将他们领入我所应许的地之先，我先知道他们的意向。所以那日，摩西就写了这首歌，教导以色列人。

耶和华嘱咐嫩的儿子约书亚，说：你当刚强壮胆，因为你必领以色列人进入我所起誓应许他们的地；我必与你同在。

摩西将这律法的话，从头至尾写在书上，就吩咐抬耶和华约柜的利未人说：将这律法书放在耶和华你们神的约柜旁，可以在那里见证以色列人的不是；因为我知道你们是悖逆的，是硬着颈项的。我还活着，还在你们中间的今日，你们尚且悖逆耶和华，我死了之后，何况呢！要在你们各支派的长老，并你们的官长聚集到我这里，使我将这些话说给他们听，并且呼天唤地见证他们的不是；因为我知道我死后，你们必定全然败坏，偏离我所吩咐你们的道，日后必有灾祸临到你们，因为你们行了耶和华眼中看为恶的事，以手所做的惹他发怒。

**新约读经**：马太福音十九章16—30节

**财主的少年人**

有一个人来见耶稣，说：夫子，我该做什么善事才能得永生呢？耶稣对他说：你为什么以善事问我呢？只有一位是善的（有古卷：你为什么称我是良善的？除了神以外，没有一个良善的）；你若要进入永生，就当遵守诫命。他说：是哪些呢？耶稣说：就是不可杀人，不可奸淫，不可偷盗，不可作假见证，当孝敬父母，又当爱人如己。那少年人说：这一切我都遵守了，还缺少什么呢？耶稣说：你若愿意作完全人，可去变卖你所有的，分给穷人，就必有财宝在天上；你还要来跟从我。那少年人听了这话，就忧忧愁愁的走了，因为他的产业很多。

耶稣对门徒说：我实在告诉你们，财主进天国是难的。我又告诉你们，骆驼穿过针的眼，比财主进神的国还容易呢。门徒听见这话，就希奇得很，说：这样谁能得救呢？耶稣看着他们，说：在人这是不能的，在神凡事都能。那时彼得对他说：看哪，我们已经撇下所有的跟从了你，将来我们要得什么呢？耶稣说：我实在告诉你们，你们这跟从我的人，到复兴的时候，人子坐在他荣耀的宝座上，你们也要坐在十二个宝座上，审判以色列十二个支派。凡为我的名撇下房屋，或是弟兄、姊妹、父亲、母亲、儿女、田地的，必要得着百倍，并且承受永生。然而，有许多在前的将要在后，在后的将要在前。

**历史与灵修读写**

《反驳书》也说修士更丰富地赚取永生，并引用圣经"凡离开……的人"（马太福音十九29）。于是这里也为人为的宗教礼仪主张完全性。但这段圣经毫无支持修道院生活之处。基督并不是说离开父母、妻子和兄弟姊妹是一件必须做的工，因为它赚取罪的赦免和永生。事实上，这样的离开是被咒诅的。任何离开父母或妻子，以为凭此工赚取罪的赦免或永生的人，都是在羞辱基督。

有两种离开。其一是没有呼召、没有神命令的离开，基督不赞许这种离开（马太福音十五9）。我们所选择的工是无用的侍奉。当基督谈到离开妻子和孩子时，他清楚地表明他不赞许这种离开。我们知道，神的诫命禁止离开妻子和孩子。其二是神命令的离开，即当权势或暴政迫使我们要么离开、要么否认福音时。这时我们被命令忍受伤害，宁可让财富、妻子、孩子，甚至生命被夺去，也不妥协。基督赞许这种离开，所以他加上了"为福音的缘故"（马可福音十29）。他这样说，是要说明他所说的，不是那些伤害妻子和孩子的人，而是因福音的认信而忍受伤害的人。为着福音的缘故，我们甚至当离弃自己的身体。若有人认为自杀和无神命令地离弃身体是侍奉神，这是荒谬的。同样，无神命令地离弃财产、朋友、妻子和孩子，认为是侍奉神，也是荒谬的。

——《奥格斯堡信条辩护》第二十七（十三）章40—42节

**路德宗信条建议阅读**

《奥格斯堡信条辩护》第二十四（十二）章9—15节

9上帝指定基督为中保；祂愿意藉着祂施恩与我们，而不藉着我们的义施恩与我们。这些人相信，上帝因着传统而不因着基督〔与世人和好而施恩〕。如是他们夺去基督为中保之尊荣。

10由此观点来讲，我们的传统和摩西的礼节之间，没有区别。保罗咒诅摩西的礼仪，也咒诅遗传，因为他们想到行遗传的事，为在上帝面前配得义的行为，而且因此这些遗传掩蔽基督的工作和信之义。藉着将律法和传统排除在外，保罗主张罪的赦免被应许给人，不是因我们的行为，乃是白白因基督之故，只要我们藉信接受它；因为惟独信仰能接受应许。11既是我们接受罪的赦免是藉着信，并藉着信因基督之故，我们得有一位恩惠的上帝，若〔仍然〕主张我们藉着这些遗传的遵守配得罪的赦免， 就算是不虔诚的异端了。12对此，可能有人会说：虽然我们不配得罪的赦免，而那些曾经称义的人，是藉着遵守这些遗传配得恩惠。保罗对此回答，倘若我们相信称了义以后我们非因基督的缘故被算为义，而必须先藉别的遵守配得恩典，那么，基督是“罪的执事”（加二17）。“甚至没有人能废弃或加增人的文约”（加三15）。换句话说，上帝的约应许我们，祂必因基督之故为恩惠的上帝，我们不敢加添条件：我们须先藉这些遵守配得我们的悦纳和称义。

13有长时讨论的必要吗？圣教父未曾为使人配得罪的赦免和义的目的设立任何传统。他们设立传统是为教会内的良好次序与宁静缘故。14若有人想设立某一定的善行为配得罪的赦免和义，若没有上帝之道的支持，他如何知道上帝喜悦这些行为？若没有上帝的道和命令，他如何将上帝的旨意指示人呢？上帝不是曾在先知全书内禁止在祂的命令之外另设礼节吗？在以西结二十18、19记着：“不要遵行你们父亲的律例，不要谨守他们的恶规，也不要因他们的偶像玷污自己。我是耶和华你们的上帝， 你们要顺从我的律例，谨守遵行我的典章。” 15但是倘若人们被准许设立新的礼仪，并且若是他们藉着这些礼仪配得恩典，那么，我们不能不赞成众外邦宗教的仪式，也必赞成耶罗波安王等设立超乎上帝律法的礼仪（王上十二28下）。区别何在呢？若准许我们设立礼仪以配得恩或义，那么，外邦人和以色列人为何未得到此种权利呢？
