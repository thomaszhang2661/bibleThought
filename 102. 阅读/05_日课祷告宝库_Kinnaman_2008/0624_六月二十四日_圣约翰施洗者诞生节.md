# 六月二十四日　圣约翰·施洗者诞生节

**诗篇吟唱**

耶和华啊，求你向我们显示你的慈爱，将你的救恩赐给我们。
我要听神耶和华所说的话，因为他必向他的百姓、他的圣民说和平的话；他们不可再转去愚昧。
他的救恩实在与敬畏他的人相近，使荣耀住在我们的地上。
慈爱和诚实彼此相遇，公义和平安彼此相亲。
诚实从地而生，公义从天而降。
耶和华必赐下美善，我们的地也必出产土产。
公义必在他面前行，叫他的脚踪成为可走的路。

——诗篇八十五篇七—十三节

**旧约读经**：箴言三十章一—九节，十八—三十三节

**雅基的儿子亚古珥的言语**

这是雅基的儿子亚古珥的言语，就是那神谕。

这人对以铁和乌甲说：我比众人更蠢笨，也没有人的聪明。
我没有学过智慧，也不认识至圣者。
谁升天又降下来？谁聚风在掌中？谁包水在衣服里？谁立定地的四极？他名叫什么？他儿子名叫什么？你知道吗？
神的言语句句都是炼净的；投靠他的，他便做他们的盾牌。
他的言语，你不可加添，恐怕他责备你，你就显为说谎言的。
我求你两件事，在我未死之先，不要不赐给我：
求你使虚假和谎言远离我；使我不贫穷也不富足，赐给我需用的饮食，
恐怕我饱足而不认你，说"耶和华是谁呢"；又恐怕我贫穷就偷窃，以致亵渎我神的名。……

有三样事我觉得奇妙，连四样事我也不明白：
就是鹰在空中飞的道，蛇在磐石上爬的道，船在海中行的道，男与女交合的道。
淫妇的道也是这样：她吃了，把嘴一擦，说"我没有做什么坏事。"
使地震动的有三样，连地所不能担当的四样：
就是仆人作王，愚顽人吃饱，
以及不蒙喜爱的女子出嫁，婢女替代女主人。
地上有四样小物，却甚有智慧：
蚂蚁是没有能力的生物，却在夏天预备粮食；
沙番是软弱的动物，却在磐石间造房；
蝗虫没有君王，却分队而出；
蜥蜴你用手就可以拿住，它却在王宫里。
有三样行步甚是好看，连第四样走路也是好看：
就是狮子在走兽中最有能力，不躲避任何；
犬，公山羊，并没有人能抵挡的君王。
你若行了愚昧，自高自大，或是心怀恶意，就当用手捂口。
因为用力搅奶，就出奶油；用力挪鼻，就出血；照样，激动怒气，就起争端。

**新约读经**：约翰福音二十章十九—三十一节

**耶稣向门徒显现**

那日（就是七日的第一日）晚上，门徒所在的地方，因怕犹太人，门都关了。耶稣来，站在当中，对他们说："愿你们平安！"说了这话，就把手和肋旁指给他们看。门徒看见主，就喜乐了。耶稣又对他们说："愿你们平安！父怎样差遣了我，我也照样差遣你们。"说了这话，就向他们吹一口气，说："你们受圣灵！你们赦免谁的罪，谁的罪就赦免了；你们留下谁的罪，谁的罪就留下了。"

**耶稣与多马**

那十二个门徒中，有称为低土马的多马，耶稣来的时候，他没有和他们同在。那些门徒就对他说："我们已经看见主了。"多马却说："我非看见他手上的钉痕，用指头探入那钉痕，又用手探入他的肋旁，我总不信。"

过了八日，门徒又在屋里，多马也和他们同在，门都关了。耶稣来，站在当中说："愿你们平安！"就对多马说："把你的指头伸过来，摸我的手；把你的手伸过来，探入我的肋旁。不要疑惑，总要信！"多马说："我的主！我的神！"耶稣对他说："你因看见了我才信；那没有看见就信的有福了！"

**本书的目的**

耶稣在门徒面前另外行了许多神迹，没有记在这书上。但记这些事要叫你们信耶稣是基督，是神的儿子，并且叫你们信了他，就可以因他的名得生命。

**历史与灵修读写**

现在，有人或许会说：朋友，你说了许多关于信心的话，声称我们的救恩完全依赖于信心；那么我问你，人如何得到信心呢？我来告诉你。我们的主基督说："愿你们平安！你们看我的手……"（约翰福音二十章二十六至二十七节）。换言之，他是在说：看哪，人啊，我是唯一除去你罪孽、救赎你的那一位，如今你要平安。正如你从亚当那里承受了罪——并非因你亲自犯了罪，因为那苹果不是我吃的，也不是你吃的，然而我们就这样落入了罪中——同样，我们并未亲历基督的受苦，却因神的作为，而非我们自己的功劳，从死亡与罪中得了释放。因此神说：看哪，人啊，我是你的救赎（参以赛亚书四十三章三节）；正如保罗对哥林多人所说的：基督是我们的称义、救赎……（哥林多前书一章三十节）。基督是我们的称义与救赎，正如保罗在这段话中所说的。而我们罗马方面的那些师傅则说：不错，"Redemptor"，救赎主；这是真的，但这还不够。

因此我再次申明：外来的功业，这才使我们成为义人！我们的主基督说：我是你们的称义。我已除去你们身上所担的罪。因此，只管信我；信我就是那位成就此事的，你们便得称义。因为经上写着"Justicia est fides"（义即信心），公义与信心同一，并藉信心而来。因此，若我们要得着信心，就当信从福音、信从保罗……等等，而非教皇的大诏书或教令集，倒要防备它们，如同防备火一样。因为凡从教皇而来的，无不高喊：给，给；若你拒绝，你就是魔鬼的人。若他们只是剥削百姓，那还是小事。但可悲的是，世上最大的祸害，莫过于引导人相信外在的功德能够拯救人或使人成为善人。

——马丁·路德

**圣约翰·施洗者诞生节**

施洗者圣约翰，是撒迦利亚与伊利莎白之子，出身于祭司家族。天使曾向其父奇迹般地宣告他的诞生（路加福音一章五至二十三节），在他出生之际，年迈的父亲颂唱了一篇赞美诗（路加福音一章六十七至七十九节）。此诗被称为《撒迦利亚颂》（Benedictus），历来作为教会晨祷礼仪的传统福音颂歌。约翰一生的事迹与教导，在四福音书中均有记载。约翰在犹大旷野、约旦河附近开始传道，呼召人悔改，施行洗礼，并向众人宣告："看哪，神的羔羊，除去世人罪孽的！"（约翰福音一章二十九节）。约翰斥责希律王室成员的淫乱生活，因此加利利分封王希律·安提帕将他逮捕，囚禁于死海附近的马卡鲁斯要塞。希律在那里将他斩首（马可福音六章十七至二十九节）。约翰以其传道指向"神的羔羊"、为弥赛亚降临"预备道路"而被后人铭记尊崇。
