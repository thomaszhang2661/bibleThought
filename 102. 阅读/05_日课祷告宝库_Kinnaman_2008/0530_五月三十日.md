# 五月三十日

**诗篇吟唱**

¹　神啊，求你伸我的冤，辩护我的案，救我脱离不虔诚的国民；求你救我脱离诡诈不义之人。
²因为你是我避难所的　神；你为何丢弃我呢？我因仇敌的欺压，为何哀痛而行呢？
³求你发出你的光和你的真实，好引导我，带我到你的圣山，到你的居所。
⁴我就走到　神的祭坛，到我最喜乐的　神那里；　神啊，我的　神，我要弹琴称赞你。
⁵我的心哪，你为何忧闷？为何在我里面烦躁？应当仰望　神，因他笑脸帮助我，我还要称赞他。

——诗篇四十三篇

**旧约读经**：传道书七章11—29节

**智慧与愚昧的对比（续）**

¹¹智慧和产业并好，而且见天日的人得智慧更为有益。
¹²因为智慧护卫人，正如银钱护卫人；智识的益处就在乎智慧能保存有智慧人的生命。
¹³你要察看　神的作为，因他使弯曲的，不能变直。
¹⁴遇亨通的日子，你当喜乐；遭患难的日子，你当思想；因为　神使这两样并列，为的是叫人查不出身后有什么事。
¹⁵我在虚度的年日见过这两样的事：有义人行义，反致灭亡；有恶人行恶，倒享长寿。¹⁶不要行义过分，也不要过于自逞智慧，何必自取败坏呢？¹⁷不要行恶过分，也不要为人愚昧，何必不到你的期限就死呢？¹⁸你持守这个为美，那个也不要松手；因为敬畏　神的人，必从这两样都出来。
¹⁹智慧使有智慧的人，比城中十个官长更有能力。
²⁰时常行善而不犯罪的义人，世上实在没有。
²¹人所说的一切话，你不要放在心上，恐怕听见你的仆人咒诅你；²²因为你心里知道，自己也曾多次咒诅别人。
²³我用智慧试验这一切，我说我要得智慧，智慧却远离我。²⁴万事中，远处的和深藏的，谁能找到呢？
²⁵我转心，察明智慧和万事的道理，又知道邪恶为愚昧，愚昧为狂妄。²⁶我发现，比死更苦的是那妇人，因她是猎人的网罗，她的心是罗网，她的手是锁链。凡蒙　神喜悦的人必能躲避她，有罪的人却被她缠住。²⁷传道者说：我将诸事逐一合算，要寻求其中的道理，我心仍在寻找，还未寻见。²⁸我在千人中寻得一个正直人，却在这一切的人中未得到一个正直女人。²⁹你看，我所找到的结论是：　神造人原是正直，但他们寻出许多巧计。

**新约读经**：约翰福音八章39—59节

**你们是出于你们的父魔鬼**

³⁹他们回答说："我们的父就是亚伯拉罕。"耶稣说："你们若是亚伯拉罕的儿子，就必做亚伯拉罕所做的事。⁴⁰我将在　神那里所听见的真理告诉了你们，你们却想要杀我，亚伯拉罕并没有这样行。⁴¹你们是做你们父所做的事。"他们说："我们不是从淫乱生的；我们只有一位父，就是　神。"⁴²耶稣说："倘若　神是你们的父，你们就必爱我；因为我本是出于　神，也是从　神而来，并不是由着自己来，乃是他差我来。⁴³你们为什么不明白我的话呢？无非是因你们不能听我的道。⁴⁴你们是出于你们的父魔鬼，你们父的私欲你们偏要行。他从起初是杀人的，不守真理，因他心里没有真理；他说谎是出于自己，因他本来是说谎的，也是说谎之人的父。⁴⁵我将真理告诉你们，你们却不信我。⁴⁶你们中间谁能指证我有罪呢？我既然说真理，你们为什么不信我呢？⁴⁷出于　神的，必听　神的话，你们不听，因为你们不是出于　神。"

**在亚伯拉罕以先，就有了我**

⁴⁸犹太人回答说："我们说你是撒玛利亚人，而且是鬼附着的，这话岂不对吗？"⁴⁹耶稣说："我里头没有鬼，我是尊敬我父的；你们却不尊敬我。⁵⁰我不求自己的荣耀，有一位是叫我有荣耀的，也是审判人的。⁵¹我实实在在地告诉你们，人若遵守我的道，就永远不见死。"⁵²犹太人说："现在我们知道你是鬼附着的了！亚伯拉罕死了，众先知也死了，你还说'人若遵守我的道，就永远不尝死味'。⁵³难道你比我们的祖宗亚伯拉罕还大吗？他死了，众先知也死了；你将自己当作什么人呢？"⁵⁴耶稣回答说："我若荣耀自己，我的荣耀算不得什么；荣耀我的乃是我的父，就是你们所说'他是我们的　神'的那一位。⁵⁵你们未曾认识他，我却认识他；我若说我不认识他，我就是说谎的，像你们一样；但我认识他，也遵守他的道。⁵⁶你们的祖宗亚伯拉罕欢欢喜喜地仰望见我的日子，既看见了，就快乐了。"⁵⁷犹太人说："你还没有五十岁，岂见过亚伯拉罕呢？"⁵⁸耶稣说："我实实在在地告诉你们，还没有亚伯拉罕，就有了我。"⁵⁹他们就拿石头要打他，耶稣却躲避，从殿里出去了。

**历史与灵修读写**

肉体的情欲是显而易见的，就是：奸淫、污秽、邪荡、拜偶像、邪术、仇恨、争竞、忌恨、恼怒、结党、纷争、异端、嫉妒、醉酒、荒宴等类，以及与此相似的事。（加拉太书五章十九至二十一节）

因此，若你不能分辨这些，至少要相信圣经。圣经不会欺骗你，它比你自己更了解你的肉体……信靠圣经所宣告于你的定罪。简而言之，你越少感受到自己的罪和软弱，就越有理由去领圣餐，寻求帮助与医治。

其次，环顾四周。看看你是否也在这个世界上；若你不知道，可以问问你的邻居。若你在这世界上，就不要以为会缺少罪和苦难。只要开始行事如同你要敬虔，并持守福音，看看是否没有人会成为你的仇敌，进而加害你、冤屈你、暴力对待你，同时也给你犯罪堕落的缘由……

除此之外，你身边还有魔鬼。你不能将它完全踩在脚下（路加福音十章十九节），因为我们的主基督自己也不能完全避开它。如今，魔鬼是什么？不过就是圣经称他的：一个说谎者，一个杀人者（约翰福音八章四十四节）。它是说谎者，要使心离开　神的话语，使之蒙蔽，以致你感受不到自己的困境，无法来就基督。它是杀人者，不忍见你活过一小时。若你能看见每一刻有多少刀剑、标枪、箭矢对着你（以弗所书六章十六节），你就会尽可能频繁地去领圣餐。但我们之所以如此漫不经心、掉以轻心地四处游走，无非是因为我们既不思念也不相信：我们活在肉身之中，活在这邪恶的世界上，活在魔鬼的国度里。

——大问答 第五部分 第75—82条

**圣诗**

颂赞亚伯拉罕的　神，他坐在宝座之上；
他是万古常存之主，是爱的　神。
耶和华，大　神！被地与天同声承认；
我俯伏赞美这圣名，直到永远蒙福。

——《路德宗崇拜书》798:1

**今日祷文**

全能永恒的　神，你赐给我们恩典，藉着真实信仰的承认，认识永恒三位一体的荣耀，并在神圣威严的合一中敬拜你。愿你使我们在这信仰中坚立，并抵御一切的患难；因为你——父、子、圣灵——活着掌权，同为独一真神，从今直到永远。阿们。
