# 第三章：把好大门——没有骨气，就别谈合一

### 核心痛点：请神容易，送神难
建立一支打胜仗的团队，最关键的一步是**把好大门的入场券**。

很多教会团队之所以沦为平庸，或者整天陷入内耗，就是因为在选拔长执同工或者招聘全职员工时，放了不合适的人进来。只要团队里塞进一个负能量爆棚、掌控欲极强或者完全不在状态的人，整个团队的喜乐和效率就会瞬间被榨干。

更糟糕的是，**一旦一个有毒（Toxic）的人坐上了核心团队的位置，你想把他请下去，代价大到你无法想象**。因为任何一个麻烦制造者，在教会里都一定有一群支持他的小圈子。强行解聘或者开除一个人，往往会引发一连串的连带伤害和人员流失。所以，最省时省力的方法，是在大门口就把他们拦住。

### 1. 牧师和领袖必须参与“长执提名”的防线
在许多传统教会的政治体制（Polity）中，长执的选拔被交给了所谓的“海选”或者随机拼凑的提名委员会。他们觉得这样才民主、才公平。

Larry严厉地驳斥了这种做法。选拔核心领袖太重要了，绝对不能看运气。如果你们的体制把牧师和现有的核心团队排除在提名过程之外，甚至允许在全体大会上盲目地从现场举手提名，这无异于把教会的未来当儿戏。

这就好比给一个球队的主教练，强行硬塞了几个完全不认同他战术风格的助教和球员。这能打胜仗才叫活见鬼。牧师和核心领袖必须有勇气在这个过程中站稳脚跟，守住大门。

### 2. 有骨气说“不”：别在 front-end（前端）装好人，在 back-end（后端）当怨妇
Larry分享了他自己的一个惨痛教训。有一次提名，大家讨论起一位老资格的同工。这个人灵性很好，品格也没得说。但每个人都知道，他是个典型的“杠精”和自封的“牧师监察员”，他的使命就是确保牧师不会获得太多权柄。只要他进团队，以后的会议肯定天天在一些无谓的细节上兜圈子。

当时Larry为了维持面子，在开会时保持了沉默。结果这个人进来后，让团队痛苦了整整三年。

Larry的一位好朋友后来狠狠地骂了他一顿，这句话所有做领袖的人都该纹在皮肉上：**“如果你在前端选人时没有骨气说实话，你就没有资格在后端带团队时当怨妇。给我闭嘴，下次有点骨气！”**

从此以后，当提名遇到不合适、性格 bristly（带刺）的人时，Larry学会了在保护对方名誉的前提下，在核心圈子里清清楚楚地表达自己的Relational concerns（人际关系上的担忧）和哲学理念上的冲突，坚决一票否决。

### 3. 选“领袖”，别选“利益代言人”
[大白话解释]：很多教会的同工会实行的是“人大代表”制度：老张代表老年人，小李代表年轻人，大刘代表主日学。

这种代表（Representative）制度会彻底毁掉团队的合一。因为一旦开会，每个人都会变成自己背后那个特殊利益群体的“游说游说客”。老张会拼命把预算往老年事工上拉，大刘会死守主日学的地盘，大家在开会时都在玩“政治博弈和地盘争夺战”，根本无法站在全教会和上帝的心意上去做通盘考虑。

黏性团队选拔的人，必须是**“领袖导向（Leadership-oriented）”**的。他们不代表任何人，他们唯一的使命是寻求：**“上帝在这件事上的心意是什么？”**

为了配合这个转轨，北海岸教会推行了一条铁律：**团队里绝对不允许出现“他们（They）”这个词。**
开会时，如果有人说：“我听到一些会众反映，‘他们’对这个决定有很大意见。”
主持会议的人会立刻打断他：“‘他们’是谁？有名有姓吗？如果只是躲在匿名背后的键盘侠，或者你想象出来的代言群体，在我们这里一概不予讨论。请讲有名有姓、愿意露面沟通的真实回馈。”
这条“禁止‘他们’出现”的规则，瞬间让无数隐秘的政治小动作彻底现了原形。

### 4. 招聘全职同工的硬核铁律：

* **“还凑合 / 差不多”的人选，就是绝对的灾难**：
  很多教会因为一个职位空缺了很久，心里慌得要死，于是在面试时看到一个“虽然不是很理想，但目前来看是最好的人选”时，就咬牙雇了下来。Larry用带血的经验警告你：**宁可让一个职位空缺一年，也绝对不要雇一个“还凑合”的人进来。** 填补空缺的焦虑是暂时的，但请一个不对劲的人进来，他所带来的团队内耗和随后的开除成本，会折腾掉你半条命。

* **简历这东西，永远比真人好看**：
  做领袖的往往容易有一种“外来的和尚会念经”的幻觉。看外来应聘者的简历，觉得对方履历惊人、光芒万丈；看身边的同工，天天在一起，全是缺点。但Larry指出，北海岸教会的数据表明，**从内部长线培养并提拔起来的员工，成功率接近100%；而通过猎头和简历从外面招进来的“空降兵”，成功率只有70%**。因为简历会说谎，面试会演戏，只有在一起同过窗、打过仗的人，你才知道他的真实底细和DNA。

* **品格与团队默契，永远干翻才华（Giftedness）**：
  如果一个全职同工才华横溢、讲道极有恩赐、做起事来成果惊人，但他是个自我中心、不听指挥、无法跟团队协作的“刺头”——**坚决不要，或者立刻请他走人**。当一个人的才华和平台远远超越他的生命品格时，他带出来的服事越成功，对整个团队同工的士气和整个教会的灵性，其毒害就越深。赢球靠的是整体的黏性团队，而不是某一个傲慢的明星球员。
