# 引言：黏性团队，到底有什么不一样？

### 什么是“黏性团队”？
“黏性（Sticky）”团队最大的特征，就是**无论发生什么，他们都能死死地粘在一起**。

在任何团队中，大家都会有不同的利益诉求，也会有完全不同的看待问题的视角。普通的团队遇到分歧就会分崩离析，但“黏性团队”知道如何开诚布公地解决眼前的问题，而且开完会出来之后，大家在目标和异象上依然是高度统一的。他们那种真正的同工革命友谊，不会被强烈的意见分歧所破坏。

简单来说，黏性团队不仅活干得漂亮（高产），而且整个团队的组织关系非常健康。

### 为什么写这本书？
坦白说，这本书就是我刚出来带团队时，做梦都想得到的那本“团队建设手册”。这里面没有虚头巴脑的理论，全都是实打实的硬核原则和管理智慧。这些年我跟成千上万的牧者和商业领袖分享过这些经验，帮助他们成功地把团队里的每一个人都带到了“同一页纸上”（也就是达成共识）。

这本书的底子，是我早年写的一本叫《合一要素》（The Unity Factor）的小册子，那时候我主要在研究如何建立一个健康的长执会（Board）。但在《黏性团队》这本书里，我把研究的范围扩大了。因为我意识到，一个团队要走得远，有**三大核心关系**是绝对不容忽视的。这三个群体必须同心协作，如果其中任何一个群体“掉了链子”，整个事工都会跟着一起完蛋。

### 核心团队必须搞定的“三大核心关系”：

1. **同工会 / 长执会（The Board）**
   管他们叫长老、执事、理事、董事，还是单纯的“教会大咖”，这群人手里握着能成就你、也能毁掉你的生杀大权。他们要是能一条心，那是世界上最美好的事；他们要是开始内耗，那就是所有人的噩梦。

2. **全职员工 / 同工团队（The Staff）**
   带一支充满活力且高度合一的员工团队，不仅战斗力爆表，而且干起活来极其好玩。但如果想让这台机器卡死，也容易得很——团队里只要有一个人开始跟大部队唱反调、心怀怨恨，或者玩一种“悄悄的消极抵抗”，就足够把整个团队的激情和喜乐压榨干净，让负责带队的负责人痛不欲生。

3. **普通会众 / 小组长们（The Congregation）**
   虽然我们在本书中不会花长篇大论去讲怎么管理庞大的会众，但我们会深度探讨一件事：我们到底能做些什么，好把会众内部的冲突“扼杀在摇篮里”，并把那些可能会把整个教会撕裂的各种私人利益诉求，给梳理对齐了。

真正的合一绝对不是找一群没有脑子的“克隆人”。健康的团队里充满了性格迥异、观点完全相反的人，但他们知道如何“公平地吵架”。当想法碰撞的硝烟散去，最终做出决策后，他们会齐心协力地向外展现一个完全统一的战线。为了更大的使命和长远的利益，每一个人都愿意把自己的个人偏好和私人小算盘搁置一边。

这就是黏性团队的魅力。让我们开始吧。
