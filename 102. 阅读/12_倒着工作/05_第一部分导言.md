# 第一部分导言

倒着工作
## ** 第一部分导言**

在本书的前半部分，我们将详尽阐述定义*成为亚马逊人*的一些关键原则与流程。这些工作方式——多年来经过耐心打磨——成就了 Amazon 卓越的效率和它破纪录的增长。它们使 Amazon 的文化成为一种创新蓬勃生长、以取悦客户为高度优先事项的文化。我们将讲述这些原则与流程的一些起源故事，以此表明：它们都是为解决那些阻碍我们自由创新、无法持续满足客户的难题而生的解决方案。

Amazon 领导力准则是第一章的主要内容。在公司创立的最初日子里，它只由寥寥数人组成，在三间小屋里办公，那时并没有正式的领导力准则，因为从某种意义上说，Jeff 本人就是领导力准则。他撰写岗位说明，面试候选人，打包并发运箱子，还阅读每一封发给客户的电子邮件。参与业务的方方面面，使他得以非正式地向当时相对小规模的员工群体传达 Amazon 的理念。但公司扩张得如此之快，以至于很快就再也无法以这种方式来传达它了。于是，领导力准则应运而生。我们会讲述它们得以创立的那个过程的故事——这本身就是一个典型的亚马逊式故事——并描述它们是如何被注入公司运营的每一根毛细血管的。

与领导力准则相辅相成的，是各种机制（mechanisms），我们同样会在第一章讨论它们。这些机制是一套始终如一、反复运行的流程，确保领导力准则在公司里年复一年、日复一日地得到强化。我们将逐一说明 Amazon 为每一个团队以及整个公司制定年度计划和目标的方法，以及它如何在团队目标与公司目标之间建立一致性。我们还会描述 Amazon 与众不同的薪酬政策，它强化的是协作与长期专注，而非内部竞争与对短期收益的追逐。

在第二章，我们讨论抬杆人（Bar Raiser）——Amazon 独特的招聘流程。与领导力准则一样，我们之所以创立抬杆人，也是因为公司增长得极其迅猛。需要在极短时间内招聘大量新人，其中一个主要陷阱就是"紧迫性偏见"：由于工作压得你喘不过气、急需人手，你便容易对候选人的缺点视而不见。抬杆人为团队提供了一套方法，既能高效、迅速地招到最优秀的人，又不至于因此而偷工减料。

在一家以创造力著称的公司里，可分离的单线程领导（separable, single-threaded leadership）是 Amazon 最有用的发明之一。我们在第三章讨论它。这是一种组织策略，旨在把组织内部各种依赖关系所造成的效率拖累降到最低。其基本前提是：对于每一项举措或项目，都有一位专门的负责人，其关注点就是这个项目、且仅仅是这个项目，而这位负责人所统领的团队成员，其注意力同样只聚焦于这一个项目。本章既是对单线程领导的描述，也同样是我们如何一路摸索出它的故事：我们会梳理它当初为解决哪些难题而生，以及在找到那个真正奏效的方案之前，我们曾研究出的那些不尽完善的解决办法。我们还会讨论，为了让可分离的单线程团队成为现实，我们如何以及为何不得不彻底改变构建和部署技术的方式。

我们还发现，在会议中真正奏效的做法，与大多数公司在会议中所做的并不一样。尽管我们十分尊重 PowerPoint 作为一种视觉沟通工具和演讲辅助的价值，但我们通过惨痛的教训认识到，在一场一小时的会议里，要传达关于各项举措和正在进行的项目的复杂信息，它并非最佳格式。我们反而发现，由相关团队撰写的一份六页纸叙事文档，才是最能让会议中的每个人都迅速、高效地了解该团队所做工作的方法。与此同时，撰写这份叙事文档的过程，也要求团队自身去反思他们一直在做或提议要做的工作，并向他人清晰地阐明它，从而磨砺了他们对这项工作的思考。我们将在第四章讨论这种叙事文档体裁的细节——并给出一份范例。

在第五章，我们讨论 Amazon 是如何开发新想法和新产品的：从期望的客户体验倒着工作（逆向工作法）。在动手构建之前，我们会先撰写一份新闻稿（Press Release），以清晰地界定这个新想法或新产品将如何让客户受益；我们还会拟出一份常见问题（Frequently Asked Questions）清单，以便预先解决那些棘手的问题。我们会审慎而严苛地研究并修改其中的每一份文档，直到满意为止，然后才进入下一步。

客户同样处在我们分析和管理绩效指标的方式的核心。我们所强调的，是我们所称的可控输入指标（controllable input metrics），而非输出指标（output metrics）。可控输入指标（例如降低内部成本以便你能负担得起地下调产品价格、在网站上新增在售商品，或缩短标准配送时间）衡量的是这样一组活动：只要把它们做好，就会产出你所期望的结果，也就是输出指标（比如月度营收和股价）。我们将在第六章详细阐述这些指标，以及如何发现并追踪它们。

第一部分并不构成一份 Amazon 所发展出的全部原则与流程的详尽清单。我们挑选了那些我们认为最能诠释"成为亚马逊人"含义的原则与流程。我们会向你展示我们是如何一步步得出其中每一项的。我们还会给你提供具体、可付诸行动的信息，帮助你磨砺那些方法，使你的公司或组织（无论大小）都能最大限度地发挥其服务客户的潜能。
