# 与耶稣同祷

> *PRAYING WITH JESUS*

上帝儿女的祷告有别于一切其他祷告，因为他们是与耶稣及祂的圣灵一同向天父上帝祷告。这一三位一体的处境及其动态，使基督徒的祷告独特而无可比拟。因此，基督徒的祷告有赖于我们对三位一体上帝的生命与作为的参与。我们的祷告并不依靠人类内在灵性潜能的自我发展，乃是依靠上帝赐给祂软弱儿女的恩典。它完全依靠耶稣及祂的代祷之工。而耶稣将祂的名与话语赐给我们，使我们得以被纳入祂的祷告之中。

耶稣将祂的名赐给我们这一恩赐，可借一个发生于第一次世界大战的故事来加以阐明。两位毫无血缘关系的英国士兵，外貌却如同孪生，在同一军队中以同等军衔并肩服役。他们出身于社会的两极：一人来自书香世家，已婚育有幼子，且在家族企业中持有股份；另一人则出身于破碎的家庭。然而，两人成为了至交挚友。那位出身良好的士兵，常常向另一位倾诉自己的妻子与家人。他甚至告诉挚友：若他阵亡，友人应接过他的名字、冒充他的身份。后来，这一切果真应验。那个举目无亲、了无前途的士兵，与已故战友互换了身份。那位冒充者所得到的，远不止一个新名字——他还得到了一个有父母手足相爱的家庭、一位深情的妻子和儿子、家族企业的部分所有权，以及随之而来的社会地位。这一切，都借着战友之死、凭着战友的遗言与名字的馈赠，成为他的产业。

耶稣借着祂的道成肉身与牺牲之死，所成就的事远比这更伟大。祂使我们参与路德所称的"神圣交换"（the great exchange）。在祂受洗时，耶稣承担了我们的罪孽与愆尤、死亡与定罪；在我们受洗时，耶稣将祂在天父面前的地位赐给我们，以及祂作为天父独生子的身份。祂将自己的一切赐予我们，我们从祂那里得着新的自我与生命。

作为我们新身份的一部分，耶稣将祂的名以及奉祂的名向天父祷告的特权赐给我们。¹ 耶稣在约翰福音十六章二十三至二十四节、二十六至二十八节解释了其含义：

> 到那日，你们什么也就不问我了。我实实在在地告诉你们，你们若向父求什么，他必因我的名赐给你们。向来你们没有奉我的名求什么，如今你们求，就必得着，叫你们的喜乐可以满足。……到那日，你们要奉我的名祈求，我并不对你们说，我要为你们求父；父自己爱你们，因为你们已经爱我，又信我是从父出来的。我从父出来，到了世界；我又离开世界，往父那里去。

耶稣在此是在受难前夜的濯足节木，向门徒所讲的话。祂解释了为何必须经由死亡、复活与升天回归天父。祂谈到了两种截然不同的祷告方式。

**旧的祷告方式**，是人没有直接进到天父及祂恩典面前的通道。由于门徒无法直接来到父面前，他们便求耶稣代为转达请求。其格局如下：

```
         天父上帝
            ↑
      （耶稣代为转求）
            ↑
      耶稣圣子
            ↑
      耶稣的门徒（求告）
```

但在耶稣死而复活之后，门徒将能以一种新的方式祷告——这一新方式已借祂受死时圣殿帷幔的裂开（马太福音二十七章五十一节）而得到预示。耶稣说，祂复活之后，将不再需要专门为他们代求（约十六：二十三至二十四、二十六节）。祂不再作夹在天父与门徒之间的第三方，因为他们将借着信心与祂如此紧密地联合，使得他们的祷告即是祂的祷告。耶稣的门徒将奉祂的名，凭着对祂的信心，与耶稣一同直接来到父面前祷告。

```
         天父上帝
            ↑
      （奉耶稣的名祈求）
            ↑
      门徒与耶稣同在
```

这种新祷告方式最令人惊叹之处，在于它克服了我们对自身表现与被接纳资格的忧虑。天父听我们的祷告，如同那祷告是从耶稣口中发出的；祂对我们和我们的祷告，正如对耶稣和耶稣的祷告一般欣悦。祂聆听的是在耶稣里的我们——我们披戴着祂、被祂一切的美德所遮盖。这意味着，我们如今可以如同耶稣本人一样来到天父面前，为自己支取祂一切的祝福；同时也意味着，天父看待并对待我们，正如看待并对待祂的爱子一样。祂爱我们、垂听我们，正如祂爱子并垂听祂的儿子一般。我们与耶稣的联合，实如头与身体其余各肢体的联合，不可分离。天父并不按我们本来的面目看待我们，只按我们在耶稣里的面目看待我们。

路德解释了奉耶稣的名祷告的含义：

> 我有理由说：「我知道天父衷心乐意垂听我一切的祷告，因为我心中有基督——这位救主。基督为我祷告了，正因如此，我的祷告借着祂而蒙悦纳。」因此，我们必须将自己的祷告编织进祂的祷告之中。……借着祂，我们来到上帝面前。我们必须在祂里面将我们一切的祷告和一切所行的事都包裹起来、涵盖其中。正如圣保罗所宣告的（罗马书十三：十四），我们必须穿上基督；若要蒙上帝悦纳，一切都必须在祂里面行出（哥林多前书十：三十一）。
>
> 然而，这一切话语是向基督徒所说的，目的是使他们有胆量与把握，单单信靠这位救主，以全然确信的心来祷告；因为我们听见，祂如此将我们与祂自己联合，实实在在使我们与祂同等，将我们的祷告并入祂的祷告，也将祂的祷告并入我们的祷告。……还有什么荣耀能比这更大呢？我们对基督的信心使我们有权被称为祂的弟兄与共同承受者，我们的祷告当与祂的祷告相符，除了我们的祷告必须发源于祂、奉祂的名说出，方能蒙悦纳，方能使祂将这产业与荣耀赐给我们之外，实际上并无差别。除此之外，祂使我们与祂在一切事上同等；祂的祷告与我们的祷告必须合而为一，正如祂的身体是我们的，祂的肢体是我们的一样。²

按照耶稣这一教导，我们通常将祷告献给天父上帝。当然，我们也可以向耶稣祷告，也可以向圣灵祷告，但那不是通常的方式。我们普遍以「借着耶稣基督」或「奉耶稣的名」来结束向三位一体第一位的祷告，借此承认我们是与耶稣一同祷告——祂为我们代祷，也引领我们祷告。我们奉耶稣的名、凭着对祂的信心，坦然无惧地来到父面前，不忧虑被定罪或被拒绝。这一教导使我们在祷告中勇敢而有把握，原因有二：其一，我们无需焦虑上帝是否悦纳我们，是否会垂听我们（约翰一书三：二十一至二十二；四：十三至十五）；其二，我们无需为祷告什么、如何祷告而忧虑，因为耶稣以祂的义遮盖我们，并使我们的祷告趋于完全。我们的表现并不重要；重要的是耶稣，以及我们对祂作为我们中保的信心。
