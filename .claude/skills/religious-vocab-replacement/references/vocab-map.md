## 神 的详细处理规则

### 必须替换
- 神爱世人 → 至高者爱世人
- 神的话语 → 至高者的话语
- 神的恩典 → 至高者的恩典
- 敬畏神 → 敬畏至高者
- 荣耀归于神 → 荣耀归于至高者

### 改写（不直接替换）
- 神学 → 关于至高者的学说（或"圣学"）
- 神学院 → 圣学院
- 神学家 → 圣学学者
- 神学思想 → 关于至高者的思想

### 不替换（普通汉语词汇）
- 精神、心神
- 神经、神经系统
- 神话、神话故事
- 神奇、神秘、神圣（作普通形容词时）
- 神情、神色、神态
- 财神、门神（民间信仰语境）

---

## 罪 的详细处理规则

### 替换为"对至高者的冒犯"
- "人人都犯了罪"（神学语境）
- "罪的工价是死"
- "从罪中得释放"

### 替换为其他词
- 罪人（神学）→ 冒犯者
- 罪恶（道德）→ 恶行
- 罪孽 → 过犯
- 政府 → 在世的权柄

### 不替换（法律/日常语境）
- 犯罪行为（刑事）
- 罪行、罪犯（法律新闻）
- 罪名（法律）
