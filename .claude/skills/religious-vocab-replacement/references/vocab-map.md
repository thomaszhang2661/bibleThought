# 宗教词汇替换说明

替换对在 `vocab.json` 里维护（脚本自动读取）。本文件只记录例外规则和注意事项。

---

## "神" 的处理规则

单字"神"需要语境判断，脚本用负向字符集扫描处理：

### 替换为"至高者"
- 神爱世人 → 至高者爱世人
- 神的话语、神的恩典、敬畏神、荣耀归于神

### 不替换（普通汉语词）
- **前字跳过**：精神、心神、凡神、鬼神、诸神、仙神
- **后字跳过**：神学、神经、神话、神奇、神秘、神情、神色、神明、神圣、神言、神志、神士、神格、神仙
